Mintlify utiliza una aplicación de GitHub para sincronizar automáticamente tu documentación con tu repositorio de GitHub.

<div id="installing-the-github-app">
  ## Instalación de la app de GitHub
</div>

Instala la app de GitHub de Mintlify en la cuenta u organización de GitHub que es propietaria de tu repositorio de documentación. Debes ser propietario de la organización o tener permisos de administrador en el repositorio para instalar la app. Si no tienes los permisos necesarios, el propietario del repositorio debe aprobar la solicitud de instalación.

Instala la app de GitHub de Mintlify desde tu [panel](https://dashboard.mintlify.com/settings/organization/github-app).

<Tip>
Recomendamos otorgar acceso solo al repositorio donde se aloja tu documentación.
</Tip>

<div id="permissions">
  ## Permisos
</div>

Cuando instales la app de GitHub, se te pedirá que concedas los siguientes permisos:

Permisos de lectura:

* `metadata`: Información básica del repositorio

Permisos de lectura y escritura:

* `checks`: Crear verificaciones de estado en pull requests
* `code`: Leer cambios de archivos cuando hagas commits en tu rama de documentación
* `deployments`: Generar despliegues de vista previa para pull requests
* `pull requests`: Crear ramas y pull requests desde el editor web
* `workflows`: Gestionar GitHub Actions para los despliegues de documentación

<Info>
La app solo accede a los repositorios a los que le concedas acceso explícitamente. Si tienes reglas de protección de ramas habilitadas, la app no puede hacer push directamente a las ramas protegidas.
</Info>

<div id="managing-repository-access">
  ## Administración del acceso a repositorios
</div>

Al instalar nuestra app de GitHub, puedes conceder acceso a todos tus repositorios o solo a algunos específicos. Recomendamos conceder acceso únicamente a los repositorios donde se encuentra tu documentación. Puedes modificar esta selección en cualquier momento en la [configuración de la app de GitHub](https://github.com/apps/mintlify/installations/new).

<div id="configuring-docs-source">
  ## Configurar el origen de la documentación
</div>

Cambia la organización, el repositorio o la rama desde los que se genera tu documentación en la sección [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) de tu panel.

<div id="troubleshooting">
  ## Resolución de problemas
</div>

<div id="github-app-connection-issues">
  ### Problemas de conexión con la app de GitHub
</div>

Si tienes problemas con la app de GitHub, restablecer la conexión puede resolver la mayoría de los casos.

<Steps>
  <Step title="Desinstala la app de Mintlify desde GitHub.">
    1. En GitHub, ve a [installations](https://github.com/settings/installations) y selecciona **Configure** junto a la app de Mintlify. Desplázate hacia abajo y selecciona **Uninstall**.
    2. Ve a [Authorized GitHub Apps](https://github.com/settings/apps/authorizations) y selecciona **Revoke** junto a la app de Mintlify.
  </Step>
  <Step title="Vuelve a instalar la app de Mintlify.">
    1. En tu panel de Mintlify, ve a [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) e instala la app de GitHub.
    2. Autoriza tu cuenta en la sección [My Profile](https://dashboard.mintlify.com/settings/account) de tu panel.
  </Step>
</Steps>

<div id="feedback-add-ons-are-unavailable">
  ### Los complementos de comentarios no están disponibles
</div>

Si no puedes habilitar las opciones de sugerencias de edición o de informar problemas en tu panel, y tu repositorio es público, vuelve a validar tu configuración de Git.

<Steps>
  <Step title="Ve a la configuración de Git">
    Ve a [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) en tu panel.
  </Step>
  <Step title="Vuelve a validar tu configuración">
    Haz clic en la marca de verificación verde en la esquina del recuadro de configuración de Git para volver a validar la configuración de tu repositorio. Esto forzará la actualización de la configuración para reflejar si tu repositorio es público o privado.
