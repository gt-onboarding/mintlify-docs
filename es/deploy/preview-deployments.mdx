<Info>
  Los despliegues de vista previa están disponibles en los [planes Pro y Enterprise](https://mintlify.com/pricing?ref=preview-deployments).
</Info>

Los despliegues de vista previa te permiten ver cómo se verán los cambios en tu documentación antes de fusionarlos en producción. Cada vista previa crea una URL para compartir que se actualiza automáticamente a medida que haces push de nuevos cambios.

<div id="creating-preview-deployments">
  ## Creación de despliegues de vista previa
</div>

Los despliegues de vista previa se crean automáticamente mediante pull requests o manualmente desde tu panel.

<div id="automatic-previews">
  ### Vistas previas automáticas
</div>

Cuando creas una pull request, el bot de Mintlify agrega automáticamente un enlace para ver el despliegue de la vista previa en tu pull request. La vista previa se actualiza cada vez que haces push de nuevos commits a la rama.

<div id="manual-previews">
  ### Vistas previas manuales
</div>

Puedes crear manualmente una vista previa para cualquier rama.

1. Ve a tu [panel](https://dashboard.mintlify.com/).
2. Selecciona **Previews**.
3. Selecciona **Create preview**.
4. Introduce el nombre de tu rama.
5. Selecciona **Create deployment**.

<div id="redeploying-a-preview">
  ## Volver a desplegar una vista previa
</div>

Vuelve a desplegar una vista previa para actualizar el contenido o reintentarlo después de un despliegue fallido.

1. Selecciona la vista previa desde tu [panel](https://dashboard.mintlify.com/).
2. Selecciona **Redeploy**.

<div id="sharing-preview-deployments">
  ## Compartir despliegues de vista previa
</div>

Los despliegues de vista previa pueden ser de acceso público o restringidos a usuarios autenticados.

**Acceso público**: Las URL de vista previa son públicas de forma predeterminada. Comparte un enlace de vista previa con cualquiera que necesite revisar tus cambios.

**Acceso restringido**: Habilita la autenticación para los despliegues de vista previa para limitar el acceso únicamente a usuarios autenticados de Mintlify. Consulta [Configuración de autenticación](/es/deploy/authentication-setup) para obtener más información.
