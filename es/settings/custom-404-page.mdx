---
title: "Página 404 personalizada"
description: "Personaliza el título y la descripción de tu página de error 404"
icon: "file-warning"
---

Puedes controlar el título y la descripción de la página de error 404 que aparece cuando los usuarios acceden a enlaces rotos o inexistentes.

Al personalizar tu página 404, utiliza la descripción para orientar a los usuarios hacia recursos útiles o enlaces de tu documentación que les ayuden a encontrar lo que buscan.

<div id="configuration">
  ## Configuración
</div>

Configura tu página 404 en la sección `errors.404` de tu archivo `docs.json`:

```json
"errors": {
  "404": {
    "redirect": false,
    "title": "I can't be found",
    "description": "What ever **happened** to this _page_?"
  }
}
```

<div id="parameters">
  ## Parámetros
</div>

<ResponseField name="redirect" type="boolean">
  Indica si se debe redirigir automáticamente a la página de inicio cuando no se encuentra una página.
  
  Configúralo en `true` para redirigir a la página de inicio.

  Configúralo en `false` para mostrar la página 404 personalizada.
</ResponseField>

<ResponseField name="title" type="string">
  Título personalizado para la página de error 404. Sustituye el encabezado predeterminado “Page not found”.
</ResponseField>

<ResponseField name="description" type="string">
  Descripción personalizada para la página de error 404. Admite formato Markdown.
</ResponseField>
