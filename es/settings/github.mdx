Mintlify utiliza una app de GitHub para sincronizar automáticamente tu documentación con tu repositorio de GitHub.

<div id="installing-the-github-app">
  ## Instalación de la app de GitHub
</div>

Instala la app de GitHub de Mintlify en la cuenta u organización de GitHub propietaria de tu repositorio de documentación. Debes ser propietario de la organización o tener permisos de administrador en el repositorio para poder instalar la app. Si no tienes los permisos necesarios, el propietario del repositorio debe aprobar la solicitud de instalación.

Instala la app de GitHub de Mintlify desde tu [panel](https://dashboard.mintlify.com/settings/organization/github-app).

<Tip>
Recomendamos conceder acceso solo al repositorio donde se aloja tu documentación.
</Tip>

<div id="permissions">
  ## Permisos
</div>

Cuando instales la aplicación de GitHub, se te pedirá que concedas los siguientes permisos:

Permisos de lectura:

* `metadata`: Información básica del repositorio

Permisos de lectura y escritura:

* `checks`: Crear comprobaciones de estado en las pull requests
* `code`: Leer cambios de archivos cuando haces commit en tu rama de documentación
* `deployments`: Generar despliegues de vista previa para las pull requests
* `pull requests`: Crear ramas y pull requests desde el Editor web
* `workflows`: Administrar GitHub Actions para los despliegues de la documentación

<Info>
La aplicación solo accede a los repositorios a los que le otorgues acceso explícitamente. Si tienes reglas de protección de ramas habilitadas, la aplicación no puede hacer push directamente a las ramas protegidas.
</Info>

<div id="managing-repository-access">
  ## Gestión del acceso a repositorios
</div>

Al instalar nuestra aplicación de GitHub, puedes otorgar acceso a todos tus repositorios o solo a algunos específicos. Te recomendamos otorgar acceso únicamente a los repositorios donde se encuentra tu documentación. Puedes modificar esta selección en cualquier momento en la [configuración de la aplicación de GitHub](https://github.com/apps/mintlify/installations/new).

<div id="configuring-docs-source">
  ## Configurar la fuente de la documentación
</div>

Cambia la organización, el repositorio o la rama desde la que se construye tu documentación en la sección [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) de tu panel.

<div id="troubleshooting">
  ## Solución de problemas
</div>

Si tienes problemas con la app de GitHub, restablecer la conexión suele resolver la mayoría de los casos.

<Steps>
  <Step title="Desinstala la app de Mintlify desde GitHub.">
    1. En GitHub, ve a [Installations](https://github.com/settings/installations) y selecciona **Configure** junto a la app de Mintlify. Desplázate hacia abajo y selecciona **Uninstall**.
    2. Ve a [Authorized GitHub Apps](https://github.com/settings/apps/authorizations) y selecciona **Revoke** junto a la app de Mintlify.
  </Step>
  <Step title="Vuelve a instalar la app de Mintlify.">
    1. En tu panel de Mintlify, ve a [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) e instala la app de GitHub.
    2. Autoriza tu cuenta en la sección [My Profile](https://dashboard.mintlify.com/settings/account) del panel.
  </Step>
</Steps>
