---
title: "Fuentes"
description: "Configura fuentes personalizadas"
---

Configura una fuente personalizada para todo tu sitio o, si prefieres, fuentes distintas para los encabezados y el cuerpo del texto. Usa Google Fonts, archivos de fuentes locales o fuentes alojadas externamente. La fuente predeterminada varía según el tema.

Las fuentes se controlan mediante la [propiedad fonts](/es/organize/settings#param-fonts) en tu `docs.json`.

<div id="google-fonts">
  ## Google Fonts
</div>

Mintlify carga automáticamente [Google Fonts](https://fonts.google.com) cuando especificas una familia tipográfica en tu `docs.json`.

```json docs.json
"fonts": {
  "family": "Inter"
}
```

<div id="local-fonts">
  ## Fuentes locales
</div>

Para usar fuentes locales, coloca los archivos de fuentes en el directorio de tu proyecto y haz referencia a ellos en la configuración de `docs.json`.

<div id="setting-up-local-fonts">
  ### Configuración de fuentes locales
</div>

<Steps>
  <Step title="Agrega los archivos de fuente a tu proyecto">
    Por ejemplo, crea un directorio `fonts` y agrega tus archivos de fuente:

    ```text
    your-project/
    ├── fonts/
    │   ├── InterDisplay-Regular.woff2
    │   └── InterDisplay-Bold.woff2
    ├── docs.json
    └── ...
    ```
  </Step>

  <Step title="Configura las fuentes en docs.json">
    Haz referencia a tus fuentes locales usando rutas relativas desde la raíz de tu proyecto:

    ```json docs.json
    {
      "fonts": {
        "family": "InterDisplay",
        "source": "/fonts/InterDisplay-Regular.woff2",
        "format": "woff2",
        "weight": 400
      }
    }
    ```
  </Step>
</Steps>

<div id="local-fonts-for-headings-and-body">
  ### Fuentes locales para títulos y cuerpo
</div>

Configura fuentes locales independientes para los títulos y el cuerpo del texto en tu `docs.json`:

```json docs.json
{
  "fonts": {
    "heading": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Bold.woff2",
      "format": "woff2",
      "weight": 700
    },
    "body": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Regular.woff2",
      "format": "woff2",
      "weight": 400
    }
  }
}
```

<div id="externally-hosted-fonts">
  ## Fuentes alojadas externamente
</div>

Utiliza fuentes alojadas externamente haciendo referencia a una URL de origen de la fuente en tu `docs.json`:

```json docs.json
{
  "fonts": {
    "family": "Hubot Sans",
    "source": "https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2",
    "format": "woff2",
    "weight": 400
  }
}
```

<div id="font-configuration-reference">
  ## Referencia de configuración de fuentes
</div>

<ResponseField name="fonts" type="object">
  Configuración de fuentes para tu documentación.

  <Expandable title="Fuentes">
    <ResponseField name="family" type="string" required>
      Nombre de la familia tipográfica, como &quot;Inter&quot; o &quot;Playfair Display&quot;.
    </ResponseField>

    <ResponseField name="weight" type="number">
      Grosor de la fuente, como 400 o 700. Las fuentes variables admiten grosores precisos como 550.
    </ResponseField>

    <ResponseField name="source" type="string (uri)">
      URL de la fuente, como `https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2`, o ruta a tu archivo de fuente local, como `/assets/fonts/InterDisplay.woff2`. Google Fonts se carga automáticamente cuando especificas un nombre de `family` de Google Fonts, por lo que no se necesita una URL de origen.
    </ResponseField>

    <ResponseField name="format" type="'woff' | 'woff2'">
      Formato del archivo de fuente. Obligatorio al usar el campo `source`.
    </ResponseField>

    <ResponseField name="heading" type="object">
      Anula la configuración de fuentes específicamente para los encabezados.

      <Expandable title="Encabezado">
        <ResponseField name="family" type="string" required>

        <ResponseField name="weight" type="number">

        <ResponseField name="source" type="string (uri)">

        <ResponseField name="format" type="'woff' | 'woff2'">
