---
title: "PostHog"
---

Añade lo siguiente a tu archivo `docs.json` para enviar analíticas a PostHog.

Solo necesitas incluir `apiHost` si alojas PostHog por tu cuenta. De forma predeterminada, enviamos eventos a `https://app.posthog.com`.

<CodeGroup>

```json Opciones de analíticas en docs.json
"integrations": {
    "posthog": {
        "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
        "apiHost": "optional"
    }
}
```

```json Ejemplo
"integrations": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<br />

<Warning>
    Al habilitar las analíticas de PostHog, se desactivarán las analíticas en el panel de Mintlify.
</Warning>

<div id="session-recordings">
  ## Grabaciones de sesión
</div>

Antes de poder recibir grabaciones de sesión, debe añadir la URL de su sitio de documentación a “Authorized domains for recordings” en PostHog. La opción para añadir su URL se encuentra en la configuración del proyecto de PostHog.
