---
title: "Amplitude"
---

Agrega lo siguiente a tu archivo `docs.json` para enviar datos de analítica a Amplitude.

<CodeGroup>

```json Opciones de analítica en docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
