title: "Clearbit"
---

Agrega lo siguiente a tu archivo `docs.json` para enviar analítica a Clearbit.

<CodeGroup>

```json Opciones de analítica en docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Ejemplo
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
