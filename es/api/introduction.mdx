La API REST de Mintlify te permite interactuar de forma programática con tu documentación, activar actualizaciones e integrar experiencias de chat con IA.

<div id="endpoints">
  ## Endpoints
</div>

* [Activar actualización](/es/api-reference/update/trigger): Activa una actualización de tu sitio cuando lo desees.
* [Obtener estado de la actualización](/es/api-reference/update/status): Obtén el estado de una actualización y otros detalles sobre tu documentación.
* [Crear trabajo de agente](/es/api-reference/agent/create-agent-job): Crea un trabajo de agente para editar tu documentación automáticamente.
* [Obtener trabajo de agente](/es/api-reference/agent/get-agent-job): Obtén los detalles y el estado de un trabajo de agente específico.
* [Obtener todos los trabajos de agente](/es/api-reference/agent/get-all-jobs): Obtén todos los trabajos de agente de un dominio.
* [Generar mensaje del Asistente](/es/api-reference/assistant/create-assistant-message): Inserta el Asistente, entrenado con tu documentación, en cualquier aplicación que elijas.
* [Buscar en la documentación](/es/api-reference/assistant/search): Busca en tu documentación.

<div id="authentication">
  ## Autenticación
</div>

Puedes generar una clave de API desde el [panel](https://dashboard.mintlify.com/settings/organization/api-keys). Las claves de API están asociadas a toda la organización y se pueden usar en múltiples despliegues.

<div id="admin-api-key">
  ### Clave de API de administrador
</div>

La clave de API de administrador se utiliza para [activar una actualización](/es/api-reference/update/trigger), [obtener el estado de la actualización](/es/api-reference/update/status) y para todos los endpoints de agentes.

Las claves de API de administrador comienzan con el prefijo `mint_`. Mantén en secreto tus claves de API de administrador.

<div id="assistant-api-key">
  ### Clave de la API del Asistente
</div>

La clave de la API del Asistente se utiliza en los endpoints [Generate assistant message](/es/api-reference/assistant/create-assistant-message) y [Search documentation](/es/api-reference/assistant/search).

Las claves de la API del Asistente comienzan con el prefijo `mint_dsc_`.

La **clave** de la API del Asistente es un token del lado del servidor que debe mantenerse en secreto.

El **token** de la API del Asistente es público y puede utilizarse en tu código de frontend.

<Note>
  Las llamadas que usan el token de la API del Asistente pueden generar costos: ya sea consumiendo tus créditos del Asistente de IA o incurriendo en excedentes.
</Note>
