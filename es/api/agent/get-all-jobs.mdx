---
openapi: GET /agent/{projectId}/jobs
---

<div id="usage">
  ## Uso
</div>

Este endpoint obtiene todos los trabajos de agente del dominio especificado y ofrece una visión general de todas las actividades del agente y su estado actual. Resulta útil para supervisar y gestionar múltiples trabajos de agente, tanto concurrentes como históricos.

<div id="response">
  ## Respuesta
</div>

Usa este endpoint para obtener una visión completa de todas las sesiones anteriores del agente.
