---
title: "Expandables"
description: "Alternar para mostrar propiedades anidadas"
icon: "list-tree"
---

Usa expandables para mostrar y ocultar contenido anidado dentro de campos de respuesta. Los expandables son especialmente útiles para mostrar propiedades de objetos complejos en la documentación de API.

<ResponseField name="user" type="User Object">
  <Expandable title="properties">

<ResponseField name="full_name" type="string">
  El nombre completo del usuario
</ResponseField>

<ResponseField name="is_over_21" type="boolean">
  Si el usuario es mayor de 21 años
</ResponseField>

  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      The full name of the user
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Whether the user is over 21 years old
    </ResponseField>
  </Expandable>
</ResponseField>
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="title" type="string">
  El nombre del objeto que se muestra.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Establece `true` para que el elemento expandible se abra al cargar la página.
</ResponseField>
