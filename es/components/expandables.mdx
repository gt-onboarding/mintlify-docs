---
title: "Expandibles"
description: "Alterna para mostrar propiedades anidadas"
icon: "list-tree"
---

Usa expandibles para mostrar u ocultar contenido anidado dentro de los campos de respuesta. Los expandibles son especialmente útiles para presentar propiedades de objetos complejos en la documentación de API.

<ResponseField name="user" type="User Object">
  <Expandable title="properties">

<ResponseField name="full_name" type="string">
  El nombre completo del usuario
</ResponseField>

<ResponseField name="is_over_21" type="boolean">
  Indica si el usuario es mayor de 21 años
</ResponseField>

  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      El nombre completo del usuario
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Indica si el usuario es mayor de 21 años
    </ResponseField>
  </Expandable>
</ResponseField>
```

<div id="properties">
  ## Propiedades
</div>

<ResponseField name="title" type="string">
  El nombre del objeto que se está mostrando.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Establézcalo en `true` para que el elemento expandible se abra al cargarse la página.
</ResponseField>
