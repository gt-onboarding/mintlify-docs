El componente `Columns` permite agrupar varios componentes `Card`. Se utiliza principalmente para disponer varias tarjetas en una cuadrícula, especificando el número de columnas.

<Columns cols={2}>
  <Card title="Primera tarjeta" icon="panel-left-close">
    Neque porro quisquam est qui dolorem ipsum quia dolor sit amet
  </Card>
  <Card title="Segunda tarjeta" icon="panel-right-close">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit
  </Card>
</Columns>
