---
title: "Llamadas de atención"
description: "Usa llamadas de atención para añadir contexto destacado a tu contenido"
icon: "info"
---

Las llamadas de atención pueden tener estilo de Note, Warning, Info, Tip, Check o Danger, o bien puedes crear una personalizada:

<Note>Esto añade una nota en el contenido</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>Esto muestra una advertencia a tener en cuenta</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Esto resalta información importante</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>Esto sugiere un consejo útil</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Esto indica un estado verificado</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>Esto es una llamada de atención de peligro</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">Esta es una llamada de atención personalizada</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
