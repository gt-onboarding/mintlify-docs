---
title: "Páginas ocultas"
description: "Excluir páginas de la navegación"
icon: "eye-closed"
---

Las páginas ocultas se eliminan de la navegación de tu sitio, pero siguen siendo accesibles públicamente para cualquiera que conozca su URL.

Utiliza páginas ocultas para contenido que quieres que sea accesible en tu sitio o referenciado como contexto para herramientas de IA, pero que no sea localizable a través de la navegación.

Para contenido que requiera un control de acceso estricto, debes configurar la [autenticación](/es/authentication-personalization/authentication-setup).

Si quieres ocultar páginas para grupos específicos de usuarios, utiliza la personalización para controlar la [visibilidad de las páginas](/es/authentication-personalization/overview#page-visibility).

<div id="hiding-a-page">
  ## Ocultar una página
</div>

Una página queda oculta si no está incluida en la navegación de tu `docs.json`. Para ocultar una página, elimínala de la estructura de navegación.

Las páginas ocultas usan la misma estructura de URL que las páginas normales, basada en su ruta de archivo. Por ejemplo, `guides/hidden-page.mdx` sería accesible en `docs.yoursite.com/guides/hidden-page`.

Consulta un [ejemplo de página oculta](/es/guides/hidden-page-example).

<Note>
  Algunos elementos de navegación, como las barras laterales, los menús desplegables y las pestañas, pueden aparecer vacíos o alterar el diseño en las páginas ocultas.
</Note>

<div id="hiding-a-group-of-pages">
  ## Ocultar un grupo de páginas
</div>

Un grupo de páginas se oculta si la propiedad `hidden` se establece en `true` en tu archivo `docs.json`:

```json highlight={4}
"groups": [
  {
    "group": "Getting started",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  },
  {
    "group": "Guides",
    "pages": [
      "guides/hidden-page.mdx",
      "guides/hidden-groups.mdx"
    ]
  }
]
```

En este ejemplo, el grupo `Getting started` está oculto, pero el grupo `Guides` es visible.

<div id="hiding-a-tab">
  ### Ocultar una pestaña
</div>

También puedes ocultar una pestaña añadiendo la propiedad `hidden` a tu archivo `docs.json`:

```json highlight={4}
"tabs": [
  {
    "tab": "Home",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  }
]
```

<div id="search-seo-and-ai-indexing">
  ## Búsqueda, SEO e indexación con IA
</div>

De forma predeterminada, las páginas ocultas se excluyen de la indexación para los motores de búsqueda, la búsqueda interna en tu documentación y como contexto para el Asistente. Para incluir páginas ocultas en los resultados de búsqueda y como contexto para el Asistente, agrega la propiedad `seo` a tu `docs.json`:

```json
"seo": {
    "indexing": "all"
}
```

Para excluir una página específica, agrega `noindex: "true"` a su frontmatter.
