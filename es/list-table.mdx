<div id="lists">
  ## Listas
</div>

Las listas siguen la [sintaxis oficial de Markdown](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Lista ordenada
</div>

Para crear una lista ordenada, añade números seguidos de un punto antes de los elementos.

1. Primer elemento
2. Segundo elemento
3. Tercer elemento
4. Cuarto elemento

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### Lista sin ordenar
</div>

Para crear una lista sin ordenar, agrega guiones (`-`), asteriscos (`*`) o signos más (`+`) antes de los elementos de la lista.

- Primer elemento
- Segundo elemento
- Tercer elemento
- Cuarto elemento

```mdx
- Primer elemento
- Segundo elemento
- Tercer elemento
- Cuarto elemento
```

<div id="nested-list">
  ### Lista anidada
</div>

Indenta los elementos de la lista para anidarlos.

- Primer elemento
- Segundo elemento
  - Elemento adicional
  - Elemento adicional
- Tercer elemento

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## Tablas
</div>

Las tablas siguen la [sintaxis oficial de Markdown](https://www.markdownguide.org/extended-syntax/#tables).

Para añadir una tabla, utiliza tres o más guiones (`---`) para crear el encabezado de cada columna y barras verticales (`|`) para separar cada columna. Por compatibilidad, también debes añadir una barra vertical al principio y al final de la fila.

| Propiedad | Descripción                          |
| --------- | ------------------------------------ |
| Nombre    | Nombre completo del usuario          |
| Edad      | Edad informada                       |
| Unión     | Si el usuario se unió a la comunidad |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
