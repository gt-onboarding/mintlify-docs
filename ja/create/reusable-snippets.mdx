---
title: "再利用可能なスニペット"
description: "コンテンツを同期させるための再利用可能なカスタムスニペット"
---

ソフトウェア開発の基本原則のひとつに DRY（Don&#39;t Repeat Yourself）があります。これはドキュメントにも当てはまります。複数の場所で同じ内容を繰り返している場合は、コンテンツを同期させるためにカスタムスニペットを作成しましょう。

<div id="creating-a-custom-snippet">
  ## カスタムスニペットの作成
</div>

**前提条件**: インポートを機能させるには、`snippets` ディレクトリ内にスニペットファイルを作成する必要があります。

`snippets` ディレクトリ内のページはすべてスニペットとして扱われ、単独のページとしてはレンダリングされません。スニペットから単独ページを作成したい場合は、そのスニペットを別のファイルにインポートし、コンポーネントとして呼び出してください。

<div id="default-export">
  ### デフォルトエクスポート
</div>

1. 再利用したいコンテンツをスニペットファイルに追加します。必要に応じて、スニペットをインポートする際に props から渡せる変数を追加できます。
   この例では、変数は word です。

```typescript snippets/my-snippet.mdx
Hello world! これは複数のページで再利用したいコンテンツです。 
```

2. スニペットを目的のファイルにインポートします。

```typescript destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet/>

```

<div id="exporting-with-variables">
  ### 変数を使ったエクスポート
</div>

1. スニペットをインポートするときに props で値を渡せる変数を任意で追加できます。この例では、変数は word です。

```typescript snippets/my-snippet.mdx
今日のキーワードは {word} です。
```

2. 変数を使ってスニペットを目的のファイルにインポートします。プロパティは指定に基づいて自動で補完されます。

```typescript destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />

```

<div id="reusable-variables">
  ### 再利用可能な変数
</div>

1. スニペットファイルから変数をエクスポートします：

```typescript snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. 目的のファイルからスニペットをインポートし、その変数を使用します:

```typescript destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

こんにちは、私の名前は {myName} で、{myObject.fruit} が好きです。
```

<div id="jsx-snippets">
  ### JSX スニペット
</div>

1. スニペットファイルから JSX コンポーネントをエクスポートします（詳細は [React コンポーネント](/ja/customize/react-components) を参照）。

```js icon=square-js snippets/my-jsx-snippet.jsx
export const MyJSXSnippet = () => {
  return (
    <div>
      <h1>Hello, world!</h1>
    </div>
  )
}
```

<Note>
  重要: JSX スニペットを作成する際は、関数宣言ではなくアロー関数構文（`=>`）を使用してください。この文脈では `function` キーワードはサポートされていません。
</Note>

2. 参照先のファイルからスニペットをインポートし、コンポーネントを使用します:

```typescript destination-file.mdx
---
title: タイトル
description: 説明
---

import { MyJSXSnippet } from '/snippets/my-jsx-snippet.jsx';

<MyJSXSnippet />
```
