<div id="lists">
  ## リスト
</div>

リストは公式の[Markdown構文](https://www.markdownguide.org/basic-syntax/#lists-1)に準拠します。

<div id="ordered-list">
  ### 番号付きリスト
</div>

番号付きリストを作成するには、各項目の前に「数字 + ピリオド」を付けます。

1. 最初の項目
2. 2番目の項目
3. 3番目の項目
4. 4番目の項目

```mdx
1. 最初の項目
2. 2番目の項目
3. 3番目の項目
4. 4番目の項目
```


<div id="unordered-list">
  ### 箇条書き
</div>

箇条書きを作成するには、各項目の先頭にハイフン（`-`）、アスタリスク（`*`）、またはプラス（`+`）を付けます。

* 1つ目の項目
* 2つ目の項目
* 3つ目の項目
* 4つ目の項目

```mdx
- 最初の項目
- 2番目の項目
- 3番目の項目
- 4番目の項目
```


<div id="nested-list">
  ### 入れ子リスト
</div>

リスト項目をインデントして入れ子にします。

* 最初の項目
* 2番目の項目
  * 追加の項目
  * 追加の項目
* 3番目の項目

```mdx
- 最初の項目
- 2番目の項目
  - 追加項目
  - 追加項目
- 3番目の項目
```


<div id="tables">
  ## テーブル
</div>

テーブルは公式の[Markdown 構文](https://www.markdownguide.org/extended-syntax/#tables)に準拠します。

テーブルを追加するには、3 本以上のハイフン（`---`）で各列のヘッダーを作成し、パイプ（`|`）で各列を区切ります。互換性のため、行の先頭と末尾にもパイプを追加してください。

| プロパティ | 説明                              |
| ---------- | --------------------------------- |
| Name       | ユーザーの氏名                    |
| Age        | 申告年齢                          |
| Joined     | コミュニティに参加済みかどうか     |

```mdx
| プロパティ | 説明                           |
| -------- | ------------------------------------- |
| Name     | ユーザーの氏名                     |
| Age      | 年齢                          |
| Joined   | コミュニティへの参加状況 |
```
