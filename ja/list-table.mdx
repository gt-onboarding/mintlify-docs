<div id="lists">
  ## リスト
</div>

リストは公式の[Markdown構文](https://www.markdownguide.org/basic-syntax/#lists-1)に準拠します。

<div id="ordered-list">
  ### 番号付きリスト
</div>

番号付きリストを作成するには、各項目の前に数字とピリオドを付けます。

1. 最初の項目
2. 2番目の項目
3. 3番目の項目
4. 4番目の項目

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### 箇条書き（順不同）リスト
</div>

箇条書き（順不同）リストを作成するには、各項目の先頭にダッシュ（`-`）、アスタリスク（`*`）、またはプラス記号（`+`）を付けます。

- 1つ目の項目
- 2つ目の項目
- 3つ目の項目
- 4つ目の項目

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### ネストされたリスト
</div>

項目をインデントしてネストします。

- 最初の項目
- 2番目の項目
  - 追加の項目
  - 追加の項目
- 3番目の項目

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## テーブル
</div>

テーブルは公式の[Markdown構文](https://www.markdownguide.org/extended-syntax/#tables)に従います。

テーブルを追加するには、各列のヘッダーを作成するために3つ以上のハイフン（`---`）を使用し、各列を区切るためにパイプ（`|`）を使用します。互換性のため、行の両端にもパイプを追加してください。

| プロパティ | 説明                           |
| ---------- | ------------------------------ |
| Name       | ユーザーの氏名                 |
| Age        | 申告年齢                       |
| Joined     | ユーザーがコミュニティに参加したか |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
