---
title: "コードグループ"
description: "1つのコンポーネントで複数のコード例を表示する"
---

`CodeGroup` コンポーネントを使うと、タブ付きインターフェースで複数のコードブロックを表示でき、ユーザーは異なるプログラミング言語の実装を比較したり、同一タスクに対する別のアプローチを確認できます。

<CodeGroup>
  ```javascript helloWorld.js
  console.log("Hello World");
  ```

  ```python hello_world.py
  print('Hello World!')
  ```

  ```java HelloWorld.java
  class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
</CodeGroup>

コードグループは `docs.json` のグローバルスタイルを継承します。`styling.codeblocks` でテーマをカスタマイズできます。設定オプションは [Settings](/ja/organize/settings#styling) を参照してください。

<div id="creating-code-groups">
  ## コードグループの作成
</div>

コードグループを作成するには、複数のコードブロックを `<CodeGroup>` タグで囲みます。各コードブロックには、タブのラベルとなるタイトルを必ず含めてください。

````mdx
<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>
````

<div id="language-dropdown">
  ## 言語ドロップダウン
