---
title: "Callouts"
description: "コンテンツに目を引く補足情報を追加するにはコールアウトを使用します"
icon: "info"
---

コールアウトは Note、Warning、Info、Tip、Check、Danger のスタイルを使用できます。独自のコールアウトを作成することも可能です:

<Note>これはコンテンツに注記を追加します</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>これは注意喚起の警告を表示します</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>これは重要な情報に注意を促します</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>これは役に立つヒントを提案します</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>これは確認済みの状態を示します</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>これは危険のコールアウトです</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">これはカスタムコールアウトです</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
