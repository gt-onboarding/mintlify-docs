---
title: 'コールアウト'
description: 'コールアウトを使ってコンテンツに目を引く文脈情報を追加する'
icon: 'info'
---

コールアウトは Note、Warning、Info、Tip、Check、Danger としてスタイルできます。独自のコールアウトを作成することも可能です:

<Note>コンテンツ内に注記を追加します</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>注意を促す警告を表示します</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>重要な情報への注意喚起を行います</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>役立つヒントを提示します</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>確認済みであることを示します</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>危険を知らせるコールアウトです</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">カスタムのコールアウトです</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
