---
title: "コールアウト"
description: "コールアウトを使って、コンテンツに目を引く補足情報を追加しましょう"
---

コールアウトは Note、Warning、Info、Tip、Check、Danger のスタイルにでき、独自のコールアウトを作成することもできます：

<Note>これはコンテンツ内にメモを追加します</Note>

```mdx
<Note>コンテンツにノートを追加します</Note>
```

<Warning>注意が必要です</Warning>

```mdx
<Warning>注意すべき警告が表示されます</Warning>
```

<Info>重要な情報を強調します</Info>

```mdx
<Info>重要な情報を強調します</Info>
```

<Tip>これは役立つヒントです</Tip>

```mdx
<Tip>これは役立つヒントです</Tip>
```

<Check>これでステータスが「チェック済み」になります</Check>

```mdx
<Check>これによりチェック済みステータスが表示されます</Check>
```

<Danger>これは危険の警告です</Danger>

```mdx
<Danger>これは危険の注意喚起です</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> これはカスタムのコールアウトです</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">カスタムコールアウトの例</Callout>
```
