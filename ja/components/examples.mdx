`<RequestExample>` と `<ResponseExample>` コンポーネントは、右サイドバーにコードブロックを表示し、コンテンツをスクロールしている間も例を見やすく保つための二列レイアウトを作成します。これらのコンポーネントは API ドキュメント向けに設計されていますが、すべてのページで使用できます。

主なユースケース:

* リクエストとレスポンスの例を伴う API エンドポイントのドキュメント
* 説明文に沿った設定例
* 手順に沿って参照できるコードサンプル
* チュートリアルにおける Before/After の例

モバイルでは、`<RequestExample>` と `<ResponseExample>` コンポーネントは通常のコードブロックとして表示され、スクロールで通過できます。

<RequestExample>
  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```
</RequestExample>

<ResponseExample>
  ```json Response
  { "status": "success" }
  ```
</ResponseExample>

<div id="requestexample">
  ## RequestExample
</div>

右側のサイドバーにコード例をピン留めするには `<RequestExample>` を使用します。このコンポーネントは [CodeGroup](/ja/components/code-groups) コンポーネントと同様に動作しますが、コードをインラインではなくサイドバーに表示します。

1つの `<RequestExample>` 内に複数のコードブロックを含められます。各コードブロックには title 属性が必要です。

````mdx RequestExample
<RequestExample>

```bash Request
  curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

</RequestExample>
````

<div id="responseexample">
  ## ResponseExample
</div>

`<ResponseExample>` コンポーネントは、同一ページ内の `<RequestExample>` コンテンツの直下に、右サイドバーへコード例をピン留めします。

````mdx ResponseExample
<ResponseExample>

```json Response
{ "status": "success" }
```

</ResponseExample>
````
