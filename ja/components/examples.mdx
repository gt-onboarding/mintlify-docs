`<RequestExample>` と `<ResponseExample>` コンポーネントは、右サイドバーにコードブロックを表示し、ユーザーがコンテンツをスクロールしても例が常に見える二列レイアウトを作成します。これらのコンポーネントは API ドキュメント向けに設計されていますが、あらゆるページで使用できます。

主なユースケース:

- リクエストとレスポンスの例を含む API エンドポイントのドキュメント
- 解説テキストと並べて示す設定例
- 手順に沿って参照できるコードサンプル
- チュートリアルの Before/After 例

モバイルデバイスでは、`<RequestExample>` と `<ResponseExample>` は通常のコードブロックとして表示され、スクロールで通り過ぎられます。

<div id="requestexample">
  ## RequestExample
</div>

右側のサイドバーにコード例を固定するには、`<RequestExample>` を使用します。このコンポーネントは [CodeGroup](/ja/components/code-groups) と似ていますが、コードをインラインではなくサイドバーに表示します。

1つの `<RequestExample>` の中に複数のコードブロックを含められます。各コードブロックには title 属性が必要です。

````mdx RequestExample
<RequestExample>

```bash Request
  curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

</RequestExample>
````

<div id="responseexample">
  ## ResponseExample
</div>

`<ResponseExample>`コンポーネントは、同一ページ内の`<RequestExample>`コンテンツの直下に、右サイドバーでコード例をピン留め表示します。

````mdx ResponseExample
<ResponseExample>

```json Response
{ "status": "success" }
```

</ResponseExample>
````
