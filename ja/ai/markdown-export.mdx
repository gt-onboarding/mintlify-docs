Markdown は、HTML よりも AI ツールが効率的に処理できる構造化テキストを提供するため、応答の正確性が向上し、処理が高速化され、トークン使用量も抑えられます。

Mintlify は、AI ツールや外部統合向けに最適化されたページの Markdown 版を自動的に生成します。


<div id="md-url-extension">
  ## .md URL拡張子
</div>

任意のページのURLに`.md`を付けると、Markdown版を表示できます。

<PreviewButton href="https://mintlify.com/docs/ai/markdown-export.md">このページをMarkdownで開く</PreviewButton>

<div id="keyboard-shortcut">
  ## キーボードショートカット
</div>

<kbd>Command</kbd> + <kbd>C</kbd>（Windows は <kbd>Ctrl</kbd> + <kbd>C</kbd>）で、ページを Markdown としてクリップボードにコピーします。
