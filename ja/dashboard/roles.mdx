title: "ロール"
description: "ロールでダッシュボードへのアクセスを管理します。"
---

<Info>
  RBAC 機能は [Enterprise プラン](https://mintlify.com/pricing?ref=rbac)で利用できます。
</Info>

Mintlify では、ダッシュボードのアクセスレベルとして Editor と Admin の 2 種類を提供しています。

以下は Admin ロールに限定される操作です:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| ユーザーのロールを更新   |   ❌   |  ✅   |
| ユーザーを削除           |   ❌   |  ✅   |
| 管理者ユーザーを招待     |   ❌   |  ✅   |
| 請求情報の管理・更新     |   ❌   |  ✅   |
| カスタムドメインを更新   |   ❌   |  ✅   |
| Git ソースを更新         |   ❌   |  ✅   |
| 組織を削除               |   ❌   |  ✅   |

その他のダッシュボード上の操作は、両方のロールで利用できます。

管理者は必要な人数だけ招待できますが、必要なユーザーに限定することを推奨します。
