<div id="rate-limits">
  ## レート制限
</div>

アシスタント API には次の制限があります。

- 月あたりキーごとに 10,000 回の利用
- 時間あたり Mintlify 組織ごとに 10,000 件のリクエスト
- 日あたり IP ごとに 10,000 件のリクエスト

<div id="suggested-usage">
  ## 推奨される使い方
</div>

最良の結果を得るには、リクエストの送信とレスポンスの処理に [ai-sdk の useChat フック](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat) を使用してください。

フックに渡すオプションの `body` フィールドで、`fp`、`threadId`、`filter` を指定できます。
