---
title: "概要"
description: "ドキュメントのトラフィックと全体的なインサイトを確認する"
---

概要タブでは、ドキュメントの訪問者数、最も人気のあるページ、ユーザーの流入元を確認できます。これらの情報を活用して、ユーザーにとって最も価値の高いページを特定し、時間の経過に伴う傾向を追跡しましょう。

[dashboard](https://dashboard.mintlify.com/products/insights) の **Insights** ページに移動して、概要の指標にアクセスします。

<div id="metrics">
  ## メトリクス
</div>

範囲セレクターで表示期間を調整します。訪問者、ビュー、またはアクションを選ぶと、選択した期間のトレンドを示す折れ線グラフが表示されます。

- **Visitors**: ユニーク訪問者数
- **Views**: 総ページビュー数
- **Actions**: API 呼び出し、ナビゲーションバーのリンククリック、CTA ボタンクリックの合計数
- **Popular Pages**: 最も閲覧されたページのパスとそのビュー数
- **Referrers**: ドキュメントへのトラフィックを送っている主な参照元

<div id="using-overview-data">
  ## 概要データの活用
</div>

概要インサイトを確認して、次を行いましょう：

- **人気ページの特定**：人気ページを把握し、ユーザーにとって最重要なコンテンツが最新かつ網羅的であるように整備します。
- **トラフィック動向の追跡**：トラフィックの変化を監視し、更新や新規コンテンツの影響を評価します。
