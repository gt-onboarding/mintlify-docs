title: "Front"
---

[Front Chat](https://front.com) ウィジェットを追加するには、`docs.json` に次の内容を追加します。

<CodeGroup>

```json Integration options in docs.json
"integrations": {
    "frontchat": "CHAT_ID"
}
```

```json Example
"integrations": {
    "frontchat": "1365d046d7c023e9b030ce90d02d093a"
}
```

</CodeGroup>
