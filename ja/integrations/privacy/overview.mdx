<div id="enabling-data-privacy-integrations">
  ## データプライバシー向け統合の有効化
</div>

ドキュメントにデータプライバシープラットフォームを追加できます。対象のスクリプトを指定して、`docs.json` に `integrations` フィールドを追加してください。

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

<div id="cookie-consent-and-disabling-telemetry">
  ## Cookieの同意とテレメトリーの無効化
</div>

GDPR準拠のためにユーザーがすでにCookieに同意しているか確認する必要がある場合は、`cookies` の下にローカルストレージのキーと値を指定します:

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

これらの値が設定されていると、ローカルストレージをチェックしてユーザーのCookie同意の有無を判定します。未同意の場合はテレメトリーを無効化します。

すべてのユーザーでテレメトリーを無効化するには、`docs.json` に次を追加します:

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
