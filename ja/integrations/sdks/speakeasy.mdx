Speakeasy SDKで自動生成されたコードスニペットは、MintlifyのAPIリファレンスドキュメントに直接統合できます。これらのSDK利用スニペットは、Mintlifyで構築されたドキュメントサイトの[インタラクティブなAPIプレイグラウンド](/ja/api-playground/overview)に表示されます。

<Frame>
  ![Speakeasyのコードスニペットが表示されたMintlifyのAPIプレイグラウンド](/images/speakeasy/mintlify-with-speakeasy-openapi.png)
</Frame>

<div id="prerequisites">
  ## 前提条件
</div>

Mintlify を Speakeasy と統合するには、以下が必要です。

- [Mintlify のドキュメンテーションリポジトリ](/ja/quickstart#creating-the-repository)
  - [自動コードサンプル URL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls)を設定済みの、Speakeasy で生成された SDK

<div id="setting-up-the-integration">
  ## 統合の設定
</div>

Speakeasy を Mintlify と統合するには、レジストリから API の統合仕様の公開 URL を取得し、`docs.json` 設定ファイルを更新します。

<div id="get-the-apis-combined-spec-public-url-from-the-registry">
  ### レジストリからAPIの結合仕様の公開URLを取得する
</div>

[Speakeasy Dashboard](https://app.speakeasy.com) に移動し、**API Registry** タブを開きます。APIの `*-with-code-samples` エントリを開きます。

<Frame>
  ![SpeakeasyのAPI Registryページのスクリーンショット。API Registryタブが赤い四角と番号1で強調され、APIのエントリが赤い四角と番号2で強調されています。](/images/speakeasy/openapi-registry-and-combined-spec.png)
</Frame>

<Note>
  エントリに **Combined Spec** のラベルが表示されていない場合は、APIに[自動コードサンプルURL](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls)が設定されていることを確認してください。
</Note>

レジストリエントリのページで、表示されている公開URLをコピーします。

<Frame>
  ![結合仕様のレジストリエントリで、URLコピー機能が赤い四角で強調されているスクリーンショット。](/images/speakeasy/copy-combined-spec-url.png)
</Frame>

<div id="update-your-docsjson-configuration-file">
  ### `docs.json` 設定ファイルを更新する
</div>

結合済みの仕様URLを、`docs.json` ファイル内の **Anchors** または **Tabs** セクションに追加します。

アンカーに結合済みの仕様URLを追加するには、`docs.json` ファイルの `anchor` フィールドを次のように更新します:

```json docs.json
{
  "anchors": [
    {
      "name": "API Reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL",
      "url": "api-reference",
      "icon": "square-terminal"
    }
  ]
}
```

タブに結合済みの仕様URLを追加するには、`docs.json` ファイルの `tab` フィールドを次のように更新します:

```json docs.json
{
  "tabs": [
    {
      "name": "API Reference",
      "url": "api-reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL"
    }
  ]
}
```

Speakeasy が生成したコードスニペットは、APIドキュメントで閲覧でき、APIプレイグラウンドで操作できます。
