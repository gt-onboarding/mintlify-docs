---
title: "Segment"
---

分析データを Segment に送信するには、Segment の write key を `docs.json` に追加します。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Example
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
