---
title: "Heap"
---

アナリティクスを Heap に送信するには、`docs.json` ファイルに次を追加します。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
