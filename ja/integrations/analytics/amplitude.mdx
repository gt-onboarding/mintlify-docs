---
title: "Amplitude"
---

Amplitude にアナリティクスを送信するには、`docs.json` ファイルに次を追加します。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Example
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
