---
title: "Pirsch"
---

`docs.json` に以下を追加して、分析データを Pirsch に送信します。

サイト ID は Settings > Developer > Identification Code から取得できます。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
