---
title: "Pirsch"
---

分析データを Pirsch に送信するには、`docs.json` ファイルに次を追加します。

サイトIDは Settings > Developer > Identification Code で取得できます。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
