---
title: "Koala"
---

Koala にアナリティクスを送信するには、`docs.json` に次を追加します。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
