title: "Koala"
---

`docs.json` ファイルに次の設定を追加して、Koala に分析データを送信します。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
