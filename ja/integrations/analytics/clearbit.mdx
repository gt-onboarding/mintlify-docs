---
title: "Clearbit"
---

分析データを Clearbit に送信するには、`docs.json` に次を追加します。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "clearbit": {
        "publicApiKey": "required"
    }
}
```

```json Example
"integrations": {
    "clearbit": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
