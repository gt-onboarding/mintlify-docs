---
title: "Hightouch"
---

`docs.json` ファイルに次を追加して、アナリティクスデータを Hightouch に送信します。

<Info>

  `apiHost` には `https://` を含めないでください。

</Info>

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "hightouch": {
        "writeKey": "required"
        "apiHost": "optional"
    }
}
```

```json Example
"integrations": {
    "hightouch": {
        "writeKey": "9132c81do371p76sr11da0075469b54f77649c9a067dp0303p56q0q64n072336"
        "apiHost": "us-east-1.hightouch-events.com" # 任意。既定値は `us-east-1.hightouch-events.com`
    }
}
```

</CodeGroup>
