---
title: "カスタム 404 ページ"
description: "404 エラーページのタイトルと説明をカスタマイズする"
icon: "file-warning"
---

ユーザーが壊れたリンクや存在しないリンクにアクセスしたときに表示される 404 エラーページのタイトルと説明を設定できます。

404 ページをカスタマイズする際は、説明文で、ユーザーが探している情報にたどり着けるよう、役立つリソースやドキュメント内の関連リンクへ誘導しましょう。

<div id="configuration">
  ## 設定
</div>

`docs.json` ファイルの `errors.404` セクションで 404 ページを設定します：

```json
"errors": {
  "404": {
    "redirect": false,
    "title": "I can't be found",
    "description": "What ever **happened** to this _page_?"
  }
}
```

<div id="parameters">
  ## パラメータ
</div>

<ResponseField name="redirect" type="boolean">
  ページが見つからない場合にホームページへ自動リダイレクトするかどうか。
  
  ホームページへリダイレクトするには `true` を設定します。

  カスタムの 404 ページを表示するには `false` を設定します。
</ResponseField>

<ResponseField name="title" type="string">
  404 エラーページのカスタムタイトル。既定の「Page not found」という見出しを置き換えます。
</ResponseField>

<ResponseField name="description" type="string">
  404 エラーページのカスタム説明。Markdown 形式をサポートします。
</ResponseField>
