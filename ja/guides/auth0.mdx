<div id="overview">
  ## 概要
</div>

このガイドでは、Mintlify のドキュメントで Auth0 を認証プロバイダーとして設定する方法を説明します。最後まで完了すると、ユーザーは Auth0 の認証情報を使ってあなたのドキュメントにログインできるようになります。

<Steps>
  <Step title="Auth0 で Regular Web Application を作成する">
    Auth0 のダッシュボードにログインし、**Applications** > **Applications** に移動します。**Create Application** をクリックし、アプリケーション名（例: “Mintlify”）を入力して、アプリケーションタイプに **Regular Web Applications** を選択します。最後に **Create** をクリックします。
    
    <Frame>![Auth0 で新しいアプリケーションを作成](/images/guides/auth0/auth0-create-application.png)</Frame>
  </Step>
  <Step title="クライアント情報を取得する">
    アプリケーション作成後、アプリケーション設定ページに移動します。ここで、OAuth 連携に必要な主要なクレデンシャルを確認できます。
    
    <Frame>![Auth0 のクライアント設定ページ](/images/guides/auth0/auth0clientsettings.png)</Frame>
    
    次の情報を控えておいてください:
    - **Domain**: Auth0 のテナントドメイン（例: `your-tenant.auth0.com`）
    - **Client ID**: アプリケーションの公開識別子
    - **Client Secret**: アプリケーションのシークレット（厳重に管理してください）
    
    次のステップで Mintlify を設定する際にこれらの値が必要になります。
  </Step>
  <Step title="Mintlify クライアントを設定する">
    Mintlify のダッシュボードで **Settings** > **Authentication** に移動します。認証方法として **OAuth** を選択すると、OAuth の設定フォームが表示されます。
    
    <Frame>![Mintlify の OAuth クライアント設定](/images/guides/auth0/mintlifyoauthclientsettings.png)</Frame>
    
    フォームに次の値を入力します:
    
    - **Authorization URL**: `https://YOUR_AUTH0_DOMAIN/authorize`（`YOUR_AUTH0_DOMAIN` はステップ 2 の Auth0 ドメインに置き換え）
    - **Client ID**: Auth0 アプリケーションの Client ID
    - **Client Secret**: Auth0 アプリケーションの Client Secret
    - **Scopes**: Auth0 でカスタムスコープを設定していない限り空のまま
    - **Token URL**: `https://YOUR_AUTH0_DOMAIN/oauth/token`（`YOUR_AUTH0_DOMAIN` は実際の Auth0 ドメインに置き換え）
    
    入力後、**Save changes** をクリックして OAuth 設定を保存します。
  </Step>
  <Step title="コールバック URL を設定する">
    OAuth フローが正しく動作するよう、Mintlify は Auth0 に認識させる必要がある固有のリダイレクト URL を生成します。
    
    Mintlify ダッシュボードの Authentication 設定からリダイレクト URL をコピーします:
    <Frame>![Mintlify のリダイレクト URL](/images/guides/auth0/mintlifyredirecturl.png)</Frame>
    
    Auth0 のアプリケーション設定ページに戻り、**Application URIs** セクションまでスクロールして、**Allowed Callback URLs** フィールドにリダイレクト URL を貼り付けます:
    <Frame>![Auth0 のリダイレクト URL 設定](/images/guides/auth0/auth0redirecturl.png)</Frame>
    
    ページ下部の **Save Changes** をクリックして設定を適用します。
  </Step>
</Steps>
