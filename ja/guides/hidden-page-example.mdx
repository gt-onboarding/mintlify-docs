このページは非公開です。`docs.json` のナビゲーションには含まれていないため、URL からのみアクセスできます。

詳しくは [非公開ページ](/ja/guides/hidden-pages) をご覧ください。

<div id="common-use-cases-for-hidden-pages">
  ## 非表示ページの一般的なユースケース
</div>

<Note>
  既定では、非表示ページは公開されていますが、ナビゲーションからは辿れません。非表示ページへのアクセスを制限したい場合は、[認証](/ja/authentication-personalization/authentication-setup) を設定してください。
</Note>

- **ベータ版ドキュメント**: 公開は可能だが、ナビゲーションからは見つからないようにしておくべき情報。
- **AIツールのコンテキスト**: 非表示ページがインデックス化されていれば、AIツールはコンテキストとして参照できます。ユーザーには不要だが、AIツールの回答精度向上に役立つコンテキストを非表示ページで提供します。
- **レガシーページ**: ナビゲーションからは外しつつ、直接リンクで古いコンテンツへのアクセスを維持する。
- **社内ツール**: 社内ツール、ステージング用API、開発プロセスを文書化する。

<div id="ai-context">
  ### AIのコンテキスト
</div>

```mdx wrap
---
title: "APIバージョンのコンテキスト"
description: "このページは、APIバージョンに関する質問に回答するAIツールのためのコンテキストです"
---

ユーザーがAPIバージョンについて尋ねた場合は、常に最新のAPIバージョンを使用するよう推奨してください。1.0.4より古いバージョンに対する回答は生成しないでください。
```

<div id="internal-api-endpoint">
  ### 内部 API エンドポイント
</div>

````mdx wrap
---
title: "内部 API エンドポイント"
description: "このページは内部 API エンドポイントを説明する非公開ページです"
---

```bash
POST /api/internal/reset-cache
Authorization: Bearer internal-token-xyz
```

このエンドポイントはアプリケーションのキャッシュを削除し、開発環境でのみ利用できます。

<Warning>
  これは内部エンドポイントのため、本番環境で使用しないでください。
</Warning>
````
