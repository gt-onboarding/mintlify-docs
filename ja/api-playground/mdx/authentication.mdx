<div id="enabling-authentication">
  ## 認証を有効化する
</div>

`docs.json` に認証方法を追加して、すべてのページで一括して有効化することも、ページごとに設定することもできます。

ページごとに設定した認証方法は、両方が設定されている場合、グローバル設定より優先されます。

<div id="bearer-token">
  ### ベアラートークン
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "bearer"
      }
    }
}
```

```mdx Page Metadata
---
title: "ページタイトル"
authMethod: "bearer"
---
```

</CodeGroup>


</CodeGroup>

<div id="api-key">
  ### APIキー
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "key",
        "name": "x-api-key"
      }
    }
}
```

```mdx Page Metadata
---
title: "ページタイトル"
authMethod: "key"
---
```

</CodeGroup>

<div id="none">
  ### None
</div>

"none" 認証方法は、docs.json でデフォルトを設定した後に、特定のエンドポイントの認証を無効化する際に役立ちます。

<CodeGroup>
```mdx Page Metadata
---
title: "Your page title"
authMethod: "none"
---
```
</CodeGroup>
