ウェブエディタの利用中によく発生する問題の解決策を紹介します。

<AccordionGroup>
  <Accordion title="公開後に変更が反映されない">
    **想定される原因:**

    * デプロイがまだ進行中
    * ブラウザのキャッシュの問題
    * ビルドまたはデプロイのエラー

    **解決策:**

    1. ダッシュボードでデプロイのステータスを確認してください。
    2. ブラウザをハードリロードしてください（<kbd>Ctrl</kbd> + <kbd>F5</kbd> または <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>）
    3. ブラウザのキャッシュをクリアしてください。
  </Accordion>

  <Accordion title="公開時の権限エラー">
    **想定される原因:**

    * Git リポジトリへの権限不足
    * 利用中の Git プロバイダーでの認証問題

    **解決策:**

    1. リポジトリへの適切なアクセス権があることを確認してください。
    2. Git 連携が正しく設定されているか確認してください。
    3. [Editor Permissions](/ja/advanced/dashboard/permissions) のドキュメントを参照してください。
  </Accordion>

  <Accordion title="エディタが読み込めない／読み込みが遅い">
    **想定される原因:**

    * ネットワーク接続の問題
    * 大規模なドキュメントリポジトリ

    **解決策:**

    1. インターネット接続を確認してください。
    2. ページを再読み込みしてください。
    3. 解決しない場合はサポートにお問い合わせください。
  </Accordion>

  <Accordion title="ファイルが読み込まれない、またはエラーが表示される">
    **想定される原因:**

    * ファイル内の MDX 構文エラー
    * ファイルの欠落または破損
    * 大きなファイルサイズによるタイムアウト

    **解決策:**

    1. ファイルの MDX 構文に誤りがないか確認してください。
    2. ファイルがリポジトリに存在することを確認してください。
  </Accordion>
</AccordionGroup>
