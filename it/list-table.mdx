<div id="lists">
  ## Liste
</div>

Le liste seguono la sintassi ufficiale del [Markdown](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Elenco ordinato
</div>

Per creare un elenco ordinato, aggiungi numeri seguiti da un punto prima degli elementi dell’elenco.

1. Primo elemento
2. Secondo elemento
3. Terzo elemento
4. Quarto elemento

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### Elenco non ordinato
</div>

Per creare un elenco non ordinato, aggiungi trattini (`-`), asterischi (`*`) o segni più (`+`) prima degli elementi dell’elenco.

- Primo elemento
- Secondo elemento
- Terzo elemento
- Quarto elemento

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### Elenco annidato
</div>

Rientra gli elementi dell’elenco per annidarli.

- Primo elemento
- Secondo elemento
  - Elemento aggiuntivo
  - Elemento aggiuntivo
- Terzo elemento

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## Tabelle
</div>

Le tabelle seguono la [sintassi Markdown](https://www.markdownguide.org/extended-syntax/#tables) ufficiale.

Per aggiungere una tabella, usa tre o più trattini (`---`) per creare l’intestazione di ciascuna colonna e usa le barre verticali (`|`) per separare le colonne. Per compatibilità, aggiungi anche una barra verticale all’inizio e alla fine di ogni riga.

| Proprietà | Descrizione                          |
| --------- | ------------------------------------ |
| Nome      | Nome completo dell’utente            |
| Età       | Età dichiarata                       |
| Iscrizione| Indica se l’utente si è unito alla community |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
