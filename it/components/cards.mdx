Usa le card per creare contenitori visivi per i contenuti. Le card sono contenitori flessibili che possono includere testo, icone, immagini e link.

<div id="basic-card">
  ## Scheda di base
</div>

<Card title="Titolo della scheda" icon="text" href="/it/components/columns">
  Ecco come usare una scheda con un'icona e un link. Facendo clic su questa scheda
  verrai reindirizzato alla pagina Columns.
</Card>

```mdx Card example
<Card title="Card title" icon="text" href="/components/columns">
  This is how you use a card with an icon and a link. Clicking on this card
  brings you to the Columns page.
</Card>
```

<div id="card-variations">
  ## Varianti di card
</div>

Le card supportano diverse opzioni di layout e di stile per adattarsi a varie esigenze di contenuto.

<div id="horizontal-layout">
  ### Layout orizzontale
</div>

Aggiungi la proprietà `horizontal` per visualizzare le schede in un layout orizzontale più compatto.

<Card title="Scheda orizzontale" icon="text" horizontal>
  Questo è un esempio di scheda orizzontale.
</Card>

```mdx Horizontal card example
<Card title="Horizontal card" icon="text" horizontal>
  This is an example of a horizontal card.
</Card>
```

<div id="image-cards">
  ### Schede con immagine
</div>

Aggiungi la proprietà `img` per mostrare un’immagine nella parte superiore della scheda.

<Card title="Image card" img="https://mintlify-assets.b-cdn.net/yosemite.jpg">
  Questo è un esempio di scheda con un’immagine.
</Card>

```mdx Image card example
<Card title="Image card" img="/images/card-with-image.png">
  Questo è un esempio di scheda con un’immagine.
</Card>
```

<div id="link-cards-with-custom-ctas">
  ### Schede di link con CTA personalizzate
</div>

Puoi personalizzare il testo della call to action e controllare se mostrare una freccia. Per impostazione predefinita, le frecce compaiono solo per i link esterni.

<Card title="Scheda di link" icon="link" href="/it/components/columns" arrow="true" cta="Clicca qui">
  Questo è un esempio di scheda con icona e link. Facendo clic sulla scheda verrai reindirizzato alla pagina Columns.
</Card>

```mdx Esempio di scheda di link
<Card
  title="Link card"
  icon="link"
  href="/components/columns"
  arrow="true"
  cta="Click here"
>
  This is an example of a card with an icon and a link. Clicking on this card brings you to the Columns page.
</Card>
```

<div id="grouping-cards">
  ## Raggruppare le schede
</div>

Usa il [componente Columns](/it/components/columns) per disporre più schede una accanto all’altra.

<Columns cols={2}>
  <Card title="First card" icon="panel-left-close">
    Questa è la prima scheda.
  </Card>

  <Card title="Second card" icon="panel-right-close">
    Questa è la seconda scheda.
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="First card" icon="panel-left-close">
    Questa è la prima scheda.
  </Card>
  <Card title="Second card" icon="panel-right-close">
    Questa è la seconda scheda.
  </Card>
</Columns>
```

<div id="properties">
  ## Proprietà
</div>

<ResponseField name="title" type="string" required>
  Il titolo visualizzato sulla scheda
</ResponseField>

<IconsOptional />

<ResponseField name="color" type="string">
  Colore dell'icona in formato esadecimale (es. `#FF6B6B`).
</ResponseField>

<ResponseField name="href" type="string">
  URL a cui andare quando si fa clic sulla scheda.
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  Visualizza la scheda in un layout orizzontale compatto.
</ResponseField>

<ResponseField name="img" type="string">
  URL o percorso locale dell’immagine visualizzata nella parte superiore della scheda.
</ResponseField>

<ResponseField name="cta" type="string">
  Testo personalizzato per il pulsante di azione.
</ResponseField>

<ResponseField name="arrow" type="boolean">
  Mostra o nasconde l’icona della freccia del link.
</ResponseField>
