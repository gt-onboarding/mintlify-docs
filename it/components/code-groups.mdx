Usa il componente `CodeGroup` per visualizzare più blocchi di codice in un'interfaccia a schede, consentendo agli utenti di confrontare le implementazioni tra diversi linguaggi di programmazione o di vedere approcci alternativi per lo stesso task.

<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>

<div id="creating-code-groups">
  ## Creazione di gruppi di codice
</div>

Per creare un gruppo di codice, avvolgi più blocchi di codice con i tag `<CodeGroup>`. Ogni blocco deve includere un titolo, che diventerà l’etichetta della scheda.

````mdx
<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>
````

<div id="language-dropdown">
  ## Menu a discesa della lingua
</div>

Puoi sostituire le schede in un gruppo di codice con un menu a discesa per cambiare lingua usando la prop `dropdown`.
