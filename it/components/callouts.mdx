---
title: "Riquadri informativi"
description: "Usa i riquadri informativi per aggiungere contesto accattivante ai tuoi contenuti"
---

I riquadri informativi possono essere formattati come Nota, Avviso, Info, Suggerimento, Conferma, Pericolo, oppure puoi creare un riquadro personalizzato:

<Note>Questo aggiunge una nota al contenuto</Note>

```mdx
<Note>Questo aggiunge una nota al contenuto</Note>
```

<Warning>Questo genera un avviso da tenere d’occhio</Warning>

```mdx
<Warning>Questo genera un avviso da tenere presente</Warning>
```

<Info>Questo mette in evidenza informazioni importanti</Info>

```mdx
<Info>Evidenzia informazioni importanti</Info>
```

<Tip>Questo fornisce un suggerimento utile</Tip>

```mdx
<Tip>Questo suggerisce un suggerimento utile</Tip>
```

<Check>Questo ci porta a uno stato con segno di spunta</Check>

```mdx
<Check>Questo ci fornisce uno stato verificato</Check>
```

<Danger>Questo è un avviso di pericolo</Danger>

```mdx
<Danger>Questo è un avviso di pericolo</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Questo è un avviso personalizzato</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Questo è un callout personalizzato</Callout>
```
