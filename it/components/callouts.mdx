---
title: "Callouts"
description: "Usa i callout per aggiungere contesto di impatto ai tuoi contenuti"
icon: "info"
---

I callout possono essere formattati come Nota, Avviso, Info, Suggerimento, Verifica, Pericolo, oppure puoi creare un callout personalizzato:

<Note>Questo aggiunge una nota al contenuto</Note>

```mdx
<Note>Questo aggiunge una nota al contenuto</Note>
```

<Warning>Questo segnala un avviso a cui prestare attenzione</Warning>

```mdx
<Warning>Questo segnala un avviso a cui prestare attenzione</Warning>
```

<Info>Questo mette in evidenza informazioni importanti</Info>

```mdx
<Info>Questo mette in evidenza informazioni importanti</Info>
```

<Tip>Questo fornisce un suggerimento utile</Tip>

```mdx
<Tip>Questo fornisce un suggerimento utile</Tip>
```

<Check>Questo indica uno stato verificato</Check>

```mdx
<Check>Questo indica uno stato verificato</Check>
```

<Danger>Questo è un callout di pericolo</Danger>

```mdx
<Danger>Questo è un callout di pericolo</Danger>
```

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Questo è un callout personalizzato</Callout>
```
