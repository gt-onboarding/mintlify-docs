---
title: "Riquadri informativi"
description: "Usa i riquadri informativi per aggiungere un contesto che cattura l’attenzione"
icon: "info"
---

I riquadri informativi possono essere formattati come Nota, Avviso, Info, Suggerimento, Conferma, Pericolo, oppure puoi creare un riquadro personalizzato:

<Note>Questo aggiunge una nota nel contenuto</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>Questo segnala un avviso a cui prestare attenzione</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Questo mette in evidenza informazioni importanti</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>Questo propone un suggerimento utile</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Questo indica uno stato confermato</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>Questo è un riquadro di pericolo</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">Questo è un riquadro personalizzato</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
