---
title: "Tabs"
description: "Alterna i contenuti con il componente Tabs"
icon: "panel-top"
---

Usa le tab per organizzare i contenuti in più pannelli tra cui gli utenti possono scegliere. Puoi aggiungere qualsiasi numero di tab e includere altri componenti all'interno di ciascuna tab.

<Tabs>
  <Tab title="First tab">
    ☝️ Benvenuto nel contenuto visibile solo all'interno della prima tab.

    Puoi aggiungere qualsiasi numero di componenti all'interno delle tab. Per esempio, un blocco di codice:
    ```java HelloWorld.java
      class HelloWorld {
          public static void main(String[] args) {
              System.out.println("Hello, World!");
          }
      }
    ```
  </Tab>
  <Tab title="Second tab" icon="leaf">
    ✌️ Ecco contenuti presenti solo all'interno della seconda tab.

    Questa ha un'icona <Icon icon="leaf" />!
  </Tab>
  <Tab title="Third tab">
    💪 Ecco contenuti presenti solo all'interno della terza tab.
  </Tab>
</Tabs>

<div id="properties">
  ## Proprietà
</div>

<ResponseField name="title" type="string" required>
  Il titolo della scheda. I titoli brevi sono più facili da individuare nella navigazione.
</ResponseField>

<ResponseField name="icon" type="string">
  Un'icona [Font Awesome](https://fontawesome.com/icons), un'icona [Lucide](https://lucide.dev/icons), un URL di un'icona o un percorso relativo a un'icona.
</ResponseField>

<ResponseField name="iconType" type="string">
  Solo per le icone Font Awesome: uno tra `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`.
</ResponseField>
