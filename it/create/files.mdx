---
title: "File"
description: "Distribuisci asset statici dalla tua documentazione"
---

Mintlify distribuisce automaticamente gli asset statici dal repository della tua documentazione nel percorso appropriato sul tuo dominio. Ad esempio, se nel tuo repo è presente `/images/my-logo.png`, il file immagine sarà disponibile all’indirizzo `https://docs.your-project.com/images/my-logo.png`.

Puoi mettere a disposizione degli utenti qualsiasi tipo di file supportato, incluse le specifiche OpenAPI, immagini, video e altro.

Le dimensioni dei file devono essere inferiori a 20 MB per le immagini e 100 MB per gli altri tipi di file.

<div id="file-organization">
  ## Organizzazione dei file
</div>

Organizza i file in cartelle per mantenere il repository facile da esplorare:

```text
/your-project
  |- docs.json
  |- images/
    |- logo.png
    |- screenshots/
      |- dashboard.png
  |- assets/
    |- whitepaper.pdf
    |- demo-video.mp4
```

I file vengono serviti dalla radice del tuo dominio, quindi la struttura del repository corrisponde direttamente alla struttura degli URL. Nell’esempio precedente, `assets/whitepaper.pdf` sarebbe disponibile all’indirizzo `https://docs.your-project.com/assets/whitepaper.pdf`.
