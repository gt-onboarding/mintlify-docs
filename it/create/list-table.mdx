<div id="lists">
  ## Liste
</div>

Le liste seguono la sintassi Markdown ufficiale: [Markdown syntax](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Elenco numerato
</div>

Per creare un elenco numerato, aggiungi numeri seguiti da un punto prima degli elementi.

1. Primo elemento
2. Secondo elemento
3. Terzo elemento
4. Quarto elemento

```mdx
1. Primo elemento
2. Secondo elemento
3. Terzo elemento
4. Quarto elemento
```


<div id="unordered-list">
  ### Elenco puntato
</div>

Per creare un elenco puntato, aggiungi trattini (`-`), asterischi (`*`) o segni più (`+`) prima degli elementi.

* Primo elemento
* Secondo elemento
* Terzo elemento
* Quarto elemento

```mdx
- Primo elemento
- Secondo elemento
- Terzo elemento
- Quarto elemento
```


<div id="nested-list">
  ### Elenco annidato
</div>

Rientra gli elementi dell’elenco per creare un annidamento.

* Primo elemento
* Secondo elemento
  * Elemento aggiuntivo
  * Elemento aggiuntivo
* Terzo elemento

```mdx
- Primo elemento
- Secondo elemento
  - Elemento aggiuntivo
  - Elemento aggiuntivo
- Terzo elemento
```


<div id="tables">
  ## Tabelle
</div>

Le tabelle seguono la [sintassi Markdown](https://www.markdownguide.org/extended-syntax/#tables) ufficiale.

Per aggiungere una tabella, usa tre o più trattini (`---`) per creare l’intestazione di ogni colonna e usa le barre verticali (`|`) per separare le colonne. Per compatibilità, aggiungi anche una barra verticale a entrambe le estremità della riga.

| Proprietà | Descrizione                         |
| --------- | ----------------------------------- |
| Nome      | Nome completo dell’utente           |
| Età       | Età riportata                       |
| Iscrizione| Indica se l’utente ha aderito alla community |

```mdx
| Proprietà | Descrizione                                 |
| --------- | ------------------------------------------- |
| Name      | Nome completo dell'utente                   |
| Age       | Età dichiarata                              |
| Joined    | Se l'utente ha aderito alla community       |
```
