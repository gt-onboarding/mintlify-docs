---
title: "Ruoli"
description: "Gestisci l'accesso alla dashboard tramite ruoli."
---

<Info>
  La funzionalità RBAC è disponibile nel [piano Enterprise](https://mintlify.com/pricing?ref=rbac).
</Info>

Mintlify offre due livelli di accesso alla dashboard: Editor e Admin.

Di seguito sono elencate le azioni consentite solo al ruolo Admin:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Aggiornare i ruoli utente   |   ❌   |  ✅   |
| Eliminare utenti            |   ❌   |  ✅   |
| Invitare utenti admin       |   ❌   |  ✅   |
| Gestire e aggiornare la fatturazione |   ❌   |  ✅   |
| Aggiornare il dominio personalizzato |   ❌   |  ✅   |
| Aggiornare l'origine Git    |   ❌   |  ✅   |
| Eliminare l'organizzazione  |   ❌   |  ✅   |

Le altre azioni nella dashboard sono disponibili per entrambi i ruoli.

Puoi invitare quanti admin desideri, ma consigliamo di limitare l'accesso admin agli utenti che ne hanno effettivamente bisogno.
