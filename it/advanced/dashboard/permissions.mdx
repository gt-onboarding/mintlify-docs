---
title: "Permessi dell'editor"
description: "Consenti a più membri del tuo team di aggiornare la documentazione"
---

Un editor ha accesso alla dashboard e all'Editor web.

Chiunque può contribuire alla documentazione lavorando in locale e inviando modifiche al repository, ma ci sono differenze fondamentali nel modo in cui le modifiche vengono pubblicate:

* **Modifiche dell'editor**: quando un editor pubblica tramite l'Editor web o effettua il merge di una pull request nel repository della documentazione, le modifiche vengono distribuite automaticamente al sito in produzione.
* **Modifiche dei non editor**: quando una persona che non è editor effettua il merge di una pull request nel repository, devi avviare manualmente una distribuzione dalla dashboard affinché tali modifiche compaiano sul sito in produzione.

<div id="add-editors">
  ## Aggiungi editor
</div>

Per impostazione predefinita, il membro del team che ha creato la tua organizzazione Mintlify ha accesso come editor. Aggiungi altri editor dalla pagina [Members](https://dashboard.mintlify.com/settings/organization/members) della tua dashboard.

I posti editor vengono fatturati in base all’utilizzo e puoi avere tutti gli editor di cui hai bisogno. Consulta la nostra [pagina dei prezzi](https://mintlify.com/pricing) per ulteriori dettagli.
