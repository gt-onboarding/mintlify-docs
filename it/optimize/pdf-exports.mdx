Puoi esportare la documentazione come un unico file PDF per la consultazione offline. Il PDF conterrà tutti i contenuti della documentazione, incluse immagini e link, e un sommario navigabile.


<div id="exporting-a-pdf">
  ## Esportare un PDF
</div>

1. Vai alla pagina [General](https://dashboard.mintlify.com/settings/deployment/general) nella sezione Project Settings della dashboard.
2. Seleziona il pulsante **Export all content**.
3. Facoltativamente, personalizza le opzioni di esportazione:
    - **Page format**: scegli il formato pagina del PDF.
    - **Scale percentage**: regola la scala del PDF.
    - **Include footer**: includi un piè di pagina con il numero di pagina e il totale delle pagine.
4. Seleziona **Export** per avviare l’esportazione.
5. Al termine dell’esportazione, riceverai un’email con un link per il download.
6. Clicca sul link per scaricare il file PDF.
