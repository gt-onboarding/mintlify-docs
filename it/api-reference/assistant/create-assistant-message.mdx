<div id="rate-limits">
  ## Limitazioni di utilizzo
</div>

L'API dell’Assistente presenta i seguenti limiti:

- 10.000 utilizzi per chiave al mese
- 10.000 richieste per organizzazione Mintlify all’ora
- 10.000 richieste per IP al giorno

<div id="suggested-usage">
  ## Utilizzo consigliato
</div>

Per risultati ottimali, usa l’[hook useChat di ai-sdk](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat) per inviare le richieste e gestire le risposte.

Puoi impostare `fp`, `threadId` e `filter` nel campo `body` dell’oggetto options passato all’hook.
