title: "Front"
---

Aggiungi quanto segue al file `docs.json` per inserire il widget [Front Chat](https://front.com).

<CodeGroup>

```json Opzioni di integrazione in docs.json
"integrations": {
    "frontchat": "CHAT_ID"
}
```

```json Esempio
"integrations": {
    "frontchat": "1365d046d7c023e9b030ce90d02d093a"
}
```

</CodeGroup>
