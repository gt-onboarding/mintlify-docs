<div id="enabling-data-privacy-integrations">
  ## Abilitazione delle integrazioni per la protezione dei dati
</div>

Puoi aggiungere piattaforme per la protezione dei dati alla tua documentazione. Aggiungi il campo `integrations` al file `docs.json` con i relativi script.

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

<div id="cookie-consent-and-disabling-telemetry">
  ## Consenso ai cookie e disattivazione della telemetria
</div>

Se devi verificare se un utente ha già acconsentito ai cookie ai fini della conformità al GDPR, puoi specificare una chiave e un valore del local storage sotto `cookies`:

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

Se questi valori sono impostati, il local storage verrà controllato per verificare se l’utente ha acconsentito ai cookie. In caso contrario, la telemetria verrà disattivata.

Se desideri disattivare la telemetria per tutti gli utenti, puoi aggiungere quanto segue al file `docs.json`:

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
