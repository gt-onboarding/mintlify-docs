---
title: "Pirsch"
---

Aggiungi quanto segue al file `docs.json` per inviare i dati di analisi a Pirsch.

Puoi ottenere l’ID del sito da Settings > Developer > Identification Code.

<CodeGroup>

```json Opzioni di analisi in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Esempio
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
