---
title: "Fathom"
---

Aggiungi quanto segue al file `docs.json` per inviare i dati di analisi a Fathom.

Puoi ottenere il `siteId` dalle impostazioni dello script.

<CodeGroup>

```json Opzioni di analisi in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Esempio
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
