---
title: "Google Tag Manager"
---

Aggiungi l’ID del tag al file `docs.json` e inseriremo lo script di Google Tag Manager in tutte le pagine.

Se necessario, sei tu a dover configurare i banner per il consenso ai cookie tramite Google Tag Manager.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
