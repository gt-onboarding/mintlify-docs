title: "Koala"
---

Aggiungi quanto segue al file `docs.json` per inviare i dati di analytics a Koala.

<CodeGroup>

```json Opzioni di analytics in docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Esempio
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
