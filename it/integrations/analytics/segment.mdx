---
title: "Segment"
---

Aggiungi la tua write key di Segment al file `docs.json` per inviare i dati di analitica a Segment.

<CodeGroup>

```json Opzioni di Analytics in docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Esempio
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
