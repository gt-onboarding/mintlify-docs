---
title: "Hightouch"
---

Aggiungi quanto segue al file `docs.json` per inviare dati di analytics a Hightouch.

<Info>

  Non includere `https://` per `apiHost`.

</Info>

<CodeGroup>

```json Opzioni di analytics in docs.json
"integrations": {
    "hightouch": {
        "writeKey": "obbligatorio"
        "apiHost": "facoltativo"
    }
}
```

```json Esempio
"integrations": {
    "hightouch": {
        "writeKey": "9132c81do371p76sr11da0075469b54f77649c9a067dp0303p56q0q64n072336"
        "apiHost": "us-east-1.hightouch-events.com" # facoltativo, predefinito: `us-east-1.hightouch-events.com`
    }
}
```

</CodeGroup>
