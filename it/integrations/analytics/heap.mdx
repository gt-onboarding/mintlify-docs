title: "Heap"

Aggiungi quanto segue al file `docs.json` per inviare le analitiche a Heap.

<CodeGroup>

```json Opzioni di analitica in docs.json
"integrations": {
    "heap": {
        "appId": "obbligatorio"
    }
}
```

```json Esempio
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```
