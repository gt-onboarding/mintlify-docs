---
title: "Heap"
---

Aggiungi quanto segue al file `docs.json` per inviare i dati di analisi a Heap.

<CodeGroup>

```json Opzioni di Analytics in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Esempio
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
