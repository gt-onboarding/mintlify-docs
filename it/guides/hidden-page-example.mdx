Questa pagina è nascosta! Non è inclusa nella navigazione di `docs.json`, quindi puoi accedervi solo tramite il suo URL.

Consulta [Pagine nascoste](/it/guides/hidden-pages) per ulteriori informazioni.

## Casi d&#39;uso comuni per le pagine nascoste

<Note>
  Per impostazione predefinita, le pagine nascoste sono accessibili pubblicamente, ma non sono individuabili tramite la navigazione. Se vuoi limitare l&#39;accesso a una pagina nascosta, devi configurare l&#39;[autenticazione](/it/authentication-personalization/authentication-setup).
</Note>

* **Documentazione beta**: Informazioni che possono essere pubbliche, ma non dovrebbero essere individuabili tramite la navigazione.
* **Contesto per strumenti di AI**: Se le pagine nascoste sono indicizzate, gli strumenti di AI possono usarle come riferimento per il contesto. Usa le pagine nascoste per contenuti non rilevanti per gli utenti, ma utili agli strumenti di AI per fornire risposte più accurate.
* **Pagine legacy**: Mantieni i vecchi contenuti accessibili tramite link diretti rimuovendoli dai menu di navigazione.
* **Strumenti interni**: Documenta strumenti interni, API di staging o processi di sviluppo.
