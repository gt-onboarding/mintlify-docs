Questa pagina è nascosta! Non è inclusa nella navigazione di `docs.json`, quindi puoi accedervi solo tramite il suo URL.

Consulta [Pagine nascoste](/it/organize/hidden-pages) per ulteriori informazioni.

## Casi d&#39;uso comuni per le pagine nascoste

<Note>
  Per impostazione predefinita, le pagine nascoste sono accessibili pubblicamente, ma non sono individuabili tramite la navigazione. Se vuoi limitarne l’accesso, devi configurare l’[autenticazione](/it/deploy/authentication-setup).
</Note>

* **Documentazione beta**: Informazioni che possono essere pubbliche, ma che non dovrebbero essere individuabili tramite la navigazione.
* **Contesto per strumenti di AI**: Se le pagine nascoste sono indicizzate, gli strumenti di AI possono utilizzarle come contesto. Usa le pagine nascoste per contenuti non rilevanti per gli utenti, ma che possono aiutare gli strumenti di AI a fornire risposte più accurate.
* **Pagine legacy**: Mantieni i contenuti obsoleti accessibili tramite link diretti rimuovendoli dai menu di navigazione.
* **Strumenti interni**: Documenta strumenti interni, API di staging o processi di sviluppo.

````mdx wrap

```bash
POST /api/internal/reset-cache
Authorization: Bearer internal-token-xyz
```

Questo endpoint svuota la cache dell'applicazione ed è disponibile solo negli ambienti di sviluppo.

<Warning>
  Questo è un endpoint interno e non deve mai essere utilizzato in produzione.
</Warning>
````
