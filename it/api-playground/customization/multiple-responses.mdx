---
title: "Risposte multiple"
description: "Mostra variazioni di risposta per lo stesso endpoint"
---

Se la tua API restituisce risposte diverse in base ai parametri di input, al contesto dell’utente o ad altre condizioni della richiesta, puoi documentare più esempi di risposta con la proprietà `examples`.

Questa proprietà può essere aggiunta a qualsiasi risposta e segue il seguente schema.

```yaml
responses:
  "200":
    description: Risposta riuscita
    content:
      application/json:
        schema:
          $ref: "#/components/schemas/YourResponseSchema"
        examples:
          us:
            summary: Risposta per Stati Uniti
            value:
              countryCode: "US"
              currencyCode: "USD"
              taxRate: 0.0825
          gb:
            summary: Risposta per Regno Unito
            value:
              countryCode: "GB"
              currencyCode: "GBP"
              taxRate: 0.20
```
