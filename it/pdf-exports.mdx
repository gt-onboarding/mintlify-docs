<Info>
L’esportazione in PDF è disponibile nei [piani Personalizzati](https://mintlify.com/pricing).
</Info>

Puoi esportare la documentazione in un unico file PDF per la consultazione offline. Il PDF includerà tutti i contenuti della documentazione, comprese immagini e collegamenti, e un indice navigabile.

<div id="exporting-a-pdf">
  ## Esportazione di un PDF
</div>

1. Vai alla pagina [General](https://dashboard.mintlify.com/settings/deployment/general) nella sezione Project Settings della tua dashboard.
2. Seleziona il pulsante **Export all content**.
3. Facoltativamente, personalizza le opzioni di esportazione:
    - **Page format**: Scegli il formato pagina del PDF.
    - **Scale percentage**: Regola la percentuale di scala del PDF.
    - **Include footer**: Includi un piè di pagina con il numero di pagina e il totale delle pagine.
4. Seleziona **Export** per avviare il processo di esportazione.
5. Al termine dell’esportazione, riceverai un’email con un link per il download.
6. Fai clic sul link per scaricare il file PDF.
