Mintlify utilizza una GitHub App per sincronizzare automaticamente la documentazione con il repository GitHub.

<div id="installing-the-github-app">
  ## Installazione dell’app GitHub
</div>

Installa l’app GitHub di Mintlify sull’account o sull’organizzazione GitHub che possiede il repository della documentazione. Per installare l’app devi essere proprietario dell’organizzazione o avere i permessi di amministratore nel repository. Se non disponi dei permessi necessari, il proprietario del repository dovrà approvare la richiesta di installazione.

Installa l’app GitHub di Mintlify dal tuo [dashboard](https://dashboard.mintlify.com/settings/organization/github-app).

<Tip>
Consigliamo di concedere l’accesso solo al repository in cui è ospitata la documentazione.
</Tip>

<div id="permissions">
  ## Autorizzazioni
</div>

Quando installi l'app GitHub, ti verrà richiesto di concedere le seguenti autorizzazioni:

Autorizzazioni di sola lettura:

* `metadata`: Informazioni di base sul repository

Autorizzazioni di lettura e scrittura:

* `checks`: Creare controlli di stato sulle pull request
* `code`: Leggere le modifiche ai file quando esegui commit nel branch della documentazione
* `deployments`: Generare distribuzioni di anteprima per le pull request
* `pull requests`: Creare branch e pull request dal web editor
* `workflows`: Gestire GitHub Actions per le distribuzioni della documentazione

<Info>
L'app accede solo ai repository a cui le concedi esplicitamente l'accesso. Se hai abilitato regole di protezione dei branch, l'app non può eseguire push direttamente sui branch protetti.
</Info>

<div id="managing-repository-access">
  ## Gestione dell'accesso ai repository
</div>

Quando installi la nostra app GitHub, puoi concedere l'accesso a tutti i tuoi repository oppure solo a quelli specifici. Ti consigliamo di autorizzare solo i repository in cui si trova la tua documentazione. Puoi modificare questa selezione in qualsiasi momento dalle [impostazioni dell'app GitHub](https://github.com/apps/mintlify/installations/new).

<div id="configuring-docs-source">
  ## Configurare l'origine della documentazione
</div>

Modifica l'organizzazione, il repository o il branch da cui viene generata la documentazione nella sezione [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) della dashboard.

<div id="troubleshooting">
  ## Risoluzione dei problemi
</div>

Se riscontri problemi con l'app GitHub, il ripristino della connessione può risolvere la maggior parte dei casi.

<Steps>
  <Step title="Disinstalla l'app Mintlify da GitHub.">
    1. In GitHub, vai su [installations](https://github.com/settings/installations) e seleziona **Configure** accanto all'app Mintlify. Scorri verso il basso e seleziona **Uninstall**.
    2. Vai su [Authorized GitHub Apps](https://github.com/settings/apps/authorizations) e seleziona **Revoke** accanto all'app Mintlify.
  </Step>
  <Step title="Reinstalla l'app Mintlify.">
    1. Nel dashboard di Mintlify, vai su [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) e installa l'app GitHub.
    2. Autorizza il tuo account nella sezione [My Profile](https://dashboard.mintlify.com/settings/account) del dashboard.
  </Step>
</Steps>
