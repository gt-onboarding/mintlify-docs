Il [file llms.txt](https://llmstxt.org) è uno standard del settore che aiuta gli LLM a indicizzare i contenuti in modo più efficiente, in modo analogo a come una sitemap aiuta i motori di ricerca. Gli strumenti di IA possono usare questo file per comprendere la struttura della tua documentazione e trovare contenuti pertinenti alle richieste degli utenti.

Mintlify ospita automaticamente un file `llms.txt` nella root del tuo progetto che elenca tutte le pagine disponibili nella tua documentazione. Questo file è sempre aggiornato e non richiede alcuna manutenzione. In alternativa, puoi aggiungere un file `llms.txt` personalizzato nella root del tuo progetto.

Per visualizzare il tuo `llms.txt`, aggiungi `/llms.txt` all’URL del tuo sito di documentazione.

<div id="llmstxt-structure">
  ## Struttura di llms.txt
</div>

Un file `llms.txt` è un semplice file Markdown che contiene:

- **Titolo del sito** come intestazione H1.
- **Sezioni di contenuto strutturate** con link e una descrizione di ciascuna pagina della documentazione.

```mdx Example llms.txt
# Example product docs

## Guides
- [Getting started](https://example.com/docs/start): Intro guide
- [Install](https://example.com/docs/install): Setup steps

## Reference
- [API](https://example.com/docs/api): Endpoint list and usage
```

Questo approccio strutturato consente agli LLM di elaborare in modo efficiente la documentazione a livello generale e di individuare rapidamente i contenuti pertinenti per le richieste degli utenti, migliorando accuratezza e velocità delle ricerche nella documentazione assistite dall’IA.

<div id="llms-fulltxt">
  ## llms-full.txt
</div>

Il file `llms-full.txt` unisce l’intero sito di documentazione in un unico file come contesto per gli strumenti di IA ed è indicizzato dal traffico degli LLM.

Mintlify ospita automaticamente un file `llms-full.txt` nella root del tuo progetto. Visualizza il tuo `llms-full.txt` aggiungendo `/llms-full.txt` all’URL del tuo sito di documentazione.

<PreviewButton href="https://mintlify.com/docs/llms-full.txt">Apri llms-full.txt per questo sito.</PreviewButton>

<div id="custom-files">
  ## File personalizzati
</div>

Per aggiungere un file personalizzato `llms.txt` o `llms-full.txt`, crea un file `llms.txt` o `llms-full.txt` nella radice del progetto. L’aggiunta di un file personalizzato sostituirà il file generato automaticamente con lo stesso nome. Se elimini un file personalizzato, verrà nuovamente utilizzato il file predefinito.

Il file personalizzato `llms.txt` o `llms-full.txt` deve includere il titolo del sito come intestazione H1. Gli altri contenuti sono facoltativi. Consulta [Format](https://llmstxt.org/#format) nella specifica di `llms.txt` per ulteriori informazioni sulle sezioni facoltative e sulle best practice.
