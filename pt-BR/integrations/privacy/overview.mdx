<div id="enabling-data-privacy-integrations">
  ## Ativar integrações de privacidade de dados
</div>

Você pode adicionar plataformas de privacidade de dados à sua documentação. Inclua o campo `integrations` no arquivo `docs.json` com os respectivos scripts.

```json
  "integrations": {
    "osano": "SOURCE"
  }
```

<div id="cookie-consent-and-disabling-telemetry">
  ## Consentimento de cookies e desativação de telemetria
</div>

Se você precisar verificar se um usuário já consentiu com cookies para conformidade com o GDPR, poderá especificar uma chave e um valor no armazenamento local em `cookies`:

```json
  "integrations": {
    "cookies": {
      "key": "LOCAL STORAGE KEY",
      "value": "LOCAL STORAGE VALUE"
    }
  }
```

Se esses valores estiverem definidos, o armazenamento local será verificado para confirmar se o usuário consentiu com cookies. Caso não tenha, a telemetria será desativada.

Se quiser desativar a telemetria para todos os usuários, adicione o seguinte ao arquivo `docs.json`:

```json
  "integrations": {
    "telemetry": {
      "enabled": false
    }
  }
```
