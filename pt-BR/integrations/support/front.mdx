title: "Front"
---

Adicione o seguinte ao arquivo `docs.json` para incluir um widget do [Front Chat](https://front.com).

<CodeGroup>

```json Opções de integração em docs.json
"integrations": {
    "frontchat": "CHAT_ID"
}
```

```json Exemplo
"integrations": {
    "frontchat": "1365d046d7c023e9b030ce90d02d093a"
}
```

</CodeGroup>
