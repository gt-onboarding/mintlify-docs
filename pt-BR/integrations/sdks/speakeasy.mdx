Trechos de código gerados automaticamente a partir dos SDKs da Speakeasy podem ser integrados diretamente à documentação de referência de API da Mintlify. Esses trechos de uso de SDK são exibidos no [playground interativo](/pt-BR/api-playground/overview) de sites de documentação com Mintlify.

<Frame>
  ![Um Playground de API da Mintlify com trechos de código da Speakeasy.](/images/speakeasy/mintlify-with-speakeasy-openapi.png)
</Frame>

<div id="prerequisites">
  ## Pré-requisitos
</div>

Para integrar a Mintlify à Speakeasy, você precisará do seguinte:

- Um [repositório de documentação da Mintlify](/pt-BR/quickstart#creating-the-repository).
  - Um SDK gerado pela Speakeasy com um [URL de amostras de código automatizadas](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls) configurado.

<div id="setting-up-the-integration">
  ## Configurando a integração
</div>

Para integrar o Speakeasy ao Mintlify, você precisa obter a URL pública da especificação combinada da API no registry e atualizar o arquivo de configuração `docs.json`.

<div id="get-the-apis-combined-spec-public-url-from-the-registry">
  ### Obtenha a URL pública da especificação combinada da API no registro
</div>

Acesse seu [Speakeasy Dashboard](https://app.speakeasy.com) e abra a guia **API Registry**. Abra a entrada `*-with-code-samples` da API.

<Frame>
  ![Captura de tela da página Speakeasy API Registry. A guia API Registry está destacada com um quadrado vermelho e o número 1, e a entrada da API está destacada com um quadrado vermelho e o número 2.](/images/speakeasy/openapi-registry-and-combined-spec.png)
</Frame>

<Note>
  Se a entrada não estiver marcada como **Combined Spec**, verifique se a API tem uma [URL automática de exemplos de código](https://www.speakeasy.com/docs/code-samples/automated-code-sample-urls) configurada.
</Note>

Na página da entrada do registro, copie a URL pública fornecida.

<Frame>
  ![Captura de tela mostrando a entrada do registro da especificação combinada com a função de copiar URL destacada com um quadrado vermelho.](/images/speakeasy/copy-combined-spec-url.png)
</Frame>

<div id="update-your-docsjson-configuration-file">
  ### Atualize seu arquivo de configuração `docs.json`
</div>

Adicione a URL da especificação combinada a uma seção de **Anchors** ou **Tabs** no seu arquivo `docs.json`.

Adicione a URL da especificação combinada a uma âncora atualizando o campo `anchor` no seu arquivo `docs.json` da seguinte forma:

```json docs.json
{
  "anchors": [
    {
      "name": "API Reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL",
      "url": "api-reference",
      "icon": "square-terminal"
    }
  ]
}
```

Adicione a URL da especificação combinada a uma aba atualizando o campo `tab` no arquivo `docs.json` da seguinte forma:

```json docs.json
{
  "tabs": [
    {
      "name": "API Reference",
      "url": "api-reference",
      // !mark
      "openapi": "SPEAKEASY_COMBINED_SPEC_URL"
    }
  ]
}
```

Trechos de código gerados pelo Speakeasy agora podem ser visualizados na sua documentação de API e usados no Playground de API.
