---
title: "Google Tag Manager"
---

Adicione seu ID de tag ao arquivo `docs.json` e injetaremos o script do Google Tag Manager em todas as suas páginas.

Você é responsável por configurar banners de consentimento de cookies no Google Tag Manager, se necessário.

<CodeGroup>

```json Opções de análise em docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Exemplo
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
