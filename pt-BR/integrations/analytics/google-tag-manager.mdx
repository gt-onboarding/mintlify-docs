---
title: "Google Tag Manager"
---

Adicione o ID da sua tag ao arquivo `docs.json` e injetaremos o script do Google Tag Manager em todas as suas páginas.

Você é responsável por configurar banners de consentimento de cookies com o Google Tag Manager, caso necessário.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
