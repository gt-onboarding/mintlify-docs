---
title: "Fathom"
---

Adicione o seguinte ao arquivo `docs.json` para enviar dados de analytics ao Fathom.

Você pode obter o `siteId` nas configurações do script.

<CodeGroup>

```json Opções de analytics em docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Exemplo
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
