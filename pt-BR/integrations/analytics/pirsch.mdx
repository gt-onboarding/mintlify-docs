---
title: "Pirsch"
---

Adicione o seguinte ao seu arquivo `docs.json` para enviar dados de analytics ao Pirsch.

Você pode obter o ID do seu site em Settings > Developer > Identification Code.

<CodeGroup>

```json Opções de analytics em docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Exemplo
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
