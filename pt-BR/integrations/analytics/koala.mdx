---
title: "Koala"
---

Adicione o seguinte ao arquivo `docs.json` para enviar dados de analytics para o Koala.

<CodeGroup>

```json Opções de analytics em docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Exemplo
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
