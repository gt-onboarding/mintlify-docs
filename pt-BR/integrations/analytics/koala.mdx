---
title: "Koala"
---

Adicione o seguinte ao seu arquivo `docs.json` para enviar dados de analytics ao Koala.

<CodeGroup>

```json Opções de analytics no docs.json
"integrations": {
    "koala": {
        "publicApiKey": "required"
    }
}
```

```json Exemplo
"integrations": {
    "koala": {
        "publicApiKey": "pk_1a1882"
    }
}
```

</CodeGroup>
