Você precisará gerar uma nova propriedade <Tooltip tip="Google Analytics 4">GA4</Tooltip> para usar com a Mintlify. Os dados coletados serão enviados para o mesmo projeto que os seus outros dados do Google Analytics.

Se você estiver usando a versão antiga do Google Analytics, o Universal Analytics, ainda poderá gerar uma propriedade <Tooltip tip="Google Analytics 4">GA4</Tooltip>. Os dados do <Tooltip tip="Google Analytics 4">GA4</Tooltip> são um pouco diferentes dos do UA, mas ainda são coletados no mesmo projeto.

<div id="how-to-connect-ga4-to-mintlify">
  ## Como conectar o GA4 ao Mintlify
</div>

<div id="create-a-web-stream">
  ### Criar um fluxo da Web
</div>

Você precisará criar um fluxo da web para obter o Measurement ID e inseri-lo no Mintlify.

Clique no ícone de engrenagem no canto inferior esquerdo da tela do Google Analytics. Em seguida, clique em Data Streams.

<Frame>![](/images/ga4-web-streams.png)</Frame>

Crie um fluxo da Web e defina a URL do seu site de documentação do Mintlify como a URL do fluxo.

Seu Measurement ID tem o formato `G-XXXXXXX` e será exibido em Stream Details imediatamente após você criar o fluxo da Web.

<div id="put-measurement-id-in-docsjson">
  ### Adicione o Measurement ID no docs.json
</div>

Inclua seu Measurement ID no arquivo `docs.json` assim:

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<div id="wait">
  ### Aguarde
</div>

O Google Analytics leva de dois a três dias para exibir seus dados.

Você pode usar o [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) para verificar se a coleta está ativada corretamente. A extensão registrará no console do navegador toda vez que o GA4 fizer uma solicitação.

<Note>

Links de pré-visualização têm a coleta de dados desativada.

</Note>
