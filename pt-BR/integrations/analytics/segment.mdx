---
title: "Segment"
---

Adicione sua write key do Segment ao arquivo `docs.json` para enviar dados de analytics ao Segment.

<CodeGroup>

```json Opções de analytics em docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Exemplo
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
