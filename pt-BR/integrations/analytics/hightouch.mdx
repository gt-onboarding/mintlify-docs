---
title: "Hightouch"
---

Adicione o seguinte ao seu arquivo `docs.json` para enviar analytics para o Hightouch.

<Info>

  Não inclua `https://` no `apiHost`.

</Info>

<CodeGroup>

```json Opções de analytics no docs.json
"integrations": {
    "hightouch": {
        "writeKey": "obrigatório"
        "apiHost": "opcional"
    }
}
```

```json Exemplo
"integrations": {
    "hightouch": {
        "writeKey": "9132c81do371p76sr11da0075469b54f77649c9a067dp0303p56q0q64n072336"
        "apiHost": "us-east-1.hightouch-events.com" # opcional, padrão: `us-east-1.hightouch-events.com`
    }
}
```

</CodeGroup>
