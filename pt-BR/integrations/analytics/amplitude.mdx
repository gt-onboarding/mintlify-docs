---
title: "Amplitude"
---

Adicione o seguinte ao seu arquivo `docs.json` para enviar dados de analytics para o Amplitude.

<CodeGroup>

```json Opções de analytics em docs.json
"integrations": {
    "amplitude": {
        "apiKey": "required"
    }
}
```

```json Exemplo
"integrations": {
    "amplitude": {
        "apiKey": "76bb138bf3fbf58186XXX00000"
    }
}
```

</CodeGroup>
