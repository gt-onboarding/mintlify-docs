---
title: "Heap"
---

Adicione o seguinte ao seu arquivo `docs.json` para enviar dados de analytics para o Heap.

<CodeGroup>

```json Opções de analytics em docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Exemplo
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
