</Info>

As implantações de pré-visualização permitem ver como as alterações na sua documentação ficarão antes de serem mescladas em produção. Cada pré-visualização cria uma URL compartilhável que é atualizada automaticamente conforme você envia novas alterações.

<div id="creating-preview-deployments">
  ## Criando implantações de pré-visualização
</div>

As implantações de pré-visualização são criadas automaticamente por meio de pull requests ou manualmente pelo seu painel.

<div id="automatic-previews">
  ### Pré-visualizações automáticas
</div>

Quando você abre um pull request, o bot do Mintlify adiciona automaticamente um link para visualizar a implantação de pré-visualização no próprio pull request. A pré-visualização é atualizada sempre que você faz push de novos commits para a branch.

<div id="manual-previews">
  ### Pré-visualizações manuais
</div>

Você pode criar manualmente uma pré-visualização para qualquer branch.

1. Acesse o seu [dashboard](https://dashboard.mintlify.com/).
2. Selecione **Previews**.
3. Selecione **Create preview**.
4. Informe o nome da sua branch.
5. Selecione **Create deployment**.

<div id="redeploying-a-preview">
  ## Reimplantar uma pré-visualização
</div>

Reimplante uma pré-visualização para atualizar o conteúdo ou tentar novamente após uma falha na implantação.

1. Selecione a pré-visualização no seu [dashboard](https://dashboard.mintlify.com/).
2. Selecione **Reimplantar**.

<div id="sharing-preview-deployments">
  ## Compartilhando implantações de pré-visualização
</div>

As implantações de pré-visualização podem ser públicas ou restritas a usuários autenticados.

**Acesso público**: As URLs de pré-visualização são públicas por padrão. Compartilhe um link de pré-visualização com qualquer pessoa que precise revisar suas alterações.

**Acesso restrito**: Ative a autenticação para implantações de pré-visualização para limitar o acesso apenas a usuários autenticados da Mintlify. Consulte [Configuração de autenticação](/pt-BR/authentication-personalization/authentication-setup) para mais informações.
