A Mintlify usa um aplicativo do GitHub para sincronizar automaticamente sua documentação com o seu repositório no GitHub.

<div id="installing-the-github-app">
  ## Instalando o app do GitHub
</div>

Instale o aplicativo Mintlify para GitHub na conta ou organização que é proprietária do repositório das suas documentações. Você precisa ter propriedade da organização ou permissões de administrador no repositório para instalar o app. Se não tiver as permissões necessárias, o proprietário do repositório deverá aprovar a solicitação de instalação.

Faça a instalação do aplicativo Mintlify para GitHub pelo seu [dashboard](https://dashboard.mintlify.com/settings/organization/github-app).

<Tip>
Recomendamos conceder acesso apenas ao repositório onde sua documentação está hospedada.
</Tip>

<div id="permissions">
  ## Permissões
</div>

Ao instalar o aplicativo do GitHub, você será solicitado a conceder as seguintes permissões:

Permissões de leitura:

* `metadata`: Informações básicas do repositório

Permissões de leitura e escrita:

* `checks`: Criar verificações de status em pull requests
* `code`: Ler alterações de arquivos quando você fizer commit no branch de docs
* `deployments`: Gerar implantações de pré-visualização para pull requests
* `pull requests`: Criar branches e pull requests a partir do Editor web
* `workflows`: Gerenciar GitHub Actions para as implantações da documentação

<Info>
O aplicativo só acessa repositórios aos quais você concede acesso explicitamente. Se você tiver regras de proteção de branch ativadas, o aplicativo não poderá fazer push diretamente para branches protegidos.
</Info>

<div id="managing-repository-access">
  ## Gerenciando o acesso aos repositórios
</div>

Ao instalar nosso app do GitHub, você pode conceder acesso a todos os seus repositórios ou apenas a repositórios específicos. Recomendamos conceder acesso somente aos repositórios onde sua documentação está. Você pode alterar essa seleção a qualquer momento nas suas [configurações do app do GitHub](https://github.com/apps/mintlify/installations/new).

<div id="configuring-docs-source">
  ## Configurando a origem da documentação
</div>

Altere a organização, o repositório ou a branch de onde sua documentação é gerada na seção [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) do seu painel.

<div id="troubleshooting">
  ## Solução de problemas
</div>

Se você tiver problemas com o app do GitHub, redefinir a conexão pode resolver a maioria deles.

<Steps>
  <Step title="Desinstale o app da Mintlify pelo GitHub.">
    1. No GitHub, acesse [installations](https://github.com/settings/installations) e selecione **Configure** ao lado do app da Mintlify. Role para baixo e selecione **Uninstall**.
    2. Acesse [Authorized GitHub Apps](https://github.com/settings/apps/authorizations) e selecione **Revoke** ao lado do app da Mintlify.
  </Step>
  <Step title="Reinstale o app da Mintlify.">
    1. No seu painel da Mintlify, vá até [Git Settings](https://dashboard.mintlify.com/settings/deployment/git-settings) e instale o app do GitHub.
    2. Autorize sua conta na seção [My Profile](https://dashboard.mintlify.com/settings/account) do seu painel.
  </Step>
</Steps>
