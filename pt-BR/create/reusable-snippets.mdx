Um dos princípios fundamentais do desenvolvimento de software é o DRY (“Don’t Repeat Yourself”), que também se aplica à documentação. Se você estiver repetindo o mesmo conteúdo em vários lugares, crie um trecho personalizado para manter seu conteúdo sincronizado.

<div id="creating-a-custom-snippet">
  ## Criando um snippet personalizado
</div>

**Pré-requisito**: Você deve criar o arquivo do snippet no diretório `snippets` para que a importação funcione.

Qualquer página no diretório `snippets` será tratada como um snippet e não
será renderizada como uma página independente. Se quiser criar uma página independente
a partir do snippet, importe-o em outro arquivo e use-o como um
componente.

<div id="default-export">
  ### Exportação padrão
</div>

1. Adicione ao arquivo do snippet o conteúdo que deseja reutilizar. Opcionalmente, você pode adicionar variáveis que serão preenchidas via props
   ao importar o snippet. Neste exemplo, nossa variável é word.

```typescript snippets/my-snippet.mdx
Olá mundo! Este é o conteúdo que quero reutilizar em várias páginas. 
```

2. Importe o snippet no arquivo de destino.

```typescript destination-file.mdx
---
title: Meu título
description: Minha descrição
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Cabeçalho

Lorem ipsum dolor sit amet.

<MySnippet/>

```

<div id="exporting-with-variables">
  ### Exportando com variáveis
</div>

1. Opcionalmente, você pode adicionar variáveis que podem ser preenchidas via props ao importar o snippet. Neste exemplo, nossa variável é &quot;word&quot;.

```typescript snippets/my-snippet.mdx
Minha palavra-chave do dia é {word}.
```

2. Importe o snippet no arquivo de destino usando a variável. A propriedade será preenchida conforme a sua especificação.

```typescript destination-file.mdx
---
title: Meu título
description: Minha descrição
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Cabeçalho

Lorem ipsum dolor sit amet.

<MySnippet word="bananas" />

```

<div id="reusable-variables">
  ### Variáveis reutilizáveis
</div>

1. Exporte uma variável do seu arquivo de snippets:

```typescript snippets/path/to/custom-variables.mdx
export const myName = 'meu nome';

export const myObject = { fruit: 'morangos' };
```

2. Importe o snippet do arquivo de destino e use a variável:

```typescript destination-file.mdx
---
title: Meu título
description: Minha Descrição
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Olá, meu nome é {myName} e eu gosto de {myObject.fruit}.
```

<div id="jsx-snippets">
  ### Trechos de JSX
</div>

1. Exporte um componente JSX do seu arquivo de trechos. (Consulte [componentes React](/pt-BR/customize/react-components) para mais informações):

```js icon=square-js snippets/my-jsx-snippet.jsx
export const MyJSXSnippet = () => {
  return (
    <div>
      <h1>Olá, mundo!</h1>
    </div>
  )
}
```

<Note>
  Importante: ao criar snippets em JSX, use a sintaxe de função seta (`=>`) em vez de declarações de função. A palavra-chave `function` não é compatível nesse contexto.
</Note>

2. Importe o snippet do arquivo de destino e use o componente:

```typescript destination-file.mdx
---
title: Meu título
description: Minha descrição
---

import { MyJSXSnippet } from '/snippets/my-jsx-snippet.jsx';

<MyJSXSnippet />
```
