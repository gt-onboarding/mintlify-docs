<Info>
  A funcionalidade de RBAC está disponível no [plano Enterprise](https://mintlify.com/pricing?ref=rbac).
</Info>

A Mintlify oferece dois níveis de acesso ao painel: Editor e Administrador.

As ações a seguir são restritas ao papel de Administrador:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Atualizar funções de usuário |   ❌   |  ✅   |
| Excluir usuários        |   ❌   |  ✅   |
| Convidar usuários administradores |   ❌   |  ✅   |
| Gerenciar e atualizar cobrança |   ❌   |  ✅   |
| Atualizar domínio personalizado |   ❌   |  ✅   |
| Atualizar fonte do Git  |   ❌   |  ✅   |
| Excluir organização     |   ❌   |  ✅   |

As demais ações no painel estão disponíveis para ambas as funções.

Você pode convidar quantos administradores quiser, mas recomendamos limitar o acesso de administrador aos usuários que precisarem dele.
