title: "Páginas ocultas"
description: "Excluir páginas da navegação"

Páginas ocultas são removidas da navegação do seu site, mas permanecem publicamente acessíveis para quem souber a URL.

Use páginas ocultas para conteúdo que você deseja que esteja acessível no seu site ou referenciado como contexto por ferramentas de IA, mas que não deve ser encontrado por meio da navegação.

Para conteúdo que exige controle de acesso rigoroso, você deve configurar a [autenticação](/pt-BR/authentication-personalization/authentication-setup).

Se você quiser ocultar páginas para grupos específicos de usuários, use a personalização para controlar a [visibilidade de páginas](/pt-BR/authentication-personalization/overview#page-visibility).

<div id="hiding-a-page">
  ## Ocultando uma página
</div>

Uma página fica oculta se não estiver incluída na navegação do seu `docs.json`. Para ocultar uma página, remova-a da sua estrutura de navegação.

Páginas ocultas usam a mesma estrutura de URL que páginas comuns, baseada no caminho do arquivo. Por exemplo, `guides/hidden-page.mdx` ficará acessível em `docs.yoursite.com/guides/hidden-page`.

Veja um [exemplo de página oculta](/pt-BR/guides/hidden-page-example).

<Note>
  Alguns elementos de navegação, como barras laterais, menus suspensos e abas, podem aparecer vazios ou sofrer alterações de layout em páginas ocultas.
</Note>

<div id="hiding-a-group-of-pages">
  ## Ocultando um grupo de páginas
</div>

Um grupo de páginas fica oculto se a propriedade `hidden` estiver definida como `true` no seu arquivo `docs.json`:

```json highlight={4}
"groups": [
  {
    "group": "Getting started",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  },
  {
    "group": "Guides",
    "pages": [
      "guides/hidden-page.mdx",
      "guides/hidden-groups.mdx"
    ]
  }
]
```

Neste exemplo, o grupo `Getting started` está oculto, enquanto o grupo `Guides` permanece visível.

<div id="hiding-a-tab">
  ### Ocultando uma aba
</div>

Você também pode ocultar uma aba adicionando a propriedade `hidden` ao arquivo `docs.json`:

```json highlight={4}
"tabs": [
  {
    "tab": "Home",
    "hidden": true,
    "pages": [
      "index",
      "quickstart"
    ]
  }
]
```

<div id="search-seo-and-ai-indexing">
  ## Pesquisa, SEO e indexação por IA
</div>

Por padrão, páginas ocultas são excluídas da indexação por mecanismos de busca, da pesquisa interna da sua documentação e do contexto utilizado pelo Assistente. Para incluir páginas ocultas nos resultados de pesquisa e como contexto para o assistente, adicione a propriedade `seo` ao seu `docs.json`:

```json
"seo": {
    "indexing": "all"
}
```

Para excluir uma página específica, adicione `noindex: "true"` ao frontmatter dela.
