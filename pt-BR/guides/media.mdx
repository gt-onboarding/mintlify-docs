<Tip>
  Esta página apresenta as melhores práticas para usar capturas de tela, GIFs e vídeos na documentação.
</Tip>

Capturas de tela, GIFs e vídeos podem enriquecer a documentação, mas exigem manutenção contínua conforme a interface muda. Use-os de forma seletiva para evitar retrabalho desnecessário.

Diretrizes principais:

- **A mídia deve ser complementar.** Se um fluxo de trabalho estiver claro apenas em texto, evite adicionar elementos visuais.
- **Garanta a acessibilidade.** Adicione texto alternativo para imagens, legendas para vídeos e transcrições para conteúdos em áudio. Muitas pessoas usam tecnologias assistivas, e conteúdo acessível beneficia todos os usuários.
- **Equilibre clareza e manutenção.** Mudanças frequentes na interface podem tornar capturas de tela e vídeos rapidamente desatualizados. Avalie se o esforço para atualizá-los compensa o valor que agregam.

<div id="when-to-use-media">
  ## Quando usar mídia
</div>

* **Capturas de tela** para tarefas difíceis de explicar só com palavras.
* **GIFs** para fins promocionais e fluxos de trabalho curtos, porém complexos.
* **Vídeos** para conceitos abstratos e fluxos de trabalho longos.

Use mídia de forma comedida e intencional para evitar dívida desnecessária de documentação. Quando bem aplicada, ela melhora a compreensão sem adicionar encargos de manutenção ou barreiras de acessibilidade.
