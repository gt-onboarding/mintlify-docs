Esta página está oculta! Ela não está incluída na navegação do `docs.json`, portanto, você só pode acessá-la pela URL.

Veja [Páginas ocultas](/pt-BR/guides/hidden-pages) para mais informações.

<div id="common-use-cases-for-hidden-pages">
  ## Casos de uso comuns para páginas ocultas
</div>

<Note>
  Por padrão, páginas ocultas são publicamente acessíveis, mas não são descobertas pela navegação. Se você quiser restringir o acesso a uma página oculta, configure a [autenticação](/pt-BR/authentication-personalization/authentication-setup).
</Note>

- **Documentação beta**: Informações que podem ser públicas, mas não devem ser encontradas pela navegação.
- **Contexto para ferramentas de IA**: Se páginas ocultas forem indexadas, ferramentas de IA podem referenciá-las como contexto. Use páginas ocultas para conteúdo que não é relevante para usuários, mas pode ajudar as ferramentas de IA a fornecer respostas mais precisas.
- **Páginas antigas**: Mantenha conteúdo antigo acessível por links diretos enquanto o remove dos menus de navegação.
- **Ferramentas internas**: Documente ferramentas internas, APIs de homologação ou processos de desenvolvimento.
