Esta página está oculta! Ela não está incluída na navegação do `docs.json`, então você só pode acessá-la pela URL.

Consulte [Páginas ocultas](/pt-BR/guides/hidden-pages) para mais informações.

<div id="common-use-cases-for-hidden-pages">
  ## Casos de uso comuns para páginas ocultas
</div>

<Note>
  Por padrão, as páginas ocultas são publicamente acessíveis, mas não podem ser encontradas pela navegação. Se você quiser restringir o acesso a uma página oculta, deve configurar a [autenticação](/pt-BR/authentication-personalization/authentication-setup).
</Note>

* **Documentação beta**: Informações que podem ser públicas, mas não devem ser encontradas pela navegação.
* **Contexto para ferramentas de IA**: Se as páginas ocultas forem indexadas, as ferramentas de IA poderão referenciá-las como contexto. Use páginas ocultas para conteúdo que não é relevante para os usuários, mas pode ajudar as ferramentas de IA a fornecer respostas mais precisas.
* **Páginas antigas**: Mantenha conteúdo antigo acessível por links diretos, removendo-o dos menus de navegação.
* **Ferramentas internas**: Documente ferramentas internas, APIs de staging ou processos de desenvolvimento.
