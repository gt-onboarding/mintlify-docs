---
openapi: GET /agent/{projectId}/jobs
---

<div id="usage">
  ## Uso
</div>

Este endpoint recupera todas as tarefas do agente para o domínio especificado, fornecendo uma visão geral de todas as atividades do agente e de seus status atuais. Isso é útil para monitorar e gerenciar várias tarefas de agente simultâneas ou históricas.

<div id="response">
  ## Resposta
</div>

Use este endpoint para obter uma visão completa de todas as sessões anteriores do agente.
