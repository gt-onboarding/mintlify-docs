Esta página é oculta! Ela não está incluída na navegação do `docs.json`, então você só pode acessá-la pela URL.

Veja [Páginas ocultas](/pt-BR/organize/hidden-pages) para mais informações.

<div id="common-use-cases-for-hidden-pages">
  ## Casos de uso comuns para Páginas ocultas
</div>

<Note>
  Por padrão, Páginas ocultas são publicamente acessíveis, mas não são encontradas pela navegação. Se você quiser restringir o acesso a uma página oculta, é necessário configurar a [autenticação](/pt-BR/deploy/authentication-setup).
</Note>

* **Documentação beta**: Informações que podem ser públicas, mas não devem ser encontradas pela navegação.
* **Contexto para ferramentas de IA**: Se Páginas ocultas forem indexadas, ferramentas de IA podem referenciá-las como contexto. Use Páginas ocultas para conteúdo que não é relevante para os usuários, mas pode ajudar ferramentas de IA a fornecer respostas mais precisas.
* **Páginas legadas**: Mantenha o conteúdo antigo acessível por links diretos enquanto o remove dos menus de navegação.
* **Ferramentas internas**: Documente ferramentas internas, APIs de staging ou processos de desenvolvimento.

````mdx wrap
```bash

Este endpoint limpa o cache da aplicação e está disponível apenas em ambientes de desenvolvimento.

<Warning>
````
