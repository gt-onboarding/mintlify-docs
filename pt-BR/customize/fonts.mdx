title: "Fontes"
description: "Configurar fontes personalizadas"
---

Defina uma fonte personalizada para todo o site ou, separadamente, para títulos e para o corpo do texto. Use Google Fonts, arquivos de fontes locais ou fontes hospedadas externamente. A fonte padrão varia de acordo com o tema.

As fontes são controladas pela [propriedade fonts](/pt-BR/organize/settings#param-fonts) no seu `docs.json`.

<div id="google-fonts">
  ## Google Fonts
</div>

A Mintlify carrega automaticamente as [Google Fonts](https://fonts.google.com) quando você especifica uma família tipográfica no seu `docs.json`.

```json docs.json
"fonts": {
  "family": "Inter"
}
```

<div id="local-fonts">
  ## Fontes locais
</div>

Para usar fontes locais, coloque os arquivos de fonte no diretório do projeto e faça referência a eles na configuração do `docs.json`.

<div id="setting-up-local-fonts">
  ### Configurando fontes locais
</div>

<Steps>
  <Step title="Adicione os arquivos de fonte ao seu projeto">
    Por exemplo, crie um diretório `fonts` e adicione seus arquivos de fonte:

    ```text
    your-project/
    ├── fonts/
    │   ├── InterDisplay-Regular.woff2
    │   └── InterDisplay-Bold.woff2
    ├── docs.json
    └── ...
    ```
  </Step>

  <Step title="Configure as fontes no docs.json">
    Faça referência às suas fontes locais usando caminhos relativos a partir da raiz do projeto:

    ```json docs.json
    {
      "fonts": {
        "family": "InterDisplay",
        "source": "/fonts/InterDisplay-Regular.woff2",
        "format": "woff2",
        "weight": 400
      }
    }
    ```
  </Step>
</Steps>

<div id="local-fonts-for-headings-and-body">
  ### Fontes locais para títulos e corpo
</div>

Defina fontes locais separadas para títulos e texto do corpo no seu `docs.json`:

```json docs.json
{
  "fonts": {
    "heading": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Bold.woff2",
      "format": "woff2",
      "weight": 700
    },
    "body": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Regular.woff2",
      "format": "woff2",
      "weight": 400
    }
  }
}
```

<div id="externally-hosted-fonts">
  ## Fontes hospedadas externamente
</div>

Use fontes hospedadas externamente referenciando a URL da fonte no seu `docs.json`:

```json docs.json
{
  "fonts": {
    "family": "Hubot Sans",
    "source": "https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2",
    "format": "woff2",
    "weight": 400
  }
}
```
