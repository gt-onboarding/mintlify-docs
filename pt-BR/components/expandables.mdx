---
title: "Expansíveis"
description: "Alternar para exibir propriedades aninhadas"
icon: "list-tree"
---

Use expansíveis para mostrar e ocultar conteúdo aninhado em campos de resposta. Expansíveis são especialmente úteis para exibir propriedades de objetos complexos na documentação de API.

<ResponseField name="user" type="User Object">
  <Expandable title="properties">

<ResponseField name="full_name" type="string">
  O nome completo do usuário
</ResponseField>

<ResponseField name="is_over_21" type="boolean">
  Indica se o usuário tem mais de 21 anos
</ResponseField>

  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      O nome completo do usuário
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Indica se o usuário tem mais de 21 anos
    </ResponseField>
  </Expandable>
</ResponseField>
```

<div id="properties">
  ## Propriedades
</div>

<ResponseField name="title" type="string">
  O nome do objeto que está sendo exibido.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Defina como `true` para que o painel expansível abra quando a página carregar
</ResponseField>
