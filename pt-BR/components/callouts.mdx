---
title: "Destaques"
description: "Use destaques para adicionar contexto chamativo ao seu conteúdo"
icon: "info"
---

Os destaques podem ser estilizados como Nota, Aviso, Info, Dica, Confirmação, Perigo, ou você pode criar um destaque personalizado:

<Note>Isto adiciona uma nota ao conteúdo</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>Isto exibe um aviso para ficar atento</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Isto chama a atenção para informações importantes</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>Isto sugere uma dica útil</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Isto indica um status confirmado</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>Este é um destaque de perigo</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">Este é um destaque personalizado</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
