---
title: "Destaques"
description: "Use destaques (callouts) para adicionar contexto chamativo ao seu conteúdo"
icon: "info"
---

Os destaques podem ser exibidos como Note, Warning, Info, Tip, Check, Danger, ou você pode criar seu próprio callout:

<Note>Isso adiciona uma observação ao conteúdo</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>Isso emite um aviso para ficar atento</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Isso destaca informações importantes</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>Isso sugere uma dica útil</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Isso indica um status verificado</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>Este é um destaque de perigo</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">Este é um callout personalizado</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
