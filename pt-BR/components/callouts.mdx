---
title: "Destaques"
description: "Use destaques para adicionar contexto chamativo ao seu conteúdo"
---

Os destaques podem ser configurados como Nota, Aviso, Informação, Dica, Confirmação, Perigo, ou você pode criar seu próprio destaque:

<Note>Isso adiciona uma nota ao conteúdo</Note>

```mdx
<Note>Isso adiciona uma nota no conteúdo</Note>
```

<Warning>Isso gera um aviso para ficar atento(a)</Warning>

```mdx
<Warning>Isso gera um aviso importante</Warning>
```

<Info>Destaca informações importantes</Info>

```mdx
<Info>Isso destaca informações importantes</Info>
```

<Tip>Isso indica uma dica útil</Tip>

```mdx
<Tip>Isso indica uma dica útil</Tip>
```

<Check>Isso nos dá um status marcado</Check>

```mdx
<Check>Isso nos traz um status marcado</Check>
```

<Danger>Este é um alerta de perigo</Danger>

```mdx
<Danger>Este é um alerta de perigo</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Este é um callout personalizado</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Este é um destaque personalizado</Callout>
```
