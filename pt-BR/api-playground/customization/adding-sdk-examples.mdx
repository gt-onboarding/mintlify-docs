---
title: "Adicionando exemplos de SDK"
description: "Exiba exemplos de código específicos por linguagem ao lado dos seus endpoints de API para mostrar aos desenvolvedores como usar seus SDKs"
---

Se os seus usuários interagem com sua API usando um SDK em vez de diretamente por meio de uma requisição de rede, você pode usar a extensão `x-codeSamples` para adicionar exemplos de código ao seu documento OpenAPI e exibí-los nas suas páginas OpenAPI.

Essa propriedade pode ser adicionada a qualquer método de requisição e tem o seguinte esquema.

<ParamField body="lang" type="string" required>
  O idioma do exemplo de código.
</ParamField>

<ParamField body="label" type="string">
  O rótulo do exemplo. Útil ao fornecer vários exemplos para um único endpoint.
</ParamField>

<ParamField body="source" type="string" required>
  O código-fonte do exemplo.
</ParamField>

A seguir, um exemplo de snippets de código para um app de acompanhamento de plantas, que possui uma ferramenta de linha de comando (CLI) em Bash e um SDK em JavaScript.

```yaml
paths:
  /plants:
    get:
      # ...
      x-codeSamples:
        - lang: bash
          label: List all unwatered plants
          source: |
            planter list -u
        - lang: javascript
          label: List all unwatered plants
          source: |
            const planter = require('planter');
            planter.list({ unwatered: true });
        - lang: bash
          label: List all potted plants
          source: |
            planter list -p
        - lang: javascript
          label: List all potted plants
          source: |
            const planter = require('planter');
            planter.list({ potted: true });
```
