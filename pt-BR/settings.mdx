O arquivo `docs.json` permite transformar uma coleção de arquivos Markdown em um site de documentação navegável e personalizado. Esse arquivo de configuração obrigatório controla o estilo, a navegação, as integrações e muito mais. Pense nele como o projeto da sua documentação.

As configurações em `docs.json` se aplicam globalmente a todas as páginas.

<div id="setting-up-your-docsjson">
  ## Configurando seu `docs.json`
</div>

Para começar, você só precisa especificar `theme`, `name`, `colors.primary` e `navigation`. Os demais campos são opcionais e podem ser adicionados conforme as necessidades da sua documentação aumentarem.

Para ter a melhor experiência de edição, inclua a referência de schema no início do arquivo `docs.json`. Isso habilita autocompletar, validação e dicas úteis na maioria dos editores de código:

```json
{
  "$schema": "https://mintlify.com/docs.json",
  "theme": "mint",
  "name": "Your Docs",
  "colors": {
    "primary": "#ff0000"
  },
  "navigation": {
    // Sua estrutura de navegação
  }
  // O restante da sua configuração
}
```

<ResponseField name="theme" required>
  O tema de layout do seu site.

  Uma das seguintes opções: `mint`, `maple`, `palm`, `willow`, `linden`, `almond`, `aspen`.

  Veja [Temas](pt-BR/themes) para mais informações.
</ResponseField>

  O nome do seu projeto, organização ou produto.
</ResponseField>

<ResponseField name="colors" type="object" required>
  As cores usadas na sua documentação. As cores são aplicadas de forma diferente dependendo do tema. Se você fornecer apenas a cor primária, ela será usada para todos os elementos de cor.

  <Expandable title="Colors">
    <ResponseField name="primary" type="string matching ^#([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$" required>
      A cor primária da sua documentação. Geralmente usada para ênfase no modo claro, com alguma variação por tema.

      Deve ser um código hexadecimal começando com `#`.
    </ResponseField>

    <ResponseField name="light" type="string matching ^#([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$">
      A cor usada para ênfase no modo escuro.

      Deve ser um código hexadecimal começando com `#`.
    </ResponseField>

    <ResponseField name="dark" type="string matching ^#([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$">
      A cor usada para botões e estados de hover nos modos claro e escuro, com alguma variação por tema.

  </Expandable>
</ResponseField>

  ## Atualizando a partir de `mint.json`
</div>

Se o seu projeto de documentação usa o arquivo `mint.json` descontinuado, siga estas etapas para migrar para `docs.json`.

<Steps>
  <Step title="Instale ou atualize a CLI">
    Se você ainda não instalou a [CLI](/pt-BR/installation), instale agora:

    <CodeGroup>
      ```bash npm
      npm i -g mint
      ```
