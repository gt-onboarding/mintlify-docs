<div id="lists">
  ## Listas
</div>

As listas seguem a [sintaxe oficial do Markdown](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Lista ordenada
</div>

Para criar uma lista ordenada, adicione números seguidos de ponto antes dos itens da lista.

1. Primeiro item
2. Segundo item
3. Terceiro item
4. Quarto item

```mdx
1. Primeiro item
2. Segundo item
3. Terceiro item
4. Quarto item
```

<div id="unordered-list">
  ### Lista não ordenada
</div>

Para criar uma lista não ordenada, adicione hífens (`-`), asteriscos (`*`) ou sinais de mais (`+`) antes dos itens da lista.

- Primeiro item
- Segundo item
- Terceiro item
- Quarto item

```mdx
- Primeiro item
- Segundo item
- Terceiro item
- Quarto item
```

<div id="nested-list">
  ### Lista aninhada
</div>

Recuar os itens da lista para criar níveis aninhados.

- Primeiro item
- Segundo item
  - Item adicional
  - Item adicional
- Terceiro item

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## Tabelas
</div>

As tabelas seguem a [sintaxe oficial do Markdown](https://www.markdownguide.org/extended-syntax/#tables).

Para adicionar uma tabela, use três ou mais hífens (`---`) para criar o cabeçalho de cada coluna e use pipes (`|`) para separar cada coluna. Para compatibilidade, você também deve adicionar um pipe em cada extremidade da linha.

| Propriedade | Descrição                           |
| ----------- | ----------------------------------- |
| Nome        | Nome completo do usuário            |
| Idade       | Idade informada                     |
| Entrou      | Se o usuário ingressou na comunidade |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
