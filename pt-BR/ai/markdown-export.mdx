Markdown fornece texto estruturado que ferramentas de IA conseguem processar com mais eficiência do que HTML, resultando em respostas mais precisas, processamento mais rápido e menor consumo de tokens.

A Mintlify gera automaticamente versões em Markdown das páginas, otimizadas para ferramentas de IA e integrações externas.


<div id="md-url-extension">
  ## Extensão de URL .md
</div>

Adicione `.md` ao URL de qualquer página para visualizar uma versão em Markdown.

<PreviewButton href="https://mintlify.com/docs/ai/markdown-export.md">Abrir esta página como Markdown</PreviewButton>

<div id="keyboard-shortcut">
  ## Atalho de teclado
</div>

Pressione <kbd>Command</kbd> + <kbd>C</kbd> (<kbd>Ctrl</kbd> + <kbd>C</kbd> no Windows) para copiar a página em Markdown para a área de transferência.
