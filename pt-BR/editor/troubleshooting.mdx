Encontre soluções para problemas comuns que você pode enfrentar ao usar o editor web.

<AccordionGroup>
  <Accordion title="Alterações não aparecem após a publicação">
    **Possíveis causas:**

    * A implantação ainda está em andamento
    * Problemas de cache do navegador
    * Erros de build ou de implantação

    **Soluções:**

    1. Verifique o status da implantação no seu painel.
    2. Faça uma atualização forçada no navegador (<kbd>Ctrl</kbd> + <kbd>F5</kbd> ou <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>R</kbd>)
    3. Limpe o cache do navegador.
  </Accordion>

  <Accordion title="Erros de permissão ao publicar">
    **Possíveis causas:**

    * Permissões insuficientes no repositório Git
    * Problemas de autenticação com o seu provedor Git

    **Soluções:**

    1. Verifique se você tem acesso adequado ao repositório.
    2. Confira se a integração com o Git está configurada corretamente.
    3. Consulte a documentação de [Permissões do editor](/pt-BR/advanced/dashboard/permissions).
  </Accordion>

  <Accordion title="Problemas ao carregar o editor">
    **Possíveis causas:**

    * Problemas de conectividade de rede
    * Repositórios de documentação grandes

    **Soluções:**

    1. Verifique sua conexão com a internet.
    2. Atualize a página.
    3. Entre em contato com o suporte se o problema persistir.
  </Accordion>

  <Accordion title="Arquivos não carregando ou exibindo erros">
    **Possíveis causas:**

    * Sintaxe MDX inválida nos arquivos
    * Arquivos ausentes ou corrompidos
    * Arquivos muito grandes causando timeouts

    **Soluções:**

    1. Verifique a sintaxe do arquivo para erros de formatação em MDX.
    2. Confirme se o arquivo existe no repositório.
  </Accordion>
</AccordionGroup>
