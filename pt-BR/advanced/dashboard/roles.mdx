---
title: "Papéis"
description: "Controle o acesso ao seu painel com papéis."
---

<Info>
  A funcionalidade de RBAC está disponível no [plano Enterprise](https://mintlify.com/pricing?ref=rbac).
</Info>

A Mintlify oferece dois níveis de acesso ao painel: Editor e Admin.

As ações a seguir são restritas ao papel de Admin:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Atualizar papéis de usuário |   ❌   |  ✅   |
| Excluir usuários        |   ❌   |  ✅   |
| Convidar usuários admin |   ❌   |  ✅   |
| Gerenciar e atualizar cobrança |   ❌   |  ✅   |
| Atualizar domínio personalizado |   ❌   |  ✅   |
| Atualizar origem do Git |   ❌   |  ✅   |
| Excluir organização     |   ❌   |  ✅   |

Outras ações no painel estão disponíveis para ambos os papéis.

Você pode convidar quantos admins quiser, mas recomendamos limitar o acesso de admin
aos usuários que realmente precisem dele.
