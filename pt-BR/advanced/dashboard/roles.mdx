<Info>
  A funcionalidade de RBAC está disponível no [plano Enterprise](https://mintlify.com/pricing?ref=rbac).
</Info>

A Mintlify oferece dois níveis de acesso ao dashboard: Editor e Admin.

A seguir, descrevemos as ações que são restritas à função de Admin:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Atualizar funções de usuário |   ❌   |  ✅   |
| Excluir usuários        |   ❌   |  ✅   |
| Convidar usuários Admin |   ❌   |  ✅   |
| Gerenciar e atualizar faturamento |   ❌   |  ✅   |
| Atualizar domínio personalizado |   ❌   |  ✅   |
| Atualizar origem do Git |   ❌   |  ✅   |
| Excluir organização     |   ❌   |  ✅   |

As demais ações no dashboard estão disponíveis para ambas as funções.

Você pode convidar quantos Admins quiser, mas recomendamos limitar o acesso de Admin
aos usuários que realmente precisem dele.
