---
title: "Single sign-on (SSO)"
description: "Personalize como sua equipe pode acessar o painel de administração"
---

<Info>
  A funcionalidade de SSO está disponível no [plano Enterprise](https://mintlify.com/pricing?ref=sso).
</Info>

Use o single sign-on no seu painel via SAML e OIDC. Se você usa Okta, Google Workspace ou Microsoft Entra, temos documentação específica para configurar o SSO com esses provedores. Se você usa outro provedor, [entre em contato conosco](mailto:support@mintlify.com).

<div id="okta">
  ## Okta
</div>

<Tabs>
    <Tab title="SAML">
        <Steps>
            <Step title="Criar um aplicativo">
                Em `Applications`, clique para criar uma nova integração de aplicativo usando SAML 2.0.
            </Step>
            <Step title="Configurar a integração">
                Insira o seguinte:
                * Single sign-on URL (fornecida pela Mintlify)
                * Audience URI (fornecido pela Mintlify)
                * Name ID Format: `EmailAddress`
                * Attribute Statements:
                    | Name | Name format | Value
                    | ---- | ----------- | -----
                    | `firstName` | Basic | `user.firstName` |
                    | `lastName` | Basic | `user.lastName` |
            </Step>
            <Step title="Envie para nós as informações do seu IdP">
                Depois que o aplicativo estiver configurado, acesse a guia Sign-On e envie para nós a metadata URL.
                Vamos habilitar a conexão do nosso lado usando essas informações.
            </Step>
        </Steps>
    </Tab>
    <Tab title="OIDC">
        <Steps>
            <Step title="Criar um aplicativo">
                Em `Applications`, clique para criar uma nova integração de aplicativo usando OIDC.
                Selecione o tipo de aplicativo `Web Application`.
            </Step>
            <Step title="Configurar a integração">
                Selecione o grant type Authorization Code e insira o Redirect URI fornecido pela Mintlify.
            </Step>
            <Step title="Envie para nós as informações do seu IdP">
                Depois que o aplicativo estiver configurado, acesse a guia General e localize o client ID e o client secret.
                Envie-nos esses dados com segurança, juntamente com a URL da sua instância Okta (por exemplo, `<your-tenant-name>.okta.com`). Você pode enviar por um serviço como 1Password ou SendSafely.
            </Step>
        </Steps>
    </Tab>
</Tabs>

<div id="google-workspace">
  ## Google Workspace
</div>

<Tabs>
    <Tab title="SAML">
        <Steps>
            <Step title="Criar um aplicativo">
                Em `Web and mobile apps`, selecione `Add custom SAML app` no menu suspenso `Add app`.  
                <Frame>
                    ![](/images/gsuite-add-custom-saml-app.png)
                </Frame>              
            </Step>
            <Step title="Envie para nós as informações do seu IdP">
                Copie a URL de SSO, o Entity ID e o certificado x509 fornecidos e envie-os para a equipe da Mintlify.
                <Frame>
                    ![](/images/gsuite-saml-metadata.png)                    
                </Frame>
            </Step>
            <Step title="Configurar a integração">
                Na página Service provider details, insira o seguinte:
                * ACS URL (fornecida pela Mintlify)
                * Entity ID (fornecido pela Mintlify)
                * Name ID format: `EMAIL`
                * Name ID: `Basic Information > Primary email`

                <Frame>
                    ![](/images/gsuite-sp-details.png)
                </Frame>

                Na próxima página, insira as seguintes declarações de atributo:
                | Google Directory Attribute | App Attribute |
                | -------------------------- | ------------- |
                | `First name` | `firstName`  |
                | `Last name` | `lastName` |

                Quando esta etapa for concluída e os usuários forem atribuídos ao aplicativo, avise nossa equipe e ativaremos o SSO na sua conta!
            </Step>
        </Steps>
    </Tab>
</Tabs>

<div id="microsoft-entra">
  ## Microsoft Entra
</div>

<Tabs>
    <Tab title="SAML">
        <Steps>
            <Step title="Create an application">
                1. Em "Enterprise applications", selecione **New application**.
                2. Selecione **Create your own application** e escolha "Integrate any other application you don't find in the gallery (Non-gallery)."
            </Step>
            <Step title="Configure SAML">
                Acesse a página de configuração de Single Sign-On e selecione **SAML**. Em "Basic SAML Configuration", insira o seguinte:
                * Identifier (Entity ID): o Audience URI fornecido pela Mintlify.
                * Reply URL (Assertion Consumer Service URL): a ACS URL fornecida pela Mintlify.
                
                Deixe os demais valores em branco e selecione **Save**.
            </Step>
            <Step title="Configure Attributes & Claims">
                Edite a seção Attributes & Claims:
                1. Selecione **Unique User Identifier (Name ID)** em "Required Claim".
                2. Altere o atributo Source para usar `user.primaryauthoritativeemail`.
                3. Em Additional claims, crie as seguintes declarações:
                    | Name | Value |
                    | ---- | ----- |
                    | `firstName` | `user.givenname` |
                    | `lastName` | `user.surname` |
            </Step>
            <Step title="Send Mintlify your IdP information">
                Depois que o aplicativo estiver configurado, acesse a seção "SAML Certificates" e envie para nós o App Federation Metadata URL.
                Habilitaremos a conexão do nosso lado usando essas informações.
            </Step>
            <Step title="Assign Users">
                Acesse "Users and groups" no seu aplicativo Entra e adicione os usuários que devem ter acesso ao seu dashboard.
            </Step>
        </Steps>
    </Tab>
</Tabs>
