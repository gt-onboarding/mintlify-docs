openapi: POST /agent/{projectId}/job
---

此端点会根据提供的消息和分支信息创建一个代理任务。该任务以异步方式执行，并通过流式响应返回执行详情和结果。

如果目标分支不存在，代理会创建一个新分支。若文件编辑成功，任务结束时会自动创建一个草稿 Pull Request。

<div id="rate-limits">
  ## 速率限制
</div>

Agent API 的限制如下：

- 每个 Mintlify 项目每小时最多使用 100 次

<div id="suggested-usage">
  ## 建议用法
</div>

为获得最佳效果，请使用 [ai-sdk 的 useChat 钩子](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat)来发送请求并处理响应。
