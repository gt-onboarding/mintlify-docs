<div id="rate-limits">
  ## 速率限制
</div>

助手 API 具有以下限制：

- 每个密钥每月最多使用 10,000 次
- 每个 Mintlify 组织每小时最多 10,000 次请求
- 每个 IP 每天最多 10,000 次请求

<div id="suggested-usage">
  ## 建议用法
</div>

为获得最佳效果，请使用 [ai-sdk 的 useChat hook](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat) 来发送请求并处理响应。

你可以在传递给该 hook 的 options 参数的 `body` 字段中设置 `fp`、`threadId` 和 `filter`。
