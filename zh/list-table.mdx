<div id="lists">
  ## 列表
</div>

列表遵循官方的 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#lists-1)。

<div id="ordered-list">
  ### 有序列表
</div>

要创建有序列表，请在每个列表项前加上数字并紧随一个句点。

1. 第一项
2. 第二项
3. 第三项
4. 第四项

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### 无序列表
</div>

要创建无序列表，请在列表项前添加短横线（`-`）、星号（`*`）或加号（`+`）。

- 第一个项目
- 第二个项目
- 第三个项目
- 第四个项目

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### 嵌套列表
</div>

通过缩进列表项来创建嵌套。

- 第一项
- 第二项
  - 其他项
  - 其他项
- 第三项

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## 表格
</div>

表格遵循官方的 [Markdown 语法](https://www.markdownguide.org/extended-syntax/#tables)。

要添加表格，使用三个或更多连字符（`---`）创建每一列的表头，并使用竖线（`|`）分隔各列。为确保兼容性，你还应在每行的两端各添加一个竖线。

| 属性     | 描述                                   |
| -------- | -------------------------------------- |
| Name     | 用户全名                               |
| Age      | 填报年龄                               |
| Joined   | 用户是否已加入社区                     |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
