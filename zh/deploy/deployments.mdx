---
title: "部署"
description: "排查部署问题"
---

当你向已连接的仓库推送更改时，文档站点会自动部署。此功能需要正确安装并连接 Mintlify 的 GitHub 应用。

如果最新更改未在在线站点上显示，首先检查该 GitHub 应用是否已安装在拥有文档仓库的账户或组织中。更多信息请参见 [GitHub 故障排查](/zh/deploy/github#troubleshooting)。

如果 GitHub 应用已连接但更改仍未部署，你可以在仪表板中手动触发重新构建。

<div id="manually-triggering-a-deployment">
  ## 手动触发部署
</div>

<Steps>
  <Step title="确认你的最新提交已成功。">
    检查你的最新提交是否已出现在文档仓库中，且未出现任何错误。
  </Step>

  <Step title="手动触发部署。">
    前往你的[仪表盘](https://dashboard.mintlify.com)并点击“部署”按钮。

</Steps>
