---
title: "Pirsch"
---

在您的 `docs.json` 文件中添加以下内容，将分析数据发送到 Pirsch。

您可以在 Settings \> Developer \> Identification Code 中找到您站点的 ID。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
