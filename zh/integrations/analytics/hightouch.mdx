---
title: "Hightouch"
---

将以下内容添加到你的 `docs.json` 文件，以将分析数据发送到 Hightouch。

<Info>

  对于 `apiHost`，不要包含 `https://`。

</Info>

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "hightouch": {
        "writeKey": "required"
        "apiHost": "optional"
    }
}
```

```json Example
"integrations": {
    "hightouch": {
        "writeKey": "9132c81do371p76sr11da0075469b54f77649c9a067dp0303p56q0q64n072336"
        "apiHost": "us-east-1.hightouch-events.com" # 可选，默认值为 `us-east-1.hightouch-events.com`
    }
}
```

</CodeGroup>
