title: "Google Analytics 4"
---

您需要创建一个新的<Tooltip tip="Google Analytics 4">GA4</Tooltip>属性以配合 Mintlify 使用。收集的数据将汇总到与您其他 Google Analytics 数据相同的项目中。

如果您使用的是旧版 Google Analytics（Universal Analytics），依然可以创建<Tooltip tip="Google Analytics 4">GA4</Tooltip>属性。<Tooltip tip="Google Analytics 4">GA4</Tooltip>数据与 UA 数据略有不同，但仍会归集在同一项目中。

<div id="how-to-connect-ga4-to-mintlify">
  ## 如何将 GA4 接入 Mintlify
</div>

<div id="create-a-web-stream">
  ### 创建 Web 数据流
</div>

你需要先创建一个 Web 数据流，以获取要填入 Mintlify 的 Measurement ID。

点击 Google Analytics 界面左下角的齿轮图标，然后点击 Data Streams（数据流）。

<Frame>![](/images/ga4-web-streams.png)</Frame>

创建一个 Web 数据流，并将你的 Mintlify 文档站点的 URL 填入为数据流的 URL。

你的 Measurement ID 形如 `G-XXXXXXX`，创建 Web 数据流后会立即显示在 Stream Details（数据流详情）中。

<div id="put-measurement-id-in-docsjson">
  ### 在 docs.json 中添加 Measurement ID
</div>

将你的 Measurement ID 添加到 `docs.json` 文件中，如下所示：

```json docs.json
"integrations": {
    "ga4": {
        "measurementId": "G-XXXXXXX"
    }
}
```

<div id="wait">
  ### 请稍候
</div>

Google Analytics 通常需要两到三天才会显示您的数据。

您可以使用 [Google Analytics Debugger](https://chrome.google.com/webstore/detail/google-analytics-debugger/jnkmfdileelhofjcijamephohjechhna?hl=en) 检查是否已正确启用分析功能。每当 GA4 发出请求时，该扩展都会在浏览器控制台中输出日志。
