---
title: "Google Tag Manager"
---

将你的标签 ID 添加到 `docs.json` 文件，我们会将 Google Tag Manager 脚本注入到你所有页面中。

如需使用 Cookie 同意横幅，请在 Google Tag Manager 中自行配置。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
