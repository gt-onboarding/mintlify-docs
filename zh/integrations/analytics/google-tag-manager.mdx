---
title: "Google Tag Manager"
---

将你的标签 ID 添加到 `docs.json` 文件中，我们会将 Google Tag Manager 脚本注入到你的所有页面。

如果需要，你需要使用 Google Tag Manager 自行设置 Cookie 同意横幅。

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
