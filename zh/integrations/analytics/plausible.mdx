---
title: "Plausible"
---

在 `docs.json` 中添加站点域名，将分析数据发送到 Plausible。

<Info>

    域名或服务器地址中不要包含 `https://`。

</Info>

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Example
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
