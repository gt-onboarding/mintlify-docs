<div id="enabling-authentication">
  ## 启用身份验证
</div>

您可以在 `docs.json` 中添加一种身份验证方式，将其全局应用于每个页面；也可以为单个页面分别设置。

当同时设置了全局方式和页面方式时，页面的身份验证方式将覆盖全局设置。

<div id="bearer-token">
  ### Bearer 令牌
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "bearer"
      }
    }
}
```

```mdx Page Metadata
---
title: "页面标题"
authMethod: "bearer"
---
```

</CodeGroup>

<div id="basic-authentication">
  ### 基本身份认证
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "basic"
      }
    }
}
```

```mdx Page Metadata
---
title: "你的页面标题"
authMethod: "basic"
---
```

</CodeGroup>

<div id="api-key">
  ### API 密钥
</div>

<CodeGroup>

```json docs.json
"api": {
    "mdx": {
      "auth": {
        "method": "key",
        "name": "x-api-key"
      }
    }
}
```

```mdx Page Metadata
---
title: "页面标题"
authMethod: "key"
---
```

</CodeGroup>

<div id="none">
  ### None
</div>

在 docs.json 中设置默认身份验证后，可以使用 "none" 身份验证方法为某个特定端点禁用身份验证。

<CodeGroup>
```mdx Page Metadata
---
title: "Your page title"
authMethod: "none"
---
```
</CodeGroup>
