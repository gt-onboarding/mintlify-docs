---
title: "多种响应"
description: "为同一端点展示不同的响应示例"
---

如果你的 API 会根据输入参数、用户上下文或请求的其他条件返回不同的响应，你可以使用 `examples` 属性来记录多个响应示例。

该属性可以添加到任意响应中，其结构如下所示。

```yaml
responses:
  "200":
    description: 成功响应
    content:
      application/json:
        schema:
          $ref: "#/components/schemas/YourResponseSchema"
        examples:
          us:
            summary: 美国响应示例
            value:
              countryCode: "US"
              currencyCode: "USD"
              taxRate: 0.0825
          gb:
            summary: 英国响应示例
            value:
              countryCode: "GB"
              currencyCode: "GBP"
              taxRate: 0.20
```
