要在自定义域名上托管你的文档，你需要先在 Mintlify 设置中填写所需的自定义域名，并在你的域名服务商处配置 DNS 记录。

<Info>
  想要设置像 `mintlify.com/docs` 这样的自定义子目录？请参阅[自定义子目录](/zh/advanced/subpath/cloudflare)文档。
</Info>

<div id="dashboard-settings">
  ## 仪表板设置
</div>

1. 前往你的[仪表板](https://dashboard.mintlify.com)
2. 选择**设置（Settings）**
3. 选择**域名设置（Domain Setup）**
4. 输入你想使用的自定义域名，例如 `docs.mintlify.com`

<div id="configuring-your-dns">
  ## 配置 DNS
</div>

1. 前往你的域名服务商网站，打开该域名的 DNS 设置。
2. 新增一条 DNS 记录，填写如下内容：

```bash
CNAME | docs | cname.vercel-dns.com.
```

<div id="provider-specific-settings">
  ## 面向特定服务商的设置
</div>

<AccordionGroup>
  <Accordion title="Vercel verification">
    如果 Vercel 是你的域名服务商，你需要添加一个用于验证的 `TXT` 记录。提交自定义域名后，该信息会显示在你的仪表板中，并同时通过电子邮件发送给你。
  </Accordion>

  <Accordion title="Cloudflare security settings">
    如果 Cloudflare 是你的 DNS 服务商，你需要在 HTTPS 设置中启用“Full (strict)”安全选项。
  </Accordion>
</AccordionGroup>
