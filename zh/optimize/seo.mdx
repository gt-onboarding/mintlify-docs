Mintlify 会自动处理许多 SEO 最佳实践，包括：

* 自动生成 meta 标签
* 生成站点地图（Sitemap）和 `robots.txt` 文件
* 语义化 HTML 结构
* 移动端优化

你可以通过在 `docs.json` 或页面的 frontmatter 中添加 `metatags` 字段，全面自定义站点的 meta 标签。

<div id="global-meta-tags">
  ## 全局 meta 标签
</div>

要为所有页面设置默认的 meta 标签，请在你的 `docs.json` 中添加 `metatags` 字段。

```json
"seo": {
    "metatags": {
        "og:image": "默认元标签图片的链接"
    }
}
```

<div id="set-a-canonical-url">
  ### 设置规范 URL
</div>

如果你使用自定义域名，请设置 `canonical` 元标签，确保搜索引擎索引你首选的域名。规范 URL 会告知搜索引擎你的文档哪一个版本是权威版本。当文档可通过多个 URL 访问时，这有助于提升 SEO，并避免重复内容问题。

```json
"seo": {
    "metatags": {
        "canonical": "https://www.your-custom-domain-here.com"
    }
}
```

<div id="page-specific-meta-tags">
  ## 页面级别的 meta 标签
</div>

要设置页面级别的 meta 标签，请将它们添加到该页面的 frontmatter 中。

```mdx
---
title: "您的示例页面标题"
"og:image": "元标签图片链接"
---
```

<Note>
  含有冒号的 Meta 标签必须用引号包裹。
</Note>

<div id="sitemaps-and-robotstxt-files">
  ## 站点地图与 robots.txt 文件
</div>

Mintlify 会自动生成 `sitemap.xml` 和 `robots.txt` 文件。你可以在文档站点的 URL 末尾添加 `/sitemap.xml` 来查看站点地图。

默认情况下，只会收录 `docs.json` 中包含的页面。若要纳入隐藏链接，请在 `docs.json` 中添加 `seo.indexing`：

```json
"seo": {
    "indexing": all
}
```

<div id="custom-sitemaps-and-robotstxt-files">
  ### 自定义 sitemap 和 robots.txt 文件
</div>

要添加自定义的 `sitemap.xml` 或 `robots.txt` 文件，请在项目根目录创建相应的 `sitemap.xml` 或 `robots.txt` 文件。添加自定义文件会覆盖同名的自动生成文件。删除自定义文件后，将恢复使用默认文件。

<div id="disabling-indexing">
  ## 禁用索引
</div>

如果你想让搜索引擎不收录某个页面，可以在该页面的 [frontmatter](/zh/organize/pages) 中添加以下内容：

```
---
noindex: true
---
```

你也可以在 `docs.json` 中将 `metatags.robots` 字段设置为 `"noindex"`，以为文档中的所有页面指定 `noindex`：

```json
"seo": {
    "metatags": {
      "robots": "noindex"
    }
  }
```

<div id="seo-best-practices">
  ## SEO 最佳实践
</div>

<AccordionGroup>
  <Accordion title="撰写清晰的标题与摘要">
    * 使用清晰、具描述性的页面标题（50–60 个字符）
    * 撰写有吸引力的页面摘要（150–160 个字符）
    * 包含相关关键词
    * 确保每个页面的标题与摘要具有唯一性
  </Accordion>

  <Accordion title="优化内容结构">
    * 使用正确的标题层级（H1 → H2 → H3）
    * 优先为读者而写，其次才是搜索引擎
    * 在标题与正文中纳入相关关键词
    * 保持 URL 简短、清晰，并按层级组织
    * 用小标题和列表拆分长篇内容
  </Accordion>

  <Accordion title="内部链接策略">
    * 在文档内链接到相关页面
    * 使用具描述性的锚文本，避免使用“点击这里”
    * 通过链接相关概念构建主题集群
    * 利用自动交叉引用功能
  </Accordion>

  <Accordion title="图片 SEO">
    * 为图片使用具描述性的文件名
    * 始终添加 alt 文本以提升可访问性和 SEO
    * 优化图片体积以加快加载速度
    * 使用与内容相关、能支撑叙述的图片
  </Accordion>
</AccordionGroup>
