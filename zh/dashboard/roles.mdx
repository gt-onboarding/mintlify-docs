<Info>
  RBAC 功能适用于 [Enterprise 方案](https://mintlify.com/pricing?ref=rbac)。
</Info>

Mintlify 提供两种控制台访问级别：Editor 和 Admin。

以下操作仅限 Admin 角色：

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| 更新用户角色            |   ❌   |  ✅   |
| 删除用户                |   ❌   |  ✅   |
| 邀请管理员用户          |   ❌   |  ✅   |
| 管理与更新账单          |   ❌   |  ✅   |
| 更新自定义域名          |   ❌   |  ✅   |
| 更新 Git 源             |   ❌   |  ✅   |
| 删除组织                |   ❌   |  ✅   |

控制台上的其他操作对两种角色均可用。

你可以邀请任意数量的管理员，但我们建议仅将管理员访问权限授予确有需要的用户。
