---
title: "字体"
description: "配置自定义字体"
---

可为整个站点，或分别为标题与正文字体设置自定义字体。支持使用 Google Fonts、本地字体文件或外部托管的字体。默认字体因主题而异。

通过 `docs.json` 中的 [fonts 属性](/zh/organize/settings#param-fonts) 进行控制。

<div id="google-fonts">
  ## Google Fonts
</div>

在 `docs.json` 中指定字体系列名称后，Mintlify 会自动加载 [Google Fonts](https://fonts.google.com)。

```json docs.json
"fonts": {
  "family": "Inter"
}
```

<div id="local-fonts">
  ## 本地字体
</div>

要使用本地字体，请将字体文件放入项目目录，并在 `docs.json` 配置中进行引用。

<div id="setting-up-local-fonts">
  ### 设置本地字体
</div>

<Steps>
  <Step title="将字体文件添加到项目中">
    例如，创建一个 `fonts` 目录并添加字体文件：

    ```text
    your-project/
    ├── fonts/
    │   ├── InterDisplay-Regular.woff2
    │   └── InterDisplay-Bold.woff2
    ├── docs.json
    └── ...
    ```
  </Step>

  <Step title="在 docs.json 中配置字体">
    使用相对于项目根目录的相对路径引用本地字体：

    ```json docs.json
    {
      "fonts": {
        "family": "InterDisplay",
        "source": "/fonts/InterDisplay-Regular.woff2",
        "format": "woff2",
        "weight": 400
      }
    }
    ```
  </Step>
</Steps>

<div id="local-fonts-for-headings-and-body">
  ### 标题与正文的本地字体
</div>

在 `docs.json` 中分别为标题和正文文本配置本地字体：

```json docs.json
{
  "fonts": {
    "heading": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Bold.woff2",
      "format": "woff2",
      "weight": 700
    },
    "body": {
      "family": "InterDisplay",
      "source": "/fonts/InterDisplay-Regular.woff2",
      "format": "woff2",
      "weight": 400
    }
  }
}
```

<div id="externally-hosted-fonts">
  ## 外部托管字体
</div>

在 `docs.json` 中引用字体源 URL，即可使用外部托管字体：

```json docs.json
{
  "fonts": {
    "family": "Hubot Sans",
    "source": "https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2",
    "format": "woff2",
    "weight": 400
  }
}
```

<div id="font-configuration-reference">
  ## 字体配置参考
</div>

<ResponseField name="fonts" type="object">
  文档的字体配置。

  <Expandable title="Fonts">
    <ResponseField name="family" type="string" required>
      字体系列名称，例如“Inter”或“Playfair Display”。
    </ResponseField>

    <ResponseField name="weight" type="number">
      字重，例如 400 或 700。可变字体支持精确字重，如 550。
    </ResponseField>

    <ResponseField name="source" type="string (uri)">
      字体源的 URL，例如 `https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2`，或本地字体文件路径，例如 `/assets/fonts/InterDisplay.woff2`。当你指定 Google Fonts 的 `family` 名称时，会自动加载相应字体，因此无需提供 source URL。
    </ResponseField>

    <ResponseField name="format" type="'woff' | 'woff2'">
      字体文件格式。使用 `source` 字段时必填。
    </ResponseField>

    <ResponseField name="heading" type="object">
      覆盖仅用于标题的字体设置。

      <Expandable title="Heading">
        <ResponseField name="family" type="string" required>
          标题的字体系列名称。
        </ResponseField>

        <ResponseField name="weight" type="number">
          标题的字重。
        </ResponseField>

        <ResponseField name="source" type="string (uri)">
          字体源的 URL，例如 `https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2`，或用于标题的本地字体文件路径。当你指定 Google Fonts 的 `family` 名称时，会自动加载相应字体，因此无需提供 source URL。
        </ResponseField>

        <ResponseField name="format" type="'woff' | 'woff2'">
          标题的字体文件格式。使用 `source` 字段时必填。
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="body" type="object">
      覆盖仅用于正文的字体设置。

      <Expandable title="Body">
        <ResponseField name="family" type="string" required>
          正文的字体系列名称。
        </ResponseField>

        <ResponseField name="weight" type="number">
          正文的字重。
        </ResponseField>

        <ResponseField name="source" type="string (uri)">
          字体源的 URL，例如 `https://mintlify-assets.b-cdn.net/fonts/Hubot-Sans.woff2`，或用于正文的本地字体文件路径。当你指定 Google Fonts 的 `family` 名称时，会自动加载相应字体，因此无需提供 source URL。
        </ResponseField>

        <ResponseField name="format" type="'woff' | 'woff2'">
          正文的字体文件格式。使用 `source` 字段时必填。
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>
