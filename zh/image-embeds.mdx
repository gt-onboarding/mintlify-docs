<div id="images">
  ## 图片
</div>

在文档中添加图片，以提供视觉上下文、示例或装饰效果。

<div id="basic-image-syntax">
  ### 基本图片语法
</div>

使用 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#images)将图片添加到文档：

```mdx
![Alt text describing the image](/path/to/image.png)
```

<Tip>
  请务必提供具有描述性的替代文本（alt text），以提升可访问性和 SEO。替代文本应清晰说明图片的内容。
</Tip>

图片文件必须小于 20MB。对于更大的文件，请将其托管在 [Amazon S3](https://aws.amazon.com/s3) 或 [Cloudinary](https://cloudinary.com) 等 CDN 服务上。

<div id="html-image-embeds">
  ### HTML 图片嵌入
</div>

若需更精细地控制图片显示，请使用 HTML `<img>` 标签：

```html
<img 
  src="/images/dashboard.png" 
  alt="Main dashboard interface"
  height="300"
  className="rounded-lg"
/>
```

<div id="disable-zoom-functionality">
  #### 禁用缩放功能
</div>

要禁用图片点击时的默认缩放，请添加 `noZoom` 属性：

```html highlight="4"
<img 
  src="/images/screenshot.png" 
  alt="Descriptive alt text"
  noZoom 
  height="200"
/>
```

<div id="link-images">
  #### 链接图片
</div>

要将图片设为可点击的链接，请将图片包裹在 a（锚点）标签中，并添加 `noZoom` 属性：

```html
<a href="https://mintlify.com" target="_blank">
  <img 
    src="/images/logo.png" 
    alt="Mintlify logo"
    noZoom 
    height="100"
  />
</a>
```

<Note>
  放在锚点标签内的图片会自动显示指针光标，表示可点击。
</Note>

<div id="light-and-dark-mode-images">
  #### 浅色与深色模式图片
</div>

要在浅色与深色主题下展示不同图片，请使用 Tailwind CSS 类：

```html
<!-- 浅色模式图片 -->
<img 
  className="block dark:hidden" 
  src="/images/light-mode.png" 
  alt="浅色模式界面"
/>

<!-- 深色模式图片 -->
<img 
  className="hidden dark:block" 
  src="/images/dark-mode.png" 
  alt="深色模式界面"
/>
```

<div id="videos">
  ## 视频
</div>

Mintlify 支持在 Markdown 中使用 [HTML 标签](https://www.markdownguide.org/basic-syntax/#html)，为你提供创建丰富内容的灵活性。

<Tip>
  请始终在 video 元素内包含备用文本内容，以便在不支持视频播放的浏览器中显示。
</Tip>

<div id="youtube-embeds">
  ### 嵌入 YouTube
</div>

使用 iframe 元素嵌入 YouTube 视频：

```html
<iframe
  className="w-full aspect-video rounded-xl"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>
```

<Frame>
  <iframe className="w-full aspect-video rounded-xl" src="https://www.youtube.com/embed/4KzFe50RQkQ" title="YouTube video player" frameBorder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowFullScreen />
</Frame>

<div id="self-hosted-videos">
  ### 自托管视频
</div>

对于自托管的视频内容，请使用 HTML `<video>` 元素：

```html
<video
  controls
  className="w-full aspect-video rounded-xl"
  src="link-to-your-video.com"
></video>
```

<div id="autoplay-videos">
  ### 自动播放视频
</div>

要实现视频的自动播放，请使用：

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video rounded-xl"
  src="/videos/demo.mp4"
></video>
```

<Note>
  使用 JSX 语法时，将由两个单词组成的属性写为驼峰命名：`autoPlay`、`playsInline`、`allowFullScreen`。
</Note>

<div id="iframes">
  ## iframes
</div>

使用 iframe 元素来嵌入外部内容：

```html
<iframe 
  src="https://example.com/embed" 
  title="Embedded content"
  className="w-full h-96 rounded-xl"
></iframe>
```
