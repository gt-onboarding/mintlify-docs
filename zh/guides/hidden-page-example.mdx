此页面为隐藏页面！它未包含在 `docs.json` 的导航中，因此只能通过其 URL 访问。

更多信息请参见[隐藏页面](/zh/guides/hidden-pages)。

<div id="common-use-cases-for-hidden-pages">
  ## 隐藏页面的常见用例
</div>

<Note>
  默认情况下，隐藏页面可公开访问，但无法通过导航被发现。若要限制对隐藏页面的访问，必须配置[身份验证](/zh/authentication-personalization/authentication-setup)。
</Note>

- **Beta 文档**：内容可以公开，但不应通过导航被发现。
- **AI 工具的上下文**：如果隐藏页面被收录索引，AI 工具可以将其作为上下文参考。将对用户不直接相关、但能帮助 AI 工具提供更准确回答的内容放在隐藏页面中。
- **遗留页面**：通过直接链接保留旧内容的访问，同时将其从导航菜单中移除。
- **内部工具**：记录内部工具、预发布环境的 API 或开发流程。
