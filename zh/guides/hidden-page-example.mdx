此页面为隐藏页面！它未被纳入 `docs.json` 导航，因此只能通过其 URL 直接访问。

更多信息请参见[隐藏页面](/zh/guides/hidden-pages)。

<div id="common-use-cases-for-hidden-pages">
  ## 隐藏页面的常见用例
</div>

<Note>
  默认情况下，隐藏页面可公开访问，但不会通过导航被发现。若需限制对隐藏页面的访问，必须配置[身份验证](/zh/authentication-personalization/authentication-setup)。
</Note>

* **Beta 文档**：可以公开，但不应在导航中被发现的信息。
* **AI 工具的上下文**：如果隐藏页面被索引，AI 工具可以将其作为参考上下文。将对用户不相关、但能帮助 AI 工具提供更准确回答的内容放在隐藏页面中。
* **遗留页面**：通过直接链接保留旧内容的可访问性，同时将其从导航菜单中移除。
* **内部工具**：为内部工具、预发布 API 或开发流程编写文档。
