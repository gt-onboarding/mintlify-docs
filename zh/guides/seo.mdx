<div id="content-basics">
  ## 内容基础
</div>

让你的写作与结构便于搜索引擎快速扫描。

* **标题与副标题：** 使用有序且有意义的标题来组织内容。每个页面都有一个由 frontmatter 中 `title:` 属性生成的 H1。
* **短段落与项目符号：** 将大段文本拆分为易读的部分，并在合适的地方使用项目符号和编号列表。
* **内部链接：** 使用描述性链接文本指向相关内容。例如，用“进一步了解速率限制（rate limiting）”，而不是“点击这里”。

<div id="technical-seo-basics">
  ## 技术 SEO 基础
</div>

在优化内容之后，还要确保你的文档在技术层面表现良好。

以下基础技术 SEO 实践有助于提升文档的可发现性：

* **Meta 标签与描述：** 为每个页面撰写符合 SEO 的标题（50–60 个字符）和描述（150–160 个字符）。大多数[Meta 标签](/zh/optimize/seo)会自动生成。
* **图像替代文本（alt text）：** 为图片提供包含相关关键词的描述性替代文本。例如，使用“OAuth 2.0 API 身份验证流程”，而不是只写“diagram（图示）”。这有助于提升 SEO 和可访问性。
* **站点地图（sitemap）：** 确保站点地图为最新。Mintlify 会自动生成站点地图；如果你需要自定义格式，也可手动创建。创建后，搜索引擎会随着时间收录站点地图；你也可以将其直接提交到 Google Search Console 以加快收录。

<div id="page-performance">
  ## 页面性能
</div>

使用 [Google PageSpeed Insights](https://pagespeed.web.dev) 等工具识别可改进的技术 SEO 领域。

更高级的优化示例：

* **媒体加速优化：** 使用 WebP、AVIF 等格式压缩图片，并确保页面快速加载（理想情况下不超过 3 秒）。
* **结构化数据（Schema 标注）：** 在页面中添加 Schema 标注（如 HowTo、FAQ），帮助搜索引擎更好理解并提升内容的排名。

<div id="keyword-research">
  ## 关键词研究
</div>

要提升有机流量，请花时间了解哪些关键词能帮助用户找到你的文档。

* **关键词研究：** 使用免费的工具，如 [Google Keyword Planner](https://ads.google.com/intl/en_us/home/tools/keyword-planner/) 或 [Keywords Everywhere](https://keywordseverywhere.com) 来识别常见词组或长尾关键词。
* **自然融入关键词：** 将关键词自然地运用在标题、副标题和正文中。避免关键词堆砌。你的文档应该为用户而写，而不是为搜索引擎。
