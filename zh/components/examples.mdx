`<RequestExample>` 和 `<ResponseExample>` 组件会在右侧边栏显示代码块，形成双栏布局，让示例在用户滚动阅读内容时始终可见。这些组件为 API 文档而设计，但适用于所有页面。

常见用例：

* 包含请求与响应示例的 API 端点文档
* 与说明性文本并排的配置示例
* 用户在按照指引操作时参考的代码示例
* 教程中的前后对比示例

在移动设备上，`<RequestExample>` 和 `<ResponseExample>` 组件会显示为常规代码块，可直接滑动浏览。

<RequestExample>
  ```bash Request
    curl --request POST \
      --url https://dog-api.kinduff.com/api/facts
  ```
</RequestExample>

<ResponseExample>
  ```json Response
  { "status": "success" }
  ```
</ResponseExample>

<div id="requestexample">
  ## RequestExample
</div>

使用 `<RequestExample>` 将代码示例固定在右侧边栏。该组件与 [CodeGroup](/zh/components/code-groups) 组件类似，但会将代码显示在侧边栏而不是内联展示。

你可以在单个 `<RequestExample>` 中包含多个代码块。每个代码块都必须带有 title 属性。

````mdx RequestExample
<RequestExample>

```bash Request
  curl --request POST \
    --url https://dog-api.kinduff.com/api/facts
```

</RequestExample>
````

<div id="responseexample">
  ## ResponseExample
</div>

`<ResponseExample>` 组件会将代码示例固定在右侧边栏，显示在同一页面中任何 `<RequestExample>` 内容的下方。

````mdx ResponseExample
<ResponseExample>

```json Response
{ "status": "success" }
```

</ResponseExample>
````
