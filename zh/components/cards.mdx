title: "卡片"

使用卡片为内容创建可视化容器。卡片是一种灵活的容器，可包含文本、图标、图片和链接。

<div id="basic-card">
  ## 基础卡片
</div>

<Card title="Card title" icon="text" href="/zh/components/columns">
  这是使用带有图标和链接的卡片的方法。点击此卡片
  将跳转到 Columns 页面。
</Card>

```mdx Card example
<Card title="Card title" icon="text" href="/components/columns">
  This is how you use a card with an icon and a link. Clicking on this card
  brings you to the Columns page.
</Card>
```

<div id="card-variations">
  ## 卡片变体
</div>

卡片提供多种布局和样式选项，可满足不同内容需求。

<div id="horizontal-layout">
  ### 水平布局
</div>

添加 `horizontal` 属性，以更紧凑的水平布局展示卡片。

<Card title="Horizontal card" icon="text" horizontal>
  这是一个水平卡片的示例。
</Card>

```mdx Horizontal card example
<Card title="Horizontal card" icon="text" horizontal>
  This is an example of a horizontal card.
</Card>
```

<div id="image-cards">
  ### 图片卡片
</div>

添加 `img` 属性即可在卡片顶部显示图片。

<Card title="图片卡片" img="https://mintlify-assets.b-cdn.net/yosemite.jpg">
  这是一个带图片的卡片示例。
</Card>

```mdx Image card example
<Card title="Image card" img="/images/card-with-image.png">
  This is an example of a card with an image.
</Card>
```

<div id="link-cards-with-custom-ctas">
  ### 带自定义 CTA 的链接卡片
</div>

你可以自定义号召性用语（CTA）文本，并控制是否显示箭头。默认情况下，箭头仅在外部链接上显示。

<Card title="Link card" icon="link" href="/zh/components/columns" arrow="true" cta="Click here">
  这是一个带有图标和链接的卡片示例。点击此卡片会跳转到 Columns 页面。
</Card>

```mdx Link card example
<Card
  title="Link card"
  icon="link"
  href="/components/columns"
  arrow="true"
  cta="Click here"
>
  {/* 这是一个带有图标和链接的卡片示例。点击此卡片会跳转到 Columns 页面。 */}
</Card>
```

<div id="grouping-cards">
  ## 分组卡片
</div>

使用 [Columns 组件](/zh/components/columns) 将多个卡片并排排列。

<Columns cols={2}>
  <Card title="First card" icon="panel-left-close">
    这是第一张卡片。
  </Card>

  <Card title="Second card" icon="panel-right-close">
    这是第二张卡片。
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="First card" icon="panel-left-close">
    This is the first card.
  </Card>
  <Card title="Second card" icon="panel-right-close">
    This is the second card.
  </Card>
</Columns>
```

<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string" required>
  卡片上显示的标题
</ResponseField>

<IconsOptional />

<ResponseField name="color" type="string">
  图标颜色，使用十六进制色值（例如 `#FF6B6B`）。
</ResponseField>

<ResponseField name="href" type="string">
  点击卡片时跳转的 URL。
</ResponseField>

<ResponseField name="horizontal" type="boolean">
  以紧凑的横向布局显示卡片。
</ResponseField>

<ResponseField name="img" type="string">
  卡片顶部显示的图片的 URL 或本地路径。
</ResponseField>

<ResponseField name="cta" type="string">
  操作按钮的自定义文本。
</ResponseField>

<ResponseField name="arrow" type="boolean">
  显示或隐藏链接箭头图标。
</ResponseField>
