---
title: "列"
description: "以网格形式并排显示卡片"
keywords: ["卡片分组"]
---

`Columns` 组件用于将多个 `Card` 组件分组。它通常通过指定网格列数，将卡片排布为网格。

<Columns cols={2}>
  <Card title="快速开始" icon="rocket">
    按照我们的快速入门指南设置你的项目。
  </Card>

  <Card title="API 参考" icon="code">
    浏览你的 API 的端点、参数和示例。
  </Card>
</Columns>

```mdx Columns example
<Columns cols={2}>
  <Card title="快速开始">
    通过我们的快速入门指南设置您的项目。
  </Card>
  <Card title="API 参考">
    浏览 API 的端点、参数和示例。
  </Card>
</Columns>
```


<div id="properties">
  ## 属性
</div>

<ResponseField name="cols" default={2}>
  每行的列数。
</ResponseField>
