---
title: "可展开面板"
description: "切换以显示嵌套属性"
---

使用可展开面板在响应字段中显示或隐藏嵌套内容。在 API 文档中展示复杂对象的属性时，可展开面板尤其有用。

<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      用户的全名
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      用户是否已满 21 岁
    </ResponseField>
  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      用户的全名
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      用户是否年满21岁
    </ResponseField>
  </Expandable>
</ResponseField>
```


<div id="properties">
  ## 属性
</div>

<ResponseField name="title" type="string">
  要显示的对象名称。
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  设为 `true`，页面加载时即展开
</ResponseField>
