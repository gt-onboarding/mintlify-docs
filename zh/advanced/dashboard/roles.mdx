title: "角色"
description: "通过角色控制对仪表板的访问。"
---

<Info>
  RBAC 功能适用于 [企业版计划](https://mintlify.com/pricing?ref=rbac)。
</Info>

Mintlify 提供两种仪表板访问级别：Editor 和 Admin。

以下为仅限 Admin 角色执行的操作：

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| 更新用户角色            |   ❌   |  ✅   |
| 删除用户                |   ❌   |  ✅   |
| 邀请管理员              |   ❌   |  ✅   |
| 管理和更新结算          |   ❌   |  ✅   |
| 更新自定义域名          |   ❌   |  ✅   |
| 更新 Git 来源           |   ❌   |  ✅   |
| 删除组织                |   ❌   |  ✅   |

仪表板上的其他操作对两种角色均可用。

你可以邀请任意数量的管理员，但我们建议将管理员访问权限仅授予确有需要的用户。
