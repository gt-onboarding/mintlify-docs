Le fichier [llms.txt](https://llmstxt.org) est une norme du secteur qui aide les LLM à indexer le contenu plus efficacement, à l’instar d’un plan de site pour les moteurs de recherche. Les outils d’IA peuvent utiliser ce fichier pour comprendre la structure de votre documentation et trouver le contenu pertinent pour les requêtes des utilisateurs.

Mintlify héberge automatiquement, à la racine de votre projet, un fichier `llms.txt` qui répertorie toutes les pages disponibles de votre documentation. Ce fichier est toujours à jour et ne nécessite aucune maintenance. Vous pouvez également ajouter un fichier `llms.txt` personnalisé à la racine de votre projet.

Affichez votre `llms.txt` en ajoutant `/llms.txt` à l’URL de votre site de documentation.

<div id="llmstxt-structure">
  ## Structure de llms.txt
</div>

Un fichier `llms.txt` est un fichier Markdown simple qui contient :

* **Titre du site** comme titre H1.
* **Sections de contenu structurées** avec des liens et une description de chaque page de votre documentation.

Les pages sont répertoriées par ordre alphabétique selon l’ordre dans lequel elles apparaissent dans votre dépôt, en commençant à partir du répertoire racine.

La description de chaque page provient du champ `description` de son front matter. Les pages sans champ `description` apparaissent dans le fichier `llms.txt` sans description.

```mdx Example llms.txt
# Titre du site

## Documentation

- [API](https://example.com/docs/api) : Liste des endpoints et utilisation
- [Installation](https://example.com/docs/install) : Étapes d'installation
- [Démarrage](https://example.com/docs/start) : Guide d'introduction
```

Cette approche structurée permet aux LLM de traiter efficacement votre documentation à un niveau global et d’identifier le contenu pertinent pour les requêtes des utilisateurs, ce qui améliore la précision et la rapidité des recherches de documentation assistées par l’IA.


<div id="llms-fulltxt">
  ## llms-full.txt
</div>

Le fichier `llms-full.txt` regroupe l’intégralité de votre site de documentation en un seul fichier pour servir de contexte aux outils d’IA et il est indexé par le trafic des LLM.

Mintlify héberge automatiquement un fichier `llms-full.txt` à la racine de votre projet. Affichez votre `llms-full.txt` en ajoutant `/llms-full.txt` à l’URL de votre site de documentation.

<PreviewButton href="https://mintlify.com/docs/llms-full.txt">Ouvrir le fichier llms-full.txt de ce site</PreviewButton>

<div id="custom-files">
  ## Fichiers personnalisés
</div>

Pour ajouter un fichier `llms.txt` ou `llms-full.txt` personnalisé, créez un fichier `llms.txt` ou `llms-full.txt` à la racine de votre projet. L’ajout d’un fichier personnalisé remplacera le fichier généré automatiquement portant le même nom. Si vous supprimez un fichier personnalisé, le fichier par défaut sera utilisé à nouveau.

Votre fichier `llms.txt` ou `llms-full.txt` personnalisé doit comporter le titre du site en tant que titre H1. Le reste du contenu est facultatif. Consultez la section [Format](https://llmstxt.org/#format) de la spécification `llms.txt` pour plus d’informations sur les sections optionnelles et les bonnes pratiques.
