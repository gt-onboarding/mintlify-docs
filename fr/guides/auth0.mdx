</Note>

<div id="overview">
  ## Aperçu
</div>

Ce guide vous explique comment configurer Auth0 en tant que fournisseur d’authentification pour votre documentation Mintlify. À la fin, vos utilisateurs pourront se connecter à votre documentation avec leurs identifiants Auth0.

<Steps>
  <Step title="Créer une application Web classique dans Auth0">
    Connectez-vous à votre tableau de bord Auth0 et accédez à **Applications** > **Applications**. Cliquez sur **Create Application**, donnez un nom à votre application (par exemple « Mintlify ») et sélectionnez **Regular Web Applications** comme type d’application. Cliquez ensuite sur **Create**.
    
    <Frame>![Création d'une nouvelle application dans Auth0](/images/guides/auth0/auth0-create-application.png)</Frame>
  </Step>
  <Step title="Obtenir les informations client">
    Après avoir créé votre application, vous serez redirigé vers la page des paramètres de l’application. Vous y trouverez les identifiants essentiels nécessaires à l’intégration OAuth :
    
    <Frame>![Page des paramètres client Auth0](/images/guides/auth0/auth0clientsettings.png)</Frame>
    
    Notez les informations suivantes :
    - **Domain**: le domaine de votre tenant Auth0 (par exemple, `your-tenant.auth0.com`)
    - **Client ID**: l’identifiant public de votre application
    - **Client Secret**: la clé secrète de votre application (gardez-la confidentielle)
    
    Vous aurez besoin de ces valeurs pour configurer Mintlify à l’étape suivante.
  </Step>
  <Step title="Configurer le client Mintlify">
    Accédez à votre tableau de bord Mintlify et ouvrez **Settings** > **Authentication**. Sélectionnez **OAuth** comme méthode d’authentification : le formulaire de configuration OAuth s’affiche.
    
    <Frame>![Paramètres du client OAuth Mintlify](/images/guides/auth0/mintlifyoauthclientsettings.png)</Frame>
    
    Renseignez le formulaire avec les valeurs suivantes :
    
    - **Authorization URL**: `https://YOUR_AUTH0_DOMAIN/authorize` (remplacez `YOUR_AUTH0_DOMAIN` par votre domaine Auth0 de l’étape 2)
    - **Client ID**: saisissez le Client ID de votre application Auth0
    - **Client Secret**: saisissez le Client Secret de votre application Auth0
    - **Scopes**: laissez vide, sauf si vous avez défini des scopes personnalisés dans Auth0
    - **Token URL**: `https://YOUR_AUTH0_DOMAIN/oauth/token` (remplacez `YOUR_AUTH0_DOMAIN` par votre domaine Auth0)
    
    Après avoir renseigné ces champs, cliquez sur **Save changes** pour enregistrer votre configuration OAuth.
  </Step>
  <Step title="Configurer l’URL de rappel (Callback URL)">
    Mintlify générera une URL de redirection unique qu’Auth0 doit reconnaître pour que le flux OAuth fonctionne correctement.
    
    Copiez l’URL de redirection depuis la section Authentication de votre tableau de bord Mintlify :
    <Frame>![URL de redirection Mintlify](/images/guides/auth0/mintlifyredirecturl.png)</Frame>
    
    Retournez sur la page des paramètres de votre application Auth0, faites défiler jusqu’à la section **Application URIs**, puis collez l’URL de redirection dans le champ **Allowed Callback URLs** :
    <Frame>![Configuration de l'URL de redirection Auth0](/images/guides/auth0/auth0redirecturl.png)</Frame>
    
    Cliquez sur **Save Changes** en bas de la page Auth0 pour appliquer cette configuration.
  </Step>
</Steps>
