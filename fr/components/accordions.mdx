Les accordéons permettent aux utilisateurs d’ouvrir et de replier des sections de contenu. Utilisez des accordéons pour une divulgation progressive et pour organiser l’information.


<div id="single-accordion">
  ## Accordéon simple
</div>

<Accordion title="Je suis un accordéon.">
  Vous pouvez placer n'importe quel contenu ici, y compris d'autres composants, comme du code :
   ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
</Accordion>

<div id="accordion-groups">
  ## Groupes d’accordions
</div>

Regroupez des accordions liés à l’aide de `<AccordionGroup>`. Cela crée une section cohérente d’accordions qui peuvent être déployés ou repliés individuellement.

<AccordionGroup>
  <Accordion title="Getting started">
    Vous pouvez insérer d’autres composants à l’intérieur des accordions.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="Advanced features" icon="bot">
    Ajoutez des icônes pour rendre les accordions plus distinctifs visuellement et faciles à parcourir.
  </Accordion>
  
  <Accordion title="Troubleshooting">
    Gardez le contenu connexe organisé en groupes.
  </Accordion>
</AccordionGroup>

<div id="properties">
  ## Propriétés
</div>

<ResponseField name="title" type="string" required>
  Titre dans l’aperçu de l’accordéon.
</ResponseField>

<ResponseField name="description" type="string">
  Détail sous le titre dans l’aperçu de l’accordéon.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Indique si l’accordéon est ouvert par défaut.
</ResponseField>
