---
title: "Bannière"
description: "Ajoutez une bannière pour afficher des annonces et notifications importantes à l’échelle du site"
---

Utilisez des bannières pour diffuser des annonces, des mises à jour ou des notifications importantes sur l’ensemble de votre site de documentation. Les bannières s’affichent en haut de chaque page, prennent en charge le formatage Markdown et peuvent être rendues ignorables. Elles utilisent la couleur définie par la propriété `colors.dark` dans votre `docs.json`.

Pour ajouter une bannière, utilisez la propriété `banner` dans votre `docs.json` :

<CodeGroup>
  ```json Product announcements wrap
  "banner": {
    "content": "🚀 La version 2.0 est maintenant disponible ! Consultez notre [changelog](/changelog) pour plus de détails.",
    "dismissible": true 
  }
  ```

  ```json Maintenance notices wrap
  "banner": {
    "content": "⚠️ Maintenance programmée : l’API sera indisponible le 15 décembre, de 2 h à 4 h UTC",
    "dismissible": false
  }
  ```

  ```json Required actions wrap
  "banner": {
    "content": "**Action requise :** migrez vers notre nouvelle version avant le 1er janvier. [Guide de migration](/migration)",
    "dismissible": true
  }
  ```
</CodeGroup>

<div id="properties">
  ## Propriétés
</div>

<ResponseField name="content" type="string" required>
  Le message de la bannière. Compatible avec le texte brut et la mise en forme Markdown.
</ResponseField>

<ResponseField name="dismissible" type="boolean">
  Indique si les utilisateurs peuvent fermer la bannière. Lorsque `true`, les utilisateurs peuvent fermer la bannière et elle ne réapparaît pas pendant leur session. Par défaut : `false`.
</ResponseField>
