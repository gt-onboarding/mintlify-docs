<div id="inline-icons">
  ## Icônes en ligne
</div>

Les icônes sont insérées dans le flux du texte lorsqu’elles sont utilisées au sein d’un paragraphe.

<Icon icon="flag" iconType="solid" /> La documentation dont vous avez besoin, en toute simplicité.

<ResponseField name="color" type="string">
  La couleur de l’icône en code hexadécimal (par exemple « #FF5733 »).
</ResponseField>

<ResponseField name="size" type="number">
  La taille de l’icône en pixels.
</ResponseField>
