---
title: "Onglets"
description: "Afficher/masquer du contenu avec le composant Tabs"
---

Utilisez des onglets pour organiser le contenu en plusieurs panneaux entre lesquels les utilisateurs peuvent passer. Vous pouvez ajouter autant d’onglets que nécessaire et inclure d’autres composants dans chaque onglet.

<Tabs>
  <Tab title="Premier onglet">
    ☝️ Bienvenue dans le contenu visible uniquement dans le premier onglet.

    Vous pouvez ajouter autant de composants que vous le souhaitez à l’intérieur des onglets. Par exemple, un bloc de code :

    ```java HelloWorld.java
      class HelloWorld {
          public static void main(String[] args) {
              System.out.println("Hello, World!");
          }
      }
    ```
  </Tab>

  <Tab title="Deuxième onglet" icon="leaf">
    ✌️ Voici du contenu présent uniquement dans le deuxième onglet.

    Celui-ci comporte une icône <Icon icon="leaf" /> !
  </Tab>

  <Tab title="Troisième onglet">
    💪 Voici du contenu présent uniquement dans le troisième onglet.
  </Tab>
</Tabs>

````mdx Tabs example
<Tabs>
````

<div id="properties">
  ## Propriétés
</div>

<ResponseField name="title" type="string" required>
  Le titre de l’onglet. Les titres courts sont plus faciles à parcourir.
</ResponseField>

<ResponseField name="icon" type="string">
  Une icône [Font Awesome](https://fontawesome.com/icons), une icône [Lucide](https://lucide.dev/icons), une URL d’icône ou un chemin relatif vers une icône.
</ResponseField>

<ResponseField name="iconType" type="string">
  Pour les icônes Font Awesome uniquement : l’un de `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`.
</ResponseField>
