---
title: "Groupes de code"
description: "Afficher plusieurs exemples de code dans un seul composant"
---

Utilisez le composant `CodeGroup` pour afficher plusieurs blocs de code dans une interface à onglets, afin de permettre aux utilisateurs de comparer des implémentations dans différents langages de programmation ou d’examiner des approches alternatives pour une même tâche.

<CodeGroup>
  ```javascript helloWorld.js
  console.log("Hello World");
  ```

  ```python hello_world.py
  print('Hello World!')
  ```

  ```java HelloWorld.java
  class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
</CodeGroup>

Les groupes de code héritent du style global depuis votre fichier `docs.json`. Personnalisez votre thème avec `styling.codeblocks`. Voir [Settings](/fr/organize/settings#styling) pour les options de configuration.

<div id="creating-code-groups">
  ## Créer des groupes de code
</div>

Pour créer un groupe de code, entourez plusieurs blocs de code avec des balises `<CodeGroup>`. Chaque bloc de code doit inclure un titre, qui devient le libellé de l’onglet.

````mdx
<CodeGroup>

```javascript helloWorld.js
console.log("Hello World");
```

```python hello_world.py
print('Hello World!')
```

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

</CodeGroup>
````

<div id="language-dropdown">
  ## Menu déroulant de langue
</div>

Vous pouvez remplacer les onglets d’un groupe de code par un menu déroulant pour passer d’une langue à l’autre en utilisant la prop `dropdown`.

<CodeGroup dropdown>
  ```javascript helloWorld.js
  console.log("Hello World");
  ```

  ```python hello_world.py
  print('Hello World!')
  ```

  ```java HelloWorld.java
  class HelloWorld {
      public static void main(String[] args) {
          System.out.println("Hello, World!");
      }
  }
  ```
</CodeGroup>
