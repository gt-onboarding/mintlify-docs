---
title: "Encadrés"
description: "Utilisez des encadrés pour ajouter un contexte percutant à votre contenu"
icon: "info"
---

Les encadrés peuvent être affichés en Note, Avertissement, Info, Astuce, Validation, Danger, ou vous pouvez créer votre propre encadré :

<Note>Ajoute une note dans le contenu</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>Signale un avertissement à prendre en compte</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Met en avant des informations importantes</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>Propose un conseil utile</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Indique un statut validé</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>Encadré de danger</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">Encadré personnalisé</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
