---
title: "Encadrés"
description: "Utilisez des encadrés pour ajouter un contexte percutant à votre contenu"
---

Les encadrés peuvent être de type Note, Avertissement, Info, Astuce, Validation, Danger, ou vous pouvez créer votre propre encadré :

<Note>Ajoute une note dans le contenu</Note>

```mdx
<Note>Ceci ajoute une note au contenu</Note>
```

<Warning>Cela génère un avertissement à prendre en compte</Warning>

```mdx
<Warning>Ceci signale un avertissement important</Warning>
```

<Info>Met en évidence des informations importantes</Info>

```mdx
<Info>Ceci met en évidence des informations importantes</Info>
```

<Tip>Voici une astuce utile</Tip>

```mdx
<Tip>Ceci indique un conseil utile</Tip>
```

<Check>Cela nous donne un statut validé</Check>

```mdx
<Check>Ceci affiche un statut coché</Check>
```

<Danger>Ceci est une alerte de danger</Danger>

```mdx
<Danger>Ceci est un avertissement de danger</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Ceci est un appel de note personnalisé</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Ceci est un encadré personnalisé</Callout>
```
