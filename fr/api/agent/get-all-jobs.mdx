openapi: GET /agent/{projectId}/jobs

<div id="usage">
  ## Utilisation
</div>

Cet endpoint récupère tous les travaux d’agent pour le domaine spécifié, offrant une vue d’ensemble de l’ensemble des activités des agents et de leur statut actuel. Cela est utile pour surveiller et gérer plusieurs travaux d’agent simultanés ou historiques.

<div id="response">
  ## Réponse
</div>

Utilisez cet endpoint pour obtenir une vue d’ensemble complète de toutes les sessions d’agent précédentes.
