<div id="lists">
  ## Listes
</div>

Les listes suivent la [syntaxe Markdown officielle](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Liste ordonnée
</div>

Pour créer une liste ordonnée, ajoutez des nombres suivis d’un point avant chaque élément.

1. Premier élément
2. Deuxième élément
3. Troisième élément
4. Quatrième élément

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### Liste à puces
</div>

Pour créer une liste à puces, ajoutez des tirets (`-`), des astérisques (`*`) ou des signes plus (`+`) avant les éléments de la liste.

- Premier élément
- Deuxième élément
- Troisième élément
- Quatrième élément

```mdx
- First item
- Second item
- Third item
- Fourth item
```

<div id="nested-list">
  ### Liste imbriquée
</div>

Indentez les éléments de la liste pour les imbriquer.

- Premier élément
- Deuxième élément
  - Élément supplémentaire
  - Élément supplémentaire
- Troisième élément

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## Tableaux
</div>

Les tableaux suivent la [syntaxe Markdown officielle](https://www.markdownguide.org/extended-syntax/#tables).

Pour ajouter un tableau, utilisez trois tirets ou plus (`---`) pour créer l’en-tête de chaque colonne, et utilisez des barres verticales (`|`) pour séparer chaque colonne. Par compatibilité, ajoutez également une barre verticale au début et à la fin de chaque ligne.

| Propriété | Description                           |
| --------- | ------------------------------------- |
| Nom       | Nom complet de l’utilisateur          |
| Âge       | Âge déclaré                           |
| Inscrit   | Indique si l’utilisateur a rejoint la communauté |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
