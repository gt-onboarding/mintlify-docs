---
title: "Page 404 personnalisée"
description: "Personnalisez le titre et la description de votre page d’erreur 404"
---

Vous pouvez définir le titre et la description de la page d’erreur 404 qui s’affiche lorsque les utilisateurs arrivent sur des liens brisés ou inexistants.

Lors de la personnalisation de votre page 404, utilisez la description pour orienter les utilisateurs vers des ressources utiles ou des liens dans votre documentation afin de les aider à trouver ce qu’ils cherchent.

<div id="configuration">
  ## Configuration
</div>

Configurez votre page 404 dans la section `errors.404` de votre fichier `docs.json` :

```json
"errors": {
  "404": {
    "redirect": false,
    "title": "Page introuvable",
    "description": "Qu'est-il **arrivé** à cette _page_ ?"
  }
}
```


<div id="parameters">
  ## Paramètres
</div>

<ResponseField name="redirect" type="boolean">
  Indique s’il faut rediriger automatiquement vers la page d’accueil lorsqu’une page est introuvable.
  
  Définissez sur `true` pour rediriger vers la page d’accueil.

  Définissez sur `false` pour afficher la page 404 personnalisée.
</ResponseField>

<ResponseField name="title" type="string">
  Titre personnalisé de la page d’erreur 404. Remplace l’intitulé par défaut « Page not found ».
</ResponseField>

<ResponseField name="description" type="string">
  Description personnalisée de la page d’erreur 404. Prend en charge la mise en forme Markdown.
</ResponseField>
