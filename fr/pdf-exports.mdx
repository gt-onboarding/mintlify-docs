---
title: "Export PDF"
description: "Exportez votre documentation dans un seul fichier PDF"
---

<Info>
Les exports PDF sont disponibles avec les [Custom plans](https://mintlify.com/pricing).
</Info>

Vous pouvez exporter votre documentation dans un seul fichier PDF pour une consultation hors ligne. Le PDF inclura tout le contenu de la documentation, y compris les images et les liens, ainsi qu’une table des matières navigable.

<div id="exporting-a-pdf">
  ## Exporter un PDF
</div>

1. Accédez à la page [General](https://dashboard.mintlify.com/settings/deployment/general) dans la section Paramètres du projet de votre tableau de bord.
2. Cliquez sur le bouton **Exporter tout le contenu**.
3. Vous pouvez, si vous le souhaitez, personnaliser les options d’export :
    - **Format de page** : Choisissez la taille de page du PDF.
    - **Pourcentage d’échelle** : Ajustez l’échelle du PDF.
    - **Inclure le pied de page** : Ajoute un pied de page avec le numéro de page et le nombre total de pages.
4. Cliquez sur **Exporter** pour lancer l’export.
5. Une fois l’export terminé, vous recevrez un e‑mail avec un lien de téléchargement.
6. Cliquez sur le lien de téléchargement pour récupérer le fichier PDF.
