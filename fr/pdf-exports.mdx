<Info>
Les exports PDF sont disponibles avec les [offres personnalisées](https://mintlify.com/pricing).
</Info>

Vous pouvez exporter votre documentation en un seul fichier PDF pour une consultation hors ligne. Le PDF inclura l’intégralité du contenu de la documentation, y compris les images et les liens, ainsi qu’une table des matières navigable.

<div id="exporting-a-pdf">
  ## Exportation d’un PDF
</div>

1. Accédez à la page [General](https://dashboard.mintlify.com/settings/deployment/general) dans la section Paramètres du projet de votre tableau de bord.
2. Cliquez sur le bouton **Exporter tout le contenu**.
3. Vous pouvez, si vous le souhaitez, personnaliser les options d’exportation :
    - **Format de page** : Choisissez la taille des pages du PDF.
    - **Pourcentage d’échelle** : Ajustez l’échelle du PDF.
    - **Inclure le pied de page** : Ajoutez un pied de page avec le numéro de page et le nombre total de pages.
4. Cliquez sur **Exporter** pour lancer le processus d’exportation.
5. Une fois l’exportation terminée, vous recevrez un e-mail avec un lien de téléchargement.
6. Cliquez sur le lien pour télécharger le fichier PDF.
