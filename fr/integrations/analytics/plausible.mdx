---
title: "Plausible"
---

Ajoutez le domaine de votre site dans `docs.json` pour envoyer les données d’analytique à Plausible.

<Info>

    N’incluez pas `https://` dans le domaine ou le serveur.

</Info>

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "plausible": {
        "domain": "required",
        "server": "optional"
    }
}
```

```json Example
"integrations": {
    "plausible": {
        "domain": "docs.domain.com"
    }
}
```

</CodeGroup>
