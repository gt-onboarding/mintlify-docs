---
title: "Fathom"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données d’analyse à Fathom.

Vous pouvez récupérer le `siteId` depuis les paramètres de votre script.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Example
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
