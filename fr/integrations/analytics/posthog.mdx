---
title: "PostHog"
---

Ajoutez ce qui suit à votre fichier `docs.json` pour envoyer des données d’analyse à PostHog.

Vous n’avez besoin d’inclure `apiHost` que si vous auto-hébergez PostHog. Par défaut, nous envoyons les événements à `https://app.posthog.com`.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "posthog": {
        "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
        "apiHost": "optional"
    }
}
```

```json Example
"integrations": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<br />

<Warning>
    L’activation de PostHog désactivera les statistiques sur le tableau de bord Mintlify.
</Warning>

<div id="session-recordings">
  ## Enregistrements de session
</div>

Avant de pouvoir recevoir des enregistrements de session, vous devez ajouter l’URL de votre site de documentation aux « domaines autorisés pour les enregistrements » de PostHog. L’option permettant d’ajouter votre URL se trouve dans les paramètres du projet de PostHog.
