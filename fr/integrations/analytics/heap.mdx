---
title: "Heap"
---

Ajoutez l’extrait suivant à votre fichier `docs.json` pour envoyer des données d’analyse à Heap.

<CodeGroup>

```json Options d’analyse dans docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Exemple
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
