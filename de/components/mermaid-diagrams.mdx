[Mermaid](https://mermaid.js.org/) ermöglicht es, Flowcharts, Sequenzdiagramme, Gantt-Diagramme und andere Diagramme mit Text und Code zu erstellen.

Eine vollständige Liste der unterstützten Diagrammtypen und der Syntax finden Sie in der [Mermaid-Dokumentation](https://mermaid.js.org/intro/).

<RequestExample>

````mdx Mermaid flowchart example
```mermaid
  flowchart LR
    subgraph subgraph1
        direction TB
        top1[top] --> bottom1[bottom]
    end
    subgraph subgraph2
        direction TB
        top2[top] --> bottom2[bottom]
    end
    %% ^ Diese Subgraphs sind identisch, abgesehen von den Verknüpfungen zu ihnen:

    %% Verknüpfung *zu* subgraph1: Die Richtung von subgraph1 bleibt erhalten
    outside --> subgraph1
    %% Verknüpfung *innerhalb* von subgraph2:
    %% subgraph2 übernimmt die Richtung des obersten Graphen (LR)
    outside ---> top2
```
````

  ## Syntax

Um ein Mermaid-Diagramm zu erstellen, schreiben Sie die Diagrammdefinition in einen Mermaid-Codeblock.

````mdx
```mermaid
// Hier kommt Ihr Mermaid-Diagrammcode hin
```
````
