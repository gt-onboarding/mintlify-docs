title: "Antwortfelder"
description: "API-Antwortwerte anzeigen"
---

Die Komponente `<ResponseField>` ist dafür vorgesehen, die Rückgabewerte einer API zu definieren. Viele Dokumentationen setzen `<ResponseField>` auch auf Seiten ein, wenn Typen aufgelistet werden sollen.

</ResponseField>

<div id="props">
  ## Props
</div>

<ResponseField name="name" type="string" required>
  Der Name des Response-Werts.
</ResponseField>

<ResponseField name="type" type="string" required>
  Erwarteter Typ des Response-Werts – dies kann eine beliebige Zeichenkette sein.
</ResponseField>

<ResponseField name="default" type="string">
  Der Standardwert.
</ResponseField>

<ResponseField name="required" type="boolean">
  „required“ neben dem Feldnamen anzeigen.
</ResponseField>

<ResponseField name="deprecated" type="boolean">
  Ob ein Feld als veraltet gilt oder nicht.
</ResponseField>

<ResponseField name="pre" type="string[]">
  Labels, die vor dem Feldnamen angezeigt werden
</ResponseField>

<ResponseField name="post" type="string[]">
  Labels, die nach dem Feldnamen angezeigt werden
</ResponseField>
