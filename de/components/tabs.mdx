---
title: "Tabs"
description: "Inhalte mit der Tabs-Komponente umschalten"
icon: "panel-top"
---

Verwenden Sie Tabs, um Inhalte in mehrere Bereiche zu gliedern, zwischen denen Nutzende wechseln können. Sie können beliebig viele Tabs hinzufügen und in jedem Tab weitere Komponenten einfügen.

<Tabs>
  <Tab title="First tab">
    ☝️ Willkommen bei den Inhalten, die nur im ersten Tab sichtbar sind.

    Sie können beliebig viele Komponenten in Tabs einfügen. Zum Beispiel einen Codeblock:
    ```java HelloWorld.java
      class HelloWorld {
          public static void main(String[] args) {
              System.out.println("Hello, World!");
          }
      }
    ```
  </Tab>
  <Tab title="Second tab" icon="leaf">
    ✌️ Hier finden Sie Inhalte, die nur im zweiten Tab sichtbar sind.

    Dieser hat ein <Icon icon="leaf" />!
  </Tab>
  <Tab title="Third tab">
    💪 Hier finden Sie Inhalte, die nur im dritten Tab sichtbar sind.
  </Tab>
</Tabs>

<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="title" type="string" required>
  Der Titel des Tabs. Kurze Titel erleichtern die Navigation.
</ResponseField>

<ResponseField name="icon" type="string">
  Ein [Font Awesome](https://fontawesome.com/icons)-Icon, [Lucide](https://lucide.dev/icons)-Icon, eine URL zu einem Icon oder ein relativer Pfad zu einem Icon.
</ResponseField>

<ResponseField name="iconType" type="string">
  Nur für Font-Awesome-Icons: Eines von `regular`, `solid`, `light`, `thin`, `sharp-solid`, `duotone`, `brands`.
</ResponseField>
