---
title: "Hinweise"
description: "Verwenden Sie Hinweise, um Ihren Inhalten ansprechenden Kontext hinzuzufügen"
---

Hinweise können als Note, Warning, Info, Tip, Check oder Danger gestaltet werden – oder erstellen Sie Ihren eigenen Hinweis:

<Note>Dies fügt dem Inhalt einen Hinweis hinzu</Note>

```mdx
<Note>Dies fügt eine Notiz zum Inhalt hinzu</Note>
```

<Warning>Dies löst eine Warnung aus, auf die geachtet werden sollte</Warning>

```mdx
<Warning>Dies ist eine Warnung, die Sie beachten sollten</Warning>
```

<Info>Dies weist auf wichtige Informationen hin</Info>

```mdx
<Info>Dies hebt wichtige Informationen hervor</Info>
```

<Tip>Dies ist ein hilfreicher Hinweis</Tip>

```mdx
<Tip>Dies zeigt einen hilfreichen Tipp an</Tip>
```

<Check>Damit erhalten wir einen geprüften Status</Check>

```mdx
<Check>Dies gibt uns einen geprüften Status</Check>
```

<Danger>Dies ist ein Warnhinweis</Danger>

```mdx
<Danger>Dies ist eine Gefahrenmeldung</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular"> Dies ist ein individueller Hinweis</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Dies ist ein individueller Hinweis</Callout>
```
