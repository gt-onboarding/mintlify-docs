---
title: "Hinweise"
description: "Verwenden Sie Hinweise, um Ihren Inhalten aufmerksamkeitsstarken Kontext hinzuzufügen"
icon: "info"
---

Hinweise können als Note, Warning, Info, Tip, Check oder Danger gestaltet werden – oder erstellen Sie Ihren eigenen Hinweis:

<Note>Dies fügt dem Inhalt eine Notiz hinzu</Note>

```mdx
<Note>This adds a note in the content</Note>
```

<Warning>Dies weist auf eine wichtige Warnung hin</Warning>

```mdx
<Warning>This raises a warning to watch out for</Warning>
```

<Info>Dies hebt wichtige Informationen hervor</Info>

```mdx
<Info>This draws attention to important information</Info>
```

<Tip>Dies gibt einen hilfreichen Tipp</Tip>

```mdx
<Tip>This suggests a helpful tip</Tip>
```

<Check>Dies zeigt einen bestätigten Status an</Check>

```mdx
<Check>This brings us a checked status</Check>
```

<Danger>Dies ist ein Gefahrenhinweis</Danger>

```mdx
<Danger>This is a danger callout</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">Dies ist ein benutzerdefinierter Hinweis</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">This is a custom callout</Callout>
```
