---
title: "Hinweise"
description: "Verwenden Sie Hinweise, um Ihren Inhalten auffälligen Kontext hinzuzufügen"
icon: "info"
---

Hinweise können als Note, Warning, Info, Tip, Check oder Danger gestaltet werden – oder erstellen Sie Ihren eigenen Hinweis:

<Note>Dies fügt eine Notiz in den Inhalt ein</Note>

```mdx
<Note>Dies fügt eine Notiz in den Inhalt ein</Note>
```

<Warning>Dies weist auf eine wichtige Warnung hin</Warning>

```mdx
<Warning>Dies weist auf eine wichtige Warnung hin</Warning>
```

<Info>Dies hebt wichtige Informationen hervor</Info>

```mdx
<Info>Dies hebt wichtige Informationen hervor</Info>
```

<Tip>Dies gibt einen hilfreichen Tipp</Tip>

```mdx
<Tip>Dies gibt einen hilfreichen Tipp</Tip>
```

<Check>Dies zeigt einen bestätigten Status an</Check>

```mdx
<Check>Dies zeigt einen bestätigten Status an</Check>
```

<Danger>Dies ist ein Warnhinweis vor Gefahr</Danger>

```mdx
<Danger>Dies ist ein Warnhinweis vor Gefahr</Danger>
```

<Callout icon="key" color="#FFC107" iconType="regular">Dies ist ein benutzerdefinierter Hinweis</Callout>

```mdx wrap
<Callout icon="key" color="#FFC107" iconType="regular">Dies ist ein benutzerdefinierter Hinweis</Callout>
```
