<div id="inline-icons">
  ## Inline-Icons
</div>

Icons werden inline platziert, wenn sie innerhalb eines Absatzes verwendet werden.

<Icon icon="flag" iconType="solid" /> Die Dokumentation, die Sie brauchen – mühelos.

```markdown Inline-Icon-Beispiel
<Icon icon="flag" iconType="solid" /> Die Dokumentation, die Sie brauchen – mühelos.
```
