---
title: "Aufklapper"
description: "Umschalten, um verschachtelte Eigenschaften anzuzeigen"
icon: "list-tree"
---

Verwenden Sie Aufklapper, um verschachtelte Inhalte in Response-Feldern ein- und auszublenden. Sie eignen sich besonders, um komplexe Objekteigenschaften in API-Dokumentationen darzustellen.

<ResponseField name="user" type="User Object">
  <Expandable title="properties">

<ResponseField name="full_name" type="string">
  Vollständiger Name des Benutzers
</ResponseField>

<ResponseField name="is_over_21" type="boolean">
  Ob der Benutzer über 21 Jahre alt ist
</ResponseField>

  </Expandable>
</ResponseField>

```mdx Expandable example
<ResponseField name="user" type="User Object">
  <Expandable title="properties">
    <ResponseField name="full_name" type="string">
      The full name of the user
    </ResponseField>

    <ResponseField name="is_over_21" type="boolean">
      Whether the user is over 21 years old
    </ResponseField>
  </Expandable>
</ResponseField>
```

<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="title" type="string">
  Der Name des Objekts, das angezeigt wird.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Auf `true` setzen, damit der ausklappbare Bereich beim Laden der Seite geöffnet ist.
</ResponseField>
