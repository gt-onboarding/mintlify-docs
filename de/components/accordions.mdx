Akkordeons ermöglichen es Nutzerinnen und Nutzern, Inhaltsabschnitte ein- und auszuklappen. Verwenden Sie Akkordeons für progressive Offenlegung und zur Strukturierung von Informationen.


<div id="single-accordion">
  ## Einzelnes Akkordeon
</div>

<Accordion title="Ich bin ein Akkordeon.">
  Hier können Sie beliebige Inhalte einfügen, einschließlich anderer Komponenten wie Code:
   ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
</Accordion>

<div id="accordion-groups">
  ## Akkordeon-Gruppen
</div>

Gruppieren Sie zusammengehörige Akkordeons mit `<AccordionGroup>`. So entsteht ein zusammenhängender Abschnitt mit Akkordeons, die sich einzeln ein- oder ausklappen lassen.

<AccordionGroup>
  <Accordion title="Erste Schritte">
    In Akkordeons können Sie auch andere Komponenten verwenden.

    ```java HelloWorld.java
    class HelloWorld {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }
    ```
  </Accordion>

  <Accordion title="Erweiterte Funktionen" icon="bot">
    Fügen Sie Symbole hinzu, um Akkordeons visuell unterscheidbarer und leichter erfassbar zu machen.
  </Accordion>
  
  <Accordion title="Fehlerbehebung">
    Halten Sie zusammengehörige Inhalte in Gruppen organisiert.
  </Accordion>
</AccordionGroup>

<div id="properties">
  ## Eigenschaften
</div>

<ResponseField name="title" type="string" required>
  Titel in der Akkordeon-Vorschau.
</ResponseField>

<ResponseField name="description" type="string">
  Detail unter dem Titel in der Akkordeon-Vorschau.
</ResponseField>

<ResponseField name="defaultOpen" type="boolean" default="false">
  Ob das Akkordeon standardmäßig geöffnet ist.
</ResponseField>
