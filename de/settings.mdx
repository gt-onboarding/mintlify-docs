Die Datei `docs.json` ermöglicht es, eine Sammlung von Markdown-Dateien in eine navigierbare, individuell anpassbare Dokumentations-Website zu verwandeln. Diese erforderliche Konfigurationsdatei steuert Styling, Navigation, Integrationen und mehr. Betrachten Sie sie als den Bauplan Ihrer Dokumentation.

Einstellungen in `docs.json` gelten global für alle Seiten.

<div id="setting-up-your-docsjson">
  ## Einrichten Ihrer `docs.json`
</div>

Zum Einstieg müssen Sie nur `theme`, `name`, `colors.primary` und `navigation` angeben. Weitere Felder sind optional und können hinzugefügt werden, sobald Ihre Dokumentationsanforderungen wachsen.

Für das beste Bearbeitungserlebnis fügen Sie die Schema-Referenz am Anfang Ihrer `docs.json`-Datei ein. Dadurch werden in den meisten Code-Editoren Autovervollständigung, Validierung und hilfreiche Tooltips aktiviert:

```json
{
  "$schema": "https://mintlify.com/docs.json",
  "theme": "mint",
  "name": "Your Docs",
  "colors": {
    "primary": "#ff0000"
  },
  "navigation": {
    // Your navigation structure
  }
  // The rest of your configuration
}
```

  <Expandable title="Colors">
    <ResponseField name="primary" type="string matching ^#([a-fA-F0-9]{6}|[a-fA-F0-9]{3})$" required>

<div id="upgrading-from-mintjson">
  ## Upgrade von `mint.json`
</div>

Wenn Ihr Docs-Projekt die veraltete Datei `mint.json` verwendet, befolgen Sie diese Schritte, um auf `docs.json` zu aktualisieren.

<Steps>
  <Step title="CLI installieren oder aktualisieren">
    Wenn Sie die [CLI](/de/installation) noch nicht installiert haben, installieren Sie sie jetzt:

    <CodeGroup>
      ```bash npm
      npm i -g mint
      ```

      ```bash yarn
      yarn global add mint
      ```

      ```bash pnpm
      pnpm add -g mint
      ```
    </CodeGroup>

    Wenn Sie die CLI bereits installiert haben, stellen Sie sicher, dass sie auf dem neuesten Stand ist:

    ```bash
    mint update
    ```
  </Step>

  <Step title="Ihre docs.json-Datei erstellen">
    Führen Sie in Ihrem Docs-Repository Folgendes aus:

    ```bash
    mint upgrade
    ```
