[React-Komponenten](https://react.dev) sind eine leistungsstarke Möglichkeit, interaktive und wiederverwendbare Elemente in Ihrer Dokumentation zu erstellen.

<div id="using-react-components">
  ## Verwenden von React-Komponenten
</div>

Sie können React-Komponenten direkt in Ihren `MDX`-Dateien mit [React Hooks](https://react.dev/reference/react/hooks) erstellen.

<div id="example">
  ### Beispiel
</div>

In diesem Beispiel wird eine `Counter`-Komponente deklariert und anschließend mit `<Counter />` verwendet.

```mdx
export const Counter = () => {
  const [count, setCount] = useState(0)

  const increment = () => setCount(count + 1)
  const decrement = () => setCount(count - 1)

  return (
  <div className="flex items-center justify-center">
      <div className="flex items-center rounded-xl overflow-hidden border border-zinc-950/20 dark:border-white/20">
        <button
          onClick={decrement}
          className="flex items-center justify-center h-8 w-8 text-zinc-950/80 dark:text-white/80 border-r border-zinc-950/20 dark:border-white/20"
          aria-label="Verringern"
        >
          -
        </button>

        <div className="flex text-sm items-center justify-center h-8 px-6 text-zinc-950/80 dark:text-white/80 font-medium min-w-[4rem] text-center">
          {count}
        </div>

        <button
          onClick={increment}
          className="flex items-center justify-center h-8 w-8 text-zinc-950/80 dark:text-white/80 border-l border-zinc-950/20 dark:border-white/20"
          aria-label="Erhöhen"
        >
          +
        </button>
      </div>
    </div>
  )
}

<Counter />
```

  ## Komponenten importieren
</div>

Damit du React-Komponenten in deinen `MDX`-Dateien verwenden kannst, müssen sich die Komponenten-Dateien im Ordner `snippets` befinden. Anschließend kannst du sie in jede `MDX`-Seite deiner Dokumentation importieren. Erfahre mehr über [wiederverwendbare Snippets](/de/create/reusable-snippets).

<div id="example">
  ### Beispiel
</div>

In diesem Beispiel wird eine `ColorGenerator`-Komponente deklariert, die mehrere React-Hooks verwendet, und anschließend in einer `MDX`-Datei eingesetzt.

Erstelle die Datei `color-generator.jsx` im Ordner `snippets`:

```mdx /snippets/color-generator.jsx [expandable]
export const ColorGenerator = () => {
  const [hue, setHue] = useState(180)
  const [saturation, setSaturation] = useState(50)
  const [lightness, setLightness] = useState(50)
  const [colors, setColors] = useState([])

  useEffect(() => {
    const newColors = []
    for (let i = 0; i < 5; i++) {
      const l = Math.max(10, Math.min(90, lightness - 20 + i * 10))
      newColors.push(`hsl(${hue}, ${saturation}%, ${l}%)`)
    }
    setColors(newColors)
  }, [hue, saturation, lightness])

  const copyToClipboard = (color) => {
    navigator.clipboard
      .writeText(color)
      .then(() => {
        console.log(`${color} in die Zwischenablage kopiert!`)
      })
      .catch((err) => {
        console.error("Kopieren fehlgeschlagen: ", err)
      })
  }

  return (
    <div className="p-4 border dark:border-zinc-950/80 rounded-xl not-prose">
      <div className="space-y-4">
        <div className="space-y-2">
          <label className="block text-sm text-zinc-950/70 dark:text-white/70">
            Farbton: {hue}°
            <input
              type="range"
              min="0"
              max="360"
              value={hue}
              onChange={(e) => setHue(Number.parseInt(e.target.value))}
              className="w-full h-2 bg-zinc-950/20 rounded-lg appearance-none cursor-pointer dark:bg-white/20 mt-1"
              style={{
                background: `linear-gradient(to right, 
                  hsl(0, ${saturation}%, ${lightness}%), 
                  hsl(60, ${saturation}%, ${lightness}%), 
                  hsl(120, ${saturation}%, ${lightness}%), 
                  hsl(180, ${saturation}%, ${lightness}%), 
                  hsl(240, ${saturation}%, ${lightness}%), 
                  hsl(300, ${saturation}%, ${lightness}%), 
                  hsl(360, ${saturation}%, ${lightness}%))`,
              }}
            />
          </label>

          <label className="block text-sm text-zinc-950/70 dark:text-white/70">
            Sättigung: {saturation}%
            <input
              type="range"
              min="0"
              max="100"
              value={saturation}
              onChange={(e) => setSaturation(Number.parseInt(e.target.value))}
              className="w-full h-2 bg-zinc-950/20 rounded-lg appearance-none cursor-pointer dark:bg-white/20 mt-1"
              style={{
                background: `linear-gradient(to right, 
                  hsl(${hue}, 0%, ${lightness}%), 
                  hsl(${hue}, 50%, ${lightness}%), 
                  hsl(${hue}, 100%, ${lightness}%))`,
              }}
            />
          </label>

          <label className="block text-sm text-zinc-950/70 dark:text-white/70">
            Helligkeit: {lightness}%
            <input
              type="range"
              min="0"
              max="100"
              value={lightness}
              onChange={(e) => setLightness(Number.parseInt(e.target.value))}
              className="w-full h-2 bg-zinc-950/20 rounded-lg appearance-none cursor-pointer dark:bg-white/20 mt-1"
              style={{
                background: `linear-gradient(to right, 
                  hsl(${hue}, ${saturation}%, 0%), 
                  hsl(${hue}, ${saturation}%, 50%), 
                  hsl(${hue}, ${saturation}%, 100%))`,
              }}
            />
          </label>
        </div>

        <div className="flex space-x-1">
          {colors.map((color, idx) => (
            <div
              key={idx}
              className="h-16 rounded flex-1 cursor-pointer transition-transform hover:scale-105"
              style={{ backgroundColor: color }}
              title={`Zum Kopieren klicken: ${color}`}
              onClick={() => copyToClipboard(color)}
            />
          ))}
        </div>

        <div className="text-sm font-mono text-zinc-950/70 dark:text-white/70">
          <p>
            Basisfarbe: hsl({hue}, {saturation}%, {lightness}%)
          </p>
        </div>
      </div>
    </div>
  )
}
```

Importieren Sie die Komponente `ColorGenerator` und verwenden Sie sie in einer `MDX`-Datei:

```mdx
import { ColorGenerator } from "/snippets/color-generator.jsx"

<ColorGenerator />
```

Der Farbgenerator wird als interaktive React-Komponente dargestellt.

  ## Überlegungen
</div>

<AccordionGroup>
  <Accordion title="Auswirkungen des clientseitigen Renderings">
    React-Hook-Komponenten werden clientseitig gerendert, was mehrere Auswirkungen hat:

    * **SEO**: Suchmaschinen indexieren dynamische Inhalte möglicherweise nicht vollständig.
    * **Erstladung**: Besucher können einen kurzen Ladezustand sehen, bevor Komponenten gerendert werden.
    * **Barrierefreiheit**: Stellen Sie sicher, dass Änderungen an dynamischen Inhalten von Screenreadern angekündigt werden.
  </Accordion>

  <Accordion title="Performance-Best Practices">
    * **Abhängigkeitsarrays optimieren**: Nehmen Sie in Ihre `useEffect`-Abhängigkeitsarrays nur nötige Abhängigkeiten auf.
    * **Komplexe Berechnungen memoisieren**: Verwenden Sie `useMemo` oder `useCallback` für rechenintensive Operationen.
    * **Re-Renders reduzieren**: Zerlegen Sie große Komponenten in kleinere, um Kaskaden von erneuten Renderings zu vermeiden.
    * **Lazy Loading**: Ziehen Sie Lazy Loading für komplexe Komponenten in Betracht, um die anfängliche Seitenladezeit zu verbessern.
  </Accordion>
</AccordionGroup>
