---
title: "SDK-Beispiele hinzufügen"
description: "Zeigen Sie sprachspezifische Codebeispiele neben Ihren API-Endpunkten an, um Entwickler:innen zu zeigen, wie Ihre SDKs verwendet werden"
---

Wenn Ihre Nutzer:innen mit Ihrer API über ein SDK statt direkt per Netzwerkanfrage interagieren, können Sie die Erweiterung `x-codeSamples` verwenden, um Codebeispiele zu Ihrem OpenAPI-Dokument hinzuzufügen und auf Ihren OpenAPI-Seiten anzuzeigen.

Diese Eigenschaft kann zu jeder Anfragemethode hinzugefügt werden und hat das folgende Schema.

<ParamField body="lang" type="string" required>
  Die Sprache des Codebeispiels.
</ParamField>

<ParamField body="label" type="string">
  Die Bezeichnung für das Beispiel. Dies ist hilfreich, wenn Sie mehrere Beispiele für einen einzelnen Endpunkt bereitstellen.
</ParamField>

<ParamField body="source" type="string" required>
  Der Quellcode des Beispiels.
</ParamField>

Hier ist ein Beispiel für Codebeispiele für eine App zur Pflanzenverfolgung, die sowohl ein Bash-CLI-Tool als auch ein JavaScript-SDK bietet.

```yaml
paths:
  /plants:
    get:
      # ...
      x-codeSamples:
        - lang: bash
          label: Alle unbewässerten Pflanzen auflisten
          source: |
            planter list -u
        - lang: javascript
          label: Alle unbewässerten Pflanzen auflisten
          source: |
            const planter = require('planter');
            planter.list({ unwatered: true });
        - lang: bash
          label: Alle Topfpflanzen auflisten
          source: |
            planter list -p
        - lang: javascript
          label: Alle Topfpflanzen auflisten
          source: |
            const planter = require('planter');
            planter.list({ potted: true });
```
