---
title: "Mehrere Antworten"
description: "Antwortvarianten für denselben Endpunkt anzeigen"
---

Wenn Ihre API je nach Eingabeparametern, Benutzerkontext oder anderen Bedingungen der Anfrage unterschiedliche Antworten zurückgibt, können Sie mehrere Antwortbeispiele mit der Eigenschaft `examples` dokumentieren.

Diese Eigenschaft kann zu jeder Antwort hinzugefügt werden und hat das folgende Schema.

```yaml
responses:
  "200":
    description: Erfolgreiche Antwort
    content:
      application/json:
        schema:
          $ref: "#/components/schemas/YourResponseSchema"
        examples:
          us:
            summary: Antwort für die Vereinigten Staaten
            value:
              countryCode: "US"
              currencyCode: "USD"
              taxRate: 0.0825
          gb:
            summary: Antwort für das Vereinigte Königreich
            value:
              countryCode: "GB"
              currencyCode: "GBP"
              taxRate: 0.20
```
