<div id="lists">
  ## Listen
</div>

Listen folgen der offiziellen [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#lists-1).

<div id="ordered-list">
  ### Geordnete Liste
</div>

Um eine geordnete Liste zu erstellen, setzen Sie vor die Listeneinträge Zahlen gefolgt von einem Punkt.

1. Erster Eintrag
2. Zweiter Eintrag
3. Dritter Eintrag
4. Vierter Eintrag

```mdx
1. First item
2. Second item
3. Third item
4. Fourth item
```

<div id="unordered-list">
  ### Ungeordnete Liste
</div>

Um eine ungeordnete Liste zu erstellen, fügen Sie Bindestriche (`-`), Sternchen (`*`) oder Pluszeichen (`+`) vor Listeneinträgen hinzu.

- Erstes Element
- Zweites Element
- Drittes Element
- Viertes Element

```mdx
- Erstes Element
- Zweites Element
- Drittes Element
- Viertes Element
```

<div id="nested-list">
  ### Verschachtelte Liste
</div>

Rücken Sie Listenelemente ein, um sie zu verschachteln.

- Erstes Element
- Zweites Element
  - Zusätzliches Element
  - Zusätzliches Element
- Drittes Element

```mdx
- First item
- Second item
  - Additional item
  - Additional item
- Third item
```

<div id="tables">
  ## Tabellen
</div>

Tabellen folgen der offiziellen [Markdown-Syntax](https://www.markdownguide.org/extended-syntax/#tables).

Um eine Tabelle hinzuzufügen, verwenden Sie drei oder mehr Bindestriche (`---`), um die Kopfzeile jeder Spalte zu erstellen, und Pipes (`|`), um die Spalten zu trennen. Für die Kompatibilität sollten Sie außerdem am Anfang und Ende jeder Zeile eine Pipe hinzufügen.

| Eigenschaft | Beschreibung                         |
| ----------- | ------------------------------------ |
| Name        | Vollständiger Name des Benutzers     |
| Age         | Angegebenes Alter                    |
| Joined      | Ob der Benutzer der Community beigetreten ist |

```mdx
| Property | Description                           |
| -------- | ------------------------------------- |
| Name     | Full name of user                     |
| Age      | Reported age                          |
| Joined   | Whether the user joined the community |
```
