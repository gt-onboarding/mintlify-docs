---
title: "Heap"
---

Füge Folgendes zu deiner Datei `docs.json` hinzu, um Analysedaten an Heap zu senden.

<CodeGroup>

```json Analytics-Optionen in docs.json
"integrations": {
    "heap": {
        "appId": "required"
    }
}
```

```json Beispiel
"integrations": {
    "heap": {
        "appId": "1234567890"
    }
}
```

</CodeGroup>
