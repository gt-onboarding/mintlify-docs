---
title: "Fathom"
---

Fügen Sie Folgendes in Ihre `docs.json`-Datei ein, um Analysedaten an Fathom zu senden.

Die `siteId` finden Sie in den Skripteinstellungen.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Example
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
