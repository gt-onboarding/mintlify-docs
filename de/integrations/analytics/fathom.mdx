---
title: "Fathom"
---

Fügen Sie Folgendes zu Ihrer `docs.json`-Datei hinzu, um Analytics an Fathom zu senden.

Sie können die `siteId` in Ihren Skripteinstellungen finden.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Example
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
