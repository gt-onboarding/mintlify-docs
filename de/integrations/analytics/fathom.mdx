---
title: "Fathom"
---

Fügen Sie Folgendes zu Ihrer Datei `docs.json` hinzu, um Analysedaten an Fathom zu senden.

Die `siteId` finden Sie in Ihren Skripteinstellungen.

<CodeGroup>

```json Analytics-Optionen in docs.json
"integrations": {
    "fathom": {
        "siteId": "required"
    }
}
```

```json Beispiel
"integrations": {
    "fathom": {
        "siteId": "YSVMSDAY"
    }
}
```

</CodeGroup>
