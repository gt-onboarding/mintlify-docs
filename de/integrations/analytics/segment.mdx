---
title: "Segment"
---

Fügen Sie Ihren Segment-Write-Key in Ihre `docs.json`-Datei ein, um Analysen an Segment zu senden.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "segment": {
        "key": "required",
    }
}
```

```json Example
"integrations": {
    "segment": {
        "key": "nqJxiRG15Y6M594P8Sb8ESEciU3VC2"
    }
}
```

</CodeGroup>
