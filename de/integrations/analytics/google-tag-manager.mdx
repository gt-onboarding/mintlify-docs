---
title: "Google Tag Manager"
---

Fügen Sie Ihre Tag-ID in die Datei `docs.json` ein, dann injizieren wir das Google-Tag-Manager-Skript auf allen Ihren Seiten.

Sie sind dafür verantwortlich, bei Bedarf Cookie-Einwilligungsbanner mit dem Google Tag Manager einzurichten.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "gtm": {
        "tagId": "required"
    }
}
```

```json Example
"integrations": {
    "gtm": {
        "tagId": "GTM-MGBL4PW"
    }
}
```

</CodeGroup>
