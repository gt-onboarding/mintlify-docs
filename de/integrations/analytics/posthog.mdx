---
title: "PostHog"
---

Fügen Sie Folgendes zu Ihrer Datei `docs.json` hinzu, um Analytics an PostHog zu senden.

Sie müssen `apiHost` nur angeben, wenn Sie PostHog selbst hosten. Standardmäßig senden wir Ereignisse an `https://app.posthog.com`.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "posthog": {
        "apiKey": "YOUR_POSTHOG_PROJECT_API_KEY",
        "apiHost": "optional"
    }
}
```

```json Example
"integrations": {
    "posthog": {
        "apiKey": "phc_TXdpocbYTeZVm5VJmMzHTMrCofBQu3e0kN7HGMNGTVW"
    }
}
```

</CodeGroup>

<br />

<Warning>
    Das Aktivieren von PostHog-Analytics deaktiviert die Analytics im Mintlify-Dashboard.
</Warning>

<div id="session-recordings">
  ## Sitzungsaufzeichnungen
</div>

Bevor Sie Sitzungsaufzeichnungen erhalten können, müssen Sie die URL Ihrer Docs-Website in Posthog unter „Authorized domains for recordings“ hinzufügen. Die Option zum Hinzufügen Ihrer URL finden Sie in den Projekteinstellungen von Posthog.
