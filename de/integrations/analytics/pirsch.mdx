---
title: "Pirsch"
---

Fügen Sie Folgendes zu Ihrer `docs.json`-Datei hinzu, um Analytics an Pirsch zu übermitteln.

Ihre Site-ID finden Sie unter Einstellungen > Entwickler > Identifikationscode.

<CodeGroup>

```json Analytics options in docs.json
"integrations": {
    "pirsch": {
        "id": "required"
    }
}
```

```json Example
"integrations": {
    "pirsch": {
        "id": "8Kw7OKxBfswOjnKGZa7P9Day8JmVYwTp"
    }
}
```

</CodeGroup>
