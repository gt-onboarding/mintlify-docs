---
title: "Intercom"
---

Fügen Sie Folgendes zu Ihrer `docs.json`-Datei hinzu, um ein [Intercom](https://www.intercom.com)-Widget einzubinden.

<CodeGroup>

```json Integrationsoptionen in docs.json
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

```json Beispiel
"integrations": {
      "intercom": {
            "appId": "APP_ID"
      }
}
```

</CodeGroup>
