<div id="rate-limits">
  ## Nutzungsbegrenzungen
</div>

Die Assistent-API hat die folgenden Limits:

- 10.000 Nutzungen pro Schlüssel und Monat
- 10.000 Anfragen pro Mintlify-Organisation und Stunde
- 10.000 Anfragen pro IP und Tag

<div id="suggested-usage">
  ## Empfohlene Verwendung
</div>

Verwenden Sie für beste Ergebnisse den [useChat-Hook aus ai-sdk](https://ai-sdk.dev/docs/reference/ai-sdk-ui/use-chat#usechat), um Anfragen zu senden und Antworten zu verarbeiten.

Sie können `fp`, `threadId` und `filter` im Feld `body` des an den Hook übergebenen Options-Parameters festlegen.
