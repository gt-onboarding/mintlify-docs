Fügen Sie Bilder hinzu, betten Sie Videos ein und binden Sie interaktive Inhalte über iframes in Ihre Dokumentation ein.

Alle statischen Assets in Ihrem Docs-Repository werden automatisch unter dem entsprechenden Pfad auf Ihrer Domain ausgeliefert. Wenn Sie beispielsweise `/images/my-logo.png` in Ihrem Repository haben, ist das Bild unter `https://docs.yoursite.com/images/my-logo.png` verfügbar.

<div id="images">
  ## Bilder
</div>

Fügen Sie Bilder hinzu, um Ihrer Dokumentation visuellen Kontext, Beispiele oder dekorative Elemente hinzuzufügen.

<div id="basic-image-syntax">
  ### Grundlegende Bildsyntax
</div>

Verwenden Sie die [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#images), um Bilder zu Ihrer Dokumentation hinzuzufügen:

```mdx
![Alttext, der das Bild beschreibt](/path/to/image.png)
```

<Tip>
  Fügen Sie immer einen aussagekräftigen Alttext hinzu, um die Barrierefreiheit und SEO zu verbessern. Der Alttext sollte klar beschreiben, was auf dem Bild zu sehen ist.
</Tip>

Bilddateien müssen kleiner als 20 MB sein. Größere Dateien sollten Sie auf einem CDN-Dienst wie [Amazon S3](https://aws.amazon.com/s3) oder [Cloudinary](https://cloudinary.com) hosten.

<div id="html-image-embeds">
  ### HTML-Bildeinbindungen
</div>

Für mehr Kontrolle über die Bilddarstellung verwenden Sie HTML-`<img>`-Tags:

```html
<img 
  src="/images/dashboard.png" 
  alt="Main dashboard interface"
  height="300"
  className="rounded-lg"
/>
```

<div id="disable-zoom-functionality">
  #### Zoomfunktion deaktivieren
</div>

Um das standardmäßige Zoomen per Klick bei Bildern zu deaktivieren, fügen Sie die Eigenschaft `noZoom` hinzu:

```html highlight="4"
<img 
  src="/images/screenshot.png" 
  alt="Descriptive alt text"
  noZoom 
  height="200"
/>
```

<div id="link-images">
  #### Bilder verlinken
</div>

Um ein Bild anklickbar zu machen, umschließen Sie das Bild mit einem Anchor-Tag und fügen Sie die Eigenschaft `noZoom` hinzu:

```html
<a href="https://mintlify.com" target="_blank">
  <img 
    src="/images/logo.png" 
    alt="Mintlify logo"
    noZoom 
    height="100"
  />
</a>
```

<Note>
  Bilder innerhalb von Anchor-Tags zeigen automatisch einen Zeiger-Cursor an, der ihre Klickbarkeit signalisiert.
</Note>

<div id="light-and-dark-mode-images">
  #### Bilder für Hell- und Dunkelmodus
</div>

Um unterschiedliche Bilder für helle und dunkle Themes anzuzeigen, verwenden Sie Tailwind CSS-Klassen:

```html
<!-- Bild im Hellmodus -->
<img 
  className="block dark:hidden" 
  src="/images/light-mode.png" 
  alt="Benutzeroberfläche im Hellmodus"
/>

<!-- Bild im Dunkelmodus -->
<img 
  className="hidden dark:block" 
  src="/images/dark-mode.png" 
  alt="Benutzeroberfläche im Dunkelmodus"
/>
```

<div id="videos">
  ## Videos
</div>

Mintlify unterstützt [HTML-Tags in Markdown](https://www.markdownguide.org/basic-syntax/#html) und bietet Ihnen die Flexibilität, umfangreiche Inhalte zu erstellen.

<Tip>
  Fügen Sie in Videoelementen immer einen Fallback-Text ein, damit Browser ohne Videounterstützung alternative Inhalte anzeigen können.
</Tip>

<div id="youtube-embeds">
  ### YouTube-Einbettungen
</div>

Betten Sie YouTube-Videos mit iframe-Elementen ein:

```html
<iframe
  className="w-full aspect-video rounded-xl"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube-Video-Player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>
```

<div id="self-hosted-videos">
  ### Selbstgehostete Videos
</div>

Verwenden Sie das HTML-Element `<video>` für selbstgehostete Videoinhalte:

```html
<video
  controls
  className="w-full aspect-video rounded-xl"
  src="link-to-your-video.com"
></video>
```

<div id="autoplay-videos">
  ### Videos automatisch abspielen
</div>

Um ein Video automatisch abzuspielen, verwenden Sie:

```html
<video
  autoPlay
  muted
  loop
  playsInline
  className="w-full aspect-video rounded-xl"
  src="/videos/demo.mp4"
></video>
```

<Note>
  Bei Verwendung der JSX-Syntax schreiben Sie mehrteilige Attribute in CamelCase: `autoPlay`, `playsInline`, `allowFullScreen`.
</Note>

<div id="iframes">
  ## iframes
</div>

Externe Inhalte mit iframe-Elementen einbetten:

```html
<iframe 
  src="https://example.com/embed" 
  title="Embedded content"
  className="w-full h-96 rounded-xl"
></iframe>
```
