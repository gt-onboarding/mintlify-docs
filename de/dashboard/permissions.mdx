---
title: "Editor-Berechtigungen"
description: "Ermöglichen Sie mehr Teammitgliedern, Ihre Dokumentation zu aktualisieren"
---

Ein Editor hat Zugriff auf Ihr Dashboard und den Web-Editor.

Jede Person kann zu Ihrer Dokumentation beitragen, indem sie lokal arbeitet und Änderungen in Ihr Repository pusht. Es gibt jedoch wichtige Unterschiede darin, wie Änderungen veröffentlicht werden:

* **Editor-Änderungen**: Wenn ein Editor über den Web-Editor veröffentlicht oder einen Pull Request in Ihrem Docs-Repository merged, werden die Änderungen automatisch auf Ihrer Live-Website veröffentlicht.
* **Nicht-Editor-Änderungen**: Wenn eine Person ohne Editor-Rechte einen Pull Request in Ihr Repository merged, müssen Sie die Veröffentlichung in Ihrem Dashboard manuell auslösen, damit diese Änderungen auf Ihrer Live-Website erscheinen.

<div id="add-editors">
  ## Editoren hinzufügen
</div>

Standardmäßig hat das Teammitglied, das Ihre Mintlify-Organisation erstellt hat, Editorzugriff. Fügen Sie zusätzliche Editoren auf der Seite [Members](https://dashboard.mintlify.com/settings/organization/members) in Ihrem Dashboard hinzu.

Editor-Plätze werden nutzungsbasiert abgerechnet, und Sie können so viele Editoren haben, wie Sie benötigen. Weitere Details finden Sie auf unserer [Preisseite](https://mintlify.com/pricing).
