---
title: "Rollen"
description: "Steuern Sie den Zugriff auf Ihr Dashboard mit Rollen."
---

<Info>
  RBAC-Funktionen sind im [Enterprise-Tarif](https://mintlify.com/pricing?ref=rbac) verfügbar.
</Info>

Mintlify bietet zwei Zugriffslevel für das Dashboard: Editor und Admin.

Im Folgenden sind Aktionen aufgeführt, die auf die Admin-Rolle beschränkt sind:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Benutzerrollen aktualisieren |   ❌   |  ✅   |
| Benutzer löschen        |   ❌   |  ✅   |
| Admins einladen         |   ❌   |  ✅   |
| Abrechnung verwalten und aktualisieren |   ❌   |  ✅   |
| Benutzerdefinierte Domain aktualisieren |   ❌   |  ✅   |
| Git-Quelle aktualisieren |   ❌   |  ✅   |
| Organisation löschen    |   ❌   |  ✅   |

Weitere Aktionen im Dashboard sind für beide Rollen verfügbar.

Sie können so viele Admins einladen, wie Sie möchten. Wir empfehlen jedoch, den Adminzugriff auf Nutzer zu beschränken, die ihn benötigen.
