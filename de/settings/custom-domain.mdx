Um Ihre Dokumentation auf einer benutzerdefinierten Domain einzurichten, legen Sie die gewünschte Domain in Ihren Mintlify-Einstellungen fest und konfigurieren Sie die DNS-Einstellungen bei Ihrem Domainanbieter.

<Info>
  Möchten Sie ein benutzerdefiniertes Unterverzeichnis wie `mintlify.com/docs` einrichten? Siehe die Dokumentation zum [benutzerdefinierten Unterverzeichnis](/de/advanced/subpath/cloudflare).
</Info>

<div id="dashboard-settings">
  ## Dashboard-Einstellungen
</div>

1. Navigieren Sie zu Ihrem [Dashboard](https://dashboard.mintlify.com)
2. Wählen Sie **Settings**
3. Wählen Sie **Domain Setup**
4. Geben Sie Ihre gewünschte benutzerdefinierte Domain ein, z. B. `docs.mintlify.com`

<div id="configuring-your-dns">
  ## DNS konfigurieren
</div>

1. Rufen Sie die DNS-Einstellungen Ihrer Domain beim Domainanbieter auf.
2. Erstellen Sie einen neuen DNS-Eintrag und tragen Sie die folgenden Werte ein:

```bash
CNAME | docs | cname.vercel-dns.com.
```

<div id="provider-specific-settings">
  ## Anbieterspezifische Einstellungen
</div>

<AccordionGroup>
  <Accordion title="Vercel-Verifizierung">
    Wenn Vercel Ihr Domainanbieter ist, müssen Sie einen Verifizierungs-„TXT“-Eintrag hinzufügen. Diese Informationen werden nach dem Hinzufügen Ihrer benutzerdefinierten Domain in Ihrem Dashboard angezeigt und Ihnen auch per E-Mail zugeschickt.
  </Accordion>

  <Accordion title="Cloudflare-Sicherheitseinstellungen">
    Wenn Cloudflare Ihr DNS-Anbieter ist, müssen Sie für die HTTPS-Einstellung die Sicherheitsoption „Full (strict)“ aktivieren.
  </Accordion>
</AccordionGroup>
