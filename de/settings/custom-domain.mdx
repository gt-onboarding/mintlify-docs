Um Ihre Dokumentation auf einer eigenen Domain einzurichten, legen Sie die gewünschte Domain in Ihren Mintlify-Einstellungen fest und konfigurieren Sie die DNS-Einstellungen bei Ihrem Domainanbieter.

<Info>
  Möchten Sie ein eigenes Unterverzeichnis wie `mintlify.com/docs` einrichten? Siehe die Dokumentation zum [eigenen Unterverzeichnis](/de/advanced/subpath/cloudflare).
</Info>

<div id="dashboard-settings">
  ## Dashboard-Einstellungen
</div>

1. Navigieren Sie zu Ihrem [Dashboard](https://dashboard.mintlify.com)
2. Wählen Sie **Settings**
3. Wählen Sie **Domain Setup**
4. Geben Sie Ihre gewünschte benutzerdefinierte Domain ein, z. B. `docs.mintlify.com`

  <img alt="Benutzerdefinierte Domain" className="block dark:hidden" src="/images/add-custom-domain-light.png" />

  <img alt="Benutzerdefinierte Domain" className="hidden dark:block" src="/images/add-custom-domain-dark.png" />
</Frame>

<div id="configuring-your-dns">
  ## DNS konfigurieren
</div>

1. Gehen Sie in die DNS-Einstellungen Ihrer Domain beim Domainanbieter.
2. Erstellen Sie einen neuen DNS-Eintrag und geben Sie folgende Werte ein:

```bash
CNAME | docs | cname.vercel-dns.com.
```

<div id="provider-specific-settings">
  ## Anbieter-spezifische Einstellungen
</div>

<AccordionGroup>
  <Accordion title="Vercel-Verifizierung">
    Wenn Vercel Ihr Domainanbieter ist, müssen Sie einen Verifizierungs-`TXT`-Eintrag hinzufügen. Diese Informationen werden nach dem Einreichen Ihrer benutzerdefinierten Domain in Ihrem Dashboard angezeigt und Ihnen außerdem per E-Mail zugeschickt.
  </Accordion>

  <Accordion title="Cloudflare-Sicherheitseinstellungen">
    Wenn Cloudflare Ihr DNS-Anbieter ist, müssen Sie für die HTTPS-Einstellung die Sicherheitsoption „Full (strict)“ aktivieren.
  </Accordion>
</AccordionGroup>
