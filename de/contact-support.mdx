<div id="ask-our-docs">
  ## Fragen Sie unsere Doku
</div>

Drücken Sie <kbd>Command</kbd> + <kbd>I</kbd>, um einen Chat mit unserem auf unserer Dokumentation trainierten KI-Assistenten zu starten.

<div id="watch-video-tutorials">
  ## Video-Tutorials ansehen
</div>

Besuchen Sie unseren [YouTube](https://www.youtube.com/@GetMintlify/videos)-Kanal für Tutorials und Anleitungen zur Verwendung von Mintlify.

<div id="message-support">
  ## Nachrichten-Support
</div>

Senden Sie uns eine Nachricht über Ihr [Dashboard](https://dashboard.mintlify.com/), indem Sie in der Seitenleiste **Support** auswählen.

![GIF, das zeigt, wie Sie über das Dashboard den Support aufrufen, indem Sie die Schaltfläche „Support“ auswählen, anschließend eine Option wählen und eine Frage in das erscheinende Support-Modal eingeben.](https://mintlify.s3.us-west-1.amazonaws.com/mintlify/images/support-flow.gif)

<Info>
Wir bemühen uns, alle Anfragen innerhalb von 24 Stunden zu beantworten; während Stoßzeiten kann es jedoch zu Verzögerungen kommen.
</Info>

<div id="email-support">
  ## E-Mail-Support
</div>

Wenn Sie nicht auf Ihr Dashboard zugreifen können, schreiben Sie bitte eine E-Mail an <a href="mailto:support@mintlify.com">support@mintlify.com</a>.
