---
title: "Rollen"
description: "Steuern Sie den Zugriff auf Ihr Dashboard über Rollen."
---

<Info>
  RBAC-Funktionalität ist im [Enterprise-Tarif](https://mintlify.com/pricing?ref=rbac) verfügbar.
</Info>

Mintlify bietet zwei Zugriffslevel für das Dashboard: Editor und Admin.

Im Folgenden sind Aktionen aufgeführt, die der Admin-Rolle vorbehalten sind:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Benutzerrollen aktualisieren       |   ❌   |  ✅   |
| Benutzer löschen            |   ❌   |  ✅   |
| Admins einladen      |   ❌   |  ✅   |
| Abrechnung verwalten und aktualisieren |   ❌   |  ✅   |
| Benutzerdefinierte Domain aktualisieren    |   ❌   |  ✅   |
| Git-Quelle aktualisieren       |   ❌   |  ✅   |
| Organisation löschen              |   ❌   |  ✅   |

Alle anderen Aktionen im Dashboard stehen beiden Rollen zur Verfügung.

Sie können beliebig viele Admins einladen, wir empfehlen jedoch, den Admin-Zugriff auf Benutzer zu beschränken, die ihn benötigen.
