title: "Rollen"
description: "Steuern Sie den Zugriff auf Ihr Dashboard mit Rollen."
---

<Info>
  RBAC-Funktionalität ist im [Enterprise-Tarif](https://mintlify.com/pricing?ref=rbac) verfügbar.
</Info>

Mintlify bietet zwei Zugriffsebenen für das Dashboard: Editor und Admin.

Im Folgenden sind Aktionen beschrieben, die der Admin-Rolle vorbehalten sind:

|                         | Editor | Admin |
| ----------------------- | :----: | :---: |
| Benutzerrollen aktualisieren |   ❌   |  ✅   |
| Benutzer löschen        |   ❌   |  ✅   |
| Admins einladen         |   ❌   |  ✅   |
| Abrechnung verwalten & aktualisieren |   ❌   |  ✅   |
| Benutzerdefinierte Domain aktualisieren |   ❌   |  ✅   |
| Git-Quelle aktualisieren |   ❌   |  ✅   |
| Organisation löschen    |   ❌   |  ✅   |

Andere Aktionen im Dashboard stehen beiden Rollen zur Verfügung.

Sie können beliebig viele Admins einladen. Wir empfehlen jedoch, den Admin-Zugriff auf Benutzer zu beschränken, die ihn benötigen.
