---
title: "Editor-Berechtigungen"
description: "Ermöglichen Sie mehr Teammitgliedern, Ihre Dokumentation zu aktualisieren"
---

Ein Editor hat Zugriff auf Ihr Dashboard und den Web-Editor.

Jede Person kann zu Ihrer Dokumentation beitragen, indem sie lokal arbeitet und Änderungen in Ihr Repository pusht. Es gibt jedoch wichtige Unterschiede dabei, wie Änderungen ausgeliefert werden:

* **Änderungen von Editoren**: Wenn ein Editor über den Web-Editor veröffentlicht oder einen Pull Request in Ihr Docs-Repository merged, werden die Änderungen automatisch auf Ihrer Live-Website ausgeliefert.
* **Änderungen von Nicht-Editoren**: Wenn eine Person ohne Editorrolle einen Pull Request in Ihr Repository merged, müssen Sie die Auslieferung über Ihr Dashboard manuell auslösen, damit diese Änderungen auf Ihrer Live-Website erscheinen.

<div id="add-editors">
  ## Herausgeber hinzufügen
</div>

Standardmäßig hat das Teammitglied, das Ihre Mintlify-Organisation erstellt hat, Herausgeberzugriff. Fügen Sie zusätzliche Herausgeber auf der Seite [Members](https://dashboard.mintlify.com/settings/organization/members) Ihres Dashboards hinzu.

Herausgeberplätze werden nutzungsbasiert abgerechnet, und Sie können so viele Herausgeber haben, wie Sie benötigen. Weitere Details finden Sie auf unserer [Preisseite](https://mintlify.com/pricing).
