---
title: "Editor-Berechtigungen"
description: "Ermöglichen Sie mehr Teammitgliedern, Ihre Dokumentation zu aktualisieren"
---

Ein Editor hat Zugriff auf Ihr Dashboard und den Web-Editor.

Grundsätzlich kann jede Person zu Ihrer Dokumentation beitragen, indem sie lokal arbeitet und Änderungen in Ihr Repository pusht. Es gibt jedoch wichtige Unterschiede dabei, wie Änderungen bereitgestellt werden:

* **Änderungen von Editoren**: Wenn ein Editor über den Web-Editor veröffentlicht oder einen Pull Request in Ihr Docs-Repository merged, werden die Änderungen automatisch auf Ihrer Live-Website bereitgestellt.
* **Änderungen von Nicht-Editoren**: Wenn eine Person ohne Editorrolle einen Pull Request in Ihr Repository merged, müssen Sie die Bereitstellung manuell über Ihr Dashboard auslösen, damit diese Änderungen auf Ihrer Live-Website erscheinen.

<div id="add-editors">
  ## Editoren hinzufügen
</div>

Standardmäßig hat das Teammitglied, das Ihre Mintlify-Organisation erstellt hat, Editorzugriff. Fügen Sie weitere Editoren auf der Seite [Members](https://dashboard.mintlify.com/settings/organization/members) in Ihrem Dashboard hinzu.

Editor-Sitze werden nutzungsbasiert abgerechnet, und Sie können so viele Editoren haben, wie Sie benötigen. Weitere Details finden Sie auf unserer [Preisseite](https://mintlify.com/pricing).
